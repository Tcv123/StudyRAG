1.9: Numerical Methods

1. Root Finding

Sign-Change (Interval Bisection)

If \(f\) is continuous on \([a, b]\) and \(f(a)f(b) < 0\), a root exists in \((a, b)\) by the Intermediate Value Theorem. Bisection repeatedly halves the interval, keeping the half with a sign change.

Find a root of \(f(x) = x^3 - x - 1\) on \([1, 2]\) using 3 bisection steps.
\(f(1) = -1\), \(f(2) = 5\). Midpoint \(1.5\): \(f(1.5) = 0.875\). Root in \([1, 1.5]\).
Midpoint \(1.25\): \(f(1.25) = -0.297\). Root in \([1.25, 1.5]\).
Midpoint \(1.375\): \(f(1.375) = 0.224\). Root in \([1.25, 1.375]\).

Newton–Raphson Method

\(x_{n+1} = x_n - \dfrac{f(x_n)}{f'(x_n)}.\)

Geometrically: follow the tangent at \((x_n, f(x_n))\) to its \(x\)-intercept.

Convergence

  • NR converges quadratically near a simple root (the number of correct digits roughly doubles each iteration).
  • It can fail if \(f'(x_n)\) is very small (tangent nearly horizontal) or if the starting value is poor.
  • For multiple roots, convergence is only linear.
Apply NR to \(f(x) = x^3 - x - 1\) from \(x_0 = 1.5\).
\(f'(x) = 3x^2 - 1\). \(f(1.5) = 0.875\), \(f'(1.5) = 5.75\). \(x_1 = 1.5 - 0.875/5.75 \approx 1.348\).
Continue for further iterations: \(x_2 \approx 1.3259\), \(x_3 \approx 1.3247\). Converging to the real root.

Fixed-Point Iteration

Rearrange \(f(x) = 0\) as \(x = g(x)\), then iterate \(x_{n+1} = g(x_n)\).

Convergence requires \(|g'(\alpha)| < 1\) at the root \(\alpha\). If \(|g'(\alpha)| > 1\), the iteration diverges.

2. Numerical Integration

Trapezium Rule

Split \([a, b]\) into \(n\) equal strips of width \(h = (b - a)/n\):

\(\displaystyle \int_a^b f(x)\,dx \approx \tfrac{h}{2}\left[f(x_0) + 2f(x_1) + 2f(x_2) + \cdots + 2f(x_{n-1}) + f(x_n)\right].\)

Error: \(O(h^2)\). Exact for linear functions.

Simpson's Rule

Requires an even number of strips (\(n\) even):

\(\displaystyle \int_a^b f\,dx \approx \tfrac{h}{3}\left[f(x_0) + 4(f(x_1) + f(x_3) + \cdots) + 2(f(x_2) + f(x_4) + \cdots) + f(x_n)\right].\)

Error: \(O(h^4)\). Exact for cubics — much more accurate than trapezium for smooth functions.

Approximate \(\int_0^2 x^3\,dx\) using Simpson with \(n = 2\).
\(h = 1\), \(x_0 = 0, x_1 = 1, x_2 = 2\). \(f(x_0) = 0, f(x_1) = 1, f(x_2) = 8\).
\(S = \tfrac{1}{3}(0 + 4(1) + 8) = 4\). Exact = 4, so Simpson is exact here (cubics!).

3. Numerical Solution of Differential Equations

Euler's Method (Tangent-Line)

For \(y' = f(x, y)\), \(y(x_0) = y_0\):

\(y_{n+1} = y_n + h f(x_n, y_n), \quad x_{n+1} = x_n + h.\)

Improved Euler / Midpoint

More accurate variants use the slope at the midpoint or an average of start and end slopes (predictor-corrector methods).

Global Truncation Error

Euler's method has global error \(O(h)\); higher-order methods (Runge–Kutta 4) achieve \(O(h^4)\) but are outside the OCR specification.

5. Exam Style Strategy

  • For bisection/NR, show all iterations clearly (\(x_0, x_1, x_2, \ldots\)) to the required accuracy.
  • For Simpson's rule, check \(n\) is even; set up the table with \(x_i\) and \(f(x_i)\) values.
  • State the order of truncation error if asked about accuracy.
  • For NR failures (divergence, slow convergence), give a specific reason (e.g. "because \(f'(x_0) \approx 0\)").
