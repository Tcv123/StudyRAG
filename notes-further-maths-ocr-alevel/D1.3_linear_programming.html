D1.3: Linear Programming

1. Formulating a Linear Programme

A linear programme (LP) is an optimisation problem with:
  • Decision variables \(x_1, x_2, \ldots, x_n \ge 0\).
  • A linear objective function \(P = c_1 x_1 + c_2 x_2 + \cdots\) to maximise (or minimise).
  • Linear constraints \(a_{i1}x_1 + \cdots + a_{in}x_n \le b_i\) (or \(\ge\)).

Steps to Formulate

  1. Define decision variables clearly, in words and symbols.
  2. Translate each constraint (resource, demand, non-negativity) into an inequality.
  3. Write the objective function to maximise/minimise.
A factory makes products A and B. A uses 2 hours machine time and 3 hours labour; B uses 4 hours machine and 1 hour labour. 12 machine-hours and 9 labour-hours available. Profit £30 per A, £20 per B. Set up the LP.
Let \(x\) = number of A, \(y\) = number of B. \(x, y \ge 0\).
Machine: \(2x + 4y \le 12\). Labour: \(3x + y \le 9\).
Maximise \(P = 30x + 20y\).

2. Graphical Method (2 Variables)

  1. Plot each constraint as a line and shade the feasible region.
  2. Identify the vertices of the feasible region.
  3. Evaluate the objective at each vertex — the optimum lies at a vertex.

Objective Line Method (Alternative)

Draw a line \(c_1 x + c_2 y = k\) for some \(k\), then translate it parallel to itself across the feasible region. The last vertex touched gives the optimum.

OCR Exam Tip: Sketch the feasible region neatly, label each constraint, and state the coordinates of every vertex. Then list objective values in a table.

3. The Simplex Method

For problems with more than 2 decision variables, the graphical method fails. The Simplex method is an algebraic procedure.

Standard Form

Convert inequalities to equalities using slack variables \(s_i \ge 0\):

\(a_{i1}x_1 + \cdots + a_{in}x_n + s_i = b_i.\)

Simplex Steps

  1. Choose pivot column: the one with the most negative entry in the \(P\) row.
  2. Choose pivot row: compute ratios (RHS ÷ pivot column entry, positive entries only); choose the smallest.
  3. Pivot: divide the pivot row by the pivot element, then subtract multiples of this row from others to zero out the pivot column elsewhere.
  4. Repeat until the \(P\) row has no negative entries.

Reading the Solution

When optimal, basic variables equal the RHS values; all others are zero. The objective value is the RHS of the \(P\) row.

4. Interpreting the Result

  • If a slack variable is basic with non-zero value: the corresponding constraint is slack (not binding).
  • If a slack is non-basic (zero): the constraint is tight — the resource is fully used.

5. Mixed Constraints and Artificial Variables (Support)

For \(\ge\) constraints or equalities, introduce surplus and artificial variables, and use the Big-M method or two-phase Simplex. (Mostly beyond the OCR core Discrete syllabus but appears in stretch questions.)

6. Transportation Problems (Outline)

A special LP: ship goods from suppliers to consumers at minimum cost. Can be solved with a dedicated tableau method, but exam questions usually just require formulation as an LP.

7. Exam Style Strategy

  • Always define variables clearly in words before writing inequalities.
  • Graphical: state vertices and objective values in a table; state the optimum in context.
  • Simplex: show all tableaux, identify pivot choices, and clearly mark the optimum.
  • Interpret the answer — if \(x\) must be an integer (units made), check integer solutions near the LP optimum.
