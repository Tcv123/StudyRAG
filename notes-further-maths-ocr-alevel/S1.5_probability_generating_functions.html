S1.5: Probability Generating Functions

1. Definition

For a non-negative integer-valued random variable \(X\) with pmf \(p_r = P(X = r)\), the probability generating function (pgf) is \[G_X(t) = E(t^X) = \sum_{r=0}^{\infty} p_r t^r.\]

It's a power series in \(t\); the coefficient of \(t^r\) is the probability \(p_r\).

Key Property: \(G_X(1) = 1\)

Because \(\sum p_r = 1\). Always a useful check.

2. Standard pgfs

Distributionpgf
\(X \sim \operatorname{Bernoulli}(p)\)\(1 - p + pt\)
\(X \sim B(n, p)\)\((1 - p + pt)^n\)
\(X \sim \operatorname{Po}(\lambda)\)\(e^{\lambda(t - 1)}\)
\(X \sim \operatorname{Geometric}(p)\) (on \(1, 2, \ldots\))\(\dfrac{pt}{1 - (1-p)t}\)
Derive the pgf of \(X \sim \operatorname{Po}(\lambda)\).
\(G_X(t) = \sum_{r=0}^{\infty} \dfrac{e^{-\lambda}\lambda^r}{r!}\,t^r = e^{-\lambda}\sum \dfrac{(\lambda t)^r}{r!} = e^{-\lambda}e^{\lambda t} = e^{\lambda(t - 1)}\). ∎

3. Finding Mean and Variance from a pgf

\(E(X) = G_X'(1)\)
\(E[X(X-1)] = G_X''(1)\)
\(\operatorname{Var}(X) = G_X''(1) + G_X'(1) - [G_X'(1)]^2\)

Proof of mean: differentiate term by term — \(G_X'(t) = \sum r\,p_r t^{r-1}\), evaluated at \(t = 1\) gives \(E(X)\).

Find mean and variance of \(X \sim \operatorname{Po}(\lambda)\) from \(G(t) = e^{\lambda(t-1)}\).
\(G'(t) = \lambda e^{\lambda(t-1)}\), so \(G'(1) = \lambda\). \(E(X) = \lambda\). ✓
\(G''(t) = \lambda^2 e^{\lambda(t-1)}\), so \(G''(1) = \lambda^2\).
\(\operatorname{Var}(X) = \lambda^2 + \lambda - \lambda^2 = \lambda\). ✓

4. Sums of Independent Random Variables

If \(X\) and \(Y\) are independent, the pgf of \(X + Y\) is the product: \[G_{X+Y}(t) = G_X(t)\,G_Y(t).\]

This is a very powerful tool — identifying the distribution of a sum reduces to multiplying pgfs.

Show that the sum of two independent \(\operatorname{Po}(\lambda_1)\) and \(\operatorname{Po}(\lambda_2)\) is \(\operatorname{Po}(\lambda_1 + \lambda_2)\).
\(G_{X+Y}(t) = e^{\lambda_1(t-1)}\,e^{\lambda_2(t-1)} = e^{(\lambda_1 + \lambda_2)(t-1)}\) — the pgf of \(\operatorname{Po}(\lambda_1 + \lambda_2)\). ∎
Sum of \(n\) IID \(\operatorname{Bernoulli}(p)\) is \(B(n, p)\).
Each has pgf \((1 - p + pt)\); the product of \(n\) of them is \((1 - p + pt)^n\), the pgf of \(B(n, p)\). ∎

5. Linear Transformations

For \(Y = aX + b\) with \(a, b \in \mathbb{Z}_{\ge 0}\):

\(G_Y(t) = E(t^{aX + b}) = t^b\,G_X(t^a).\)

6. Extracting Individual Probabilities

If you have the pgf \(G(t) = \sum p_r t^r\), expand as a power series and read off \(p_r\) as the coefficient.

Given \(G(t) = \tfrac{1}{4}(1 + t)^2\), find \(P(X = 1)\).
\(G(t) = \tfrac{1}{4}(1 + 2t + t^2) = \tfrac{1}{4} + \tfrac{1}{2}t + \tfrac{1}{4}t^2\).
Coefficient of \(t^1\): \(P(X = 1) = 1/2\).

7. Branching Processes (Contextual)

If \(X_1, X_2, \ldots\) are IID with common pgf \(G\), and we form successive generations where each individual has a random number of offspring, the pgf of generation \(n\) is \(G^{(n)} = G \circ G \circ \cdots \circ G\) (compose \(n\) times).

8. Exam Style Strategy

  • State \(G_X(1) = 1\) as a sanity check.
  • Use differentiation for mean/variance — don't go back to the pmf.
  • For sums, multiply pgfs and recognise the result as a standard distribution.
  • Keep \(t\) as a formal variable — don't substitute numerical values for it except in limits.
