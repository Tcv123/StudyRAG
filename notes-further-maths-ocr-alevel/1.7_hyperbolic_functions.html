1.7: Hyperbolic Functions

1. Definitions

\[\cosh x = \dfrac{e^x + e^{-x}}{2}, \quad \sinh x = \dfrac{e^x - e^{-x}}{2}, \quad \tanh x = \dfrac{\sinh x}{\cosh x} = \dfrac{e^x - e^{-x}}{e^x + e^{-x}}.\]

Reciprocals: \(\operatorname{sech} x = 1/\cosh x\), \(\operatorname{cosech} x = 1/\sinh x\), \(\coth x = 1/\tanh x\).

Key Values and Parity

  • \(\cosh 0 = 1\), \(\sinh 0 = 0\), \(\tanh 0 = 0\).
  • \(\cosh\) is even, \(\sinh\) and \(\tanh\) are odd.
  • \(\cosh x \ge 1\) for all real \(x\); range of \(\tanh x\) is \((-1, 1)\).
  • As \(x \to \infty\): \(\cosh x, \sinh x \to \tfrac{e^x}{2}\); \(\tanh x \to 1\).

2. Fundamental Identities

\(\cosh^2 x - \sinh^2 x = 1\)

(Hyperbolic Pythagorean identity — the minus sign is the key difference from trig.)

\(1 - \tanh^2 x = \operatorname{sech}^2 x\)
\(\coth^2 x - 1 = \operatorname{cosech}^2 x\)

Addition Formulae

\(\sinh(A \pm B) = \sinh A\cosh B \pm \cosh A \sinh B\)
\(\cosh(A \pm B) = \cosh A\cosh B \pm \sinh A \sinh B\)

Double-Angle Formulae

\(\sinh 2x = 2\sinh x \cosh x\)
\(\cosh 2x = \cosh^2 x + \sinh^2 x = 2\cosh^2 x - 1 = 1 + 2\sinh^2 x\)

Osborn's Rule

To convert a trig identity to its hyperbolic analogue: keep the form but change the sign of any term that is (or contains) a product of two \(\sinh\)s.

e.g. \(\cos 2A = 1 - 2\sin^2 A\) becomes \(\cosh 2A = 1 + 2\sinh^2 A\) (sign flipped because of \(\sin^2 A\)).

3. Derivatives

\(\dfrac{d}{dx}\sinh x = \cosh x\),   \(\dfrac{d}{dx}\cosh x = \sinh x\)
\(\dfrac{d}{dx}\tanh x = \operatorname{sech}^2 x\)

(The derivative of \(\cosh\) is \(+\sinh\) — no minus sign, unlike \(\cos\).)

4. Integrals

\(\int \cosh x\,dx = \sinh x + C\),  \(\int \sinh x\,dx = \cosh x + C\)
\(\int \operatorname{sech}^2 x\,dx = \tanh x + C\)

For products like \(\int\sinh^2 x\,dx\), use the double-angle identity:

\(\sinh^2 x = \tfrac{1}{2}(\cosh 2x - 1), \qquad \cosh^2 x = \tfrac{1}{2}(\cosh 2x + 1).\)

5. Inverse Hyperbolic Functions

Logarithmic Forms

\(\operatorname{arsinh} x = \ln\!\left(x + \sqrt{x^2 + 1}\right), \quad x \in \mathbb{R}\)
\(\operatorname{arcosh} x = \ln\!\left(x + \sqrt{x^2 - 1}\right), \quad x \ge 1\)
\(\operatorname{artanh} x = \tfrac{1}{2}\ln\!\left(\dfrac{1 + x}{1 - x}\right), \quad |x| < 1\)

Derivation of arsinh

If \(y = \operatorname{arsinh} x\), then \(x = \sinh y = \tfrac{1}{2}(e^y - e^{-y})\). Let \(u = e^y\): \(u^2 - 2xu - 1 = 0 \Rightarrow u = x + \sqrt{x^2 + 1}\) (positive root). Take logs.

Derivatives of Inverses

\(\dfrac{d}{dx}\operatorname{arsinh} x = \dfrac{1}{\sqrt{x^2 + 1}}\)
\(\dfrac{d}{dx}\operatorname{arcosh} x = \dfrac{1}{\sqrt{x^2 - 1}}\)
\(\dfrac{d}{dx}\operatorname{artanh} x = \dfrac{1}{1 - x^2}\)

7. Solving Equations with Hyperbolic Functions

Solve \(2\cosh x - \sinh x = 3\).
Sub \(\cosh x = \tfrac{1}{2}(e^x + e^{-x})\), \(\sinh x = \tfrac{1}{2}(e^x - e^{-x})\):
\(2\cdot\tfrac{1}{2}(e^x + e^{-x}) - \tfrac{1}{2}(e^x - e^{-x}) = 3\)
\(\tfrac{1}{2}e^x + \tfrac{3}{2}e^{-x} = 3\). Multiply by \(2e^x\): \(e^{2x} + 3 = 6e^x \Rightarrow e^{2x} - 6e^x + 3 = 0\).
\(e^x = 3 \pm \sqrt{6}\). Both positive, so \(x = \ln(3 \pm \sqrt{6})\).

8. Exam Style Strategy

  • Convert to exponentials early for solving equations.
  • Use Osborn's rule to translate trig identities — but always double-check by direct expansion.
  • Hyperbolic substitutions often make "root of \(x^2 + a^2\)" integrals cleaner than trig substitutions.
  • Know when arcosh gives \(\pm\) — the logarithmic form picks the positive branch; think about whether both signs are wanted.
