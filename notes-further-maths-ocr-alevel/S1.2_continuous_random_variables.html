OCR A-Level Further Mathematics (H245) — Statistics Option Section S1.2

S1.2: Continuous Random Variables

1. Probability Density Function (PDF)

A continuous random variable \(X\) has a probability density function \(f(x)\) satisfying
  • \(f(x) \ge 0\) for all \(x\),
  • \(\displaystyle \int_{-\infty}^{\infty} f(x)\,dx = 1\),
  • \(\displaystyle P(a \le X \le b) = \int_a^b f(x)\,dx\).

For continuous \(X\), \(P(X = x) = 0\) exactly — only intervals have non-zero probability.

\(X\) has pdf \(f(x) = kx\) for \(0 \le x \le 2\), 0 otherwise. Find \(k\).
\(\int_0^2 kx\,dx = 2k = 1 \Rightarrow k = 1/2\).

2. Cumulative Distribution Function (CDF)

\(F(x) = P(X \le x) = \displaystyle \int_{-\infty}^{x} f(t)\,dt.\)

Properties: non-decreasing, \(F(-\infty) = 0\), \(F(\infty) = 1\), and \(f = F'\) (where differentiable).

Piecewise PDFs and CDFs

If \(f(x)\) is given piecewise, compute \(F(x)\) in each piece, matching values at the break points to keep \(F\) continuous.

For \(f(x) = \tfrac{x}{2}\) on \([0, 2]\), find \(F(x)\).
\(F(x) = \int_0^x \tfrac{t}{2}\,dt = \tfrac{x^2}{4}\) for \(0 \le x \le 2\). \(F(x) = 0\) below 0, \(F(x) = 1\) above 2.

3. Mean, Variance, Mode, Median

\(\mu = E(X) = \int_{-\infty}^{\infty} x f(x)\,dx\)
\(E[g(X)] = \int_{-\infty}^{\infty} g(x) f(x)\,dx\)
\(\operatorname{Var}(X) = E(X^2) - \mu^2 = \int_{-\infty}^{\infty} x^2 f(x)\,dx - \mu^2\)

Median \(m\)

Defined by \(F(m) = \tfrac{1}{2}\). Solve the equation involving the CDF.

Mode

Value of \(x\) where \(f(x)\) is maximum (stationary point or endpoint).

For \(f(x) = \tfrac{x}{2}\) on \([0, 2]\):
\(\mu = \int_0^2 x\cdot\tfrac{x}{2}\,dx = \tfrac{1}{2}\cdot\tfrac{8}{3} = \tfrac{4}{3}\).
\(E(X^2) = \int_0^2 x^2 \cdot \tfrac{x}{2}\,dx = \tfrac{1}{2}\cdot 4 = 2\). \(\operatorname{Var}(X) = 2 - \tfrac{16}{9} = \tfrac{2}{9}\).
Median: \(F(m) = \tfrac{m^2}{4} = \tfrac{1}{2} \Rightarrow m = \sqrt{2}\).
Mode: \(f\) is increasing on \([0, 2]\), so mode \(= 2\) (endpoint).

4. Linear Transformations

\(E(aX + b) = aE(X) + b, \quad \operatorname{Var}(aX + b) = a^2 \operatorname{Var}(X).\)

CDF of a Transformed Variable

If \(Y = aX + b\) with \(a > 0\), then

\(F_Y(y) = P(Y \le y) = P(X \le (y - b)/a) = F_X\!\left(\dfrac{y - b}{a}\right).\)

Differentiate to get the pdf of \(Y\).

5. Rectangular (Uniform) Distribution

\(X \sim U(a, b)\) has pdf \(f(x) = \dfrac{1}{b - a}\) for \(a \le x \le b\).

  • \(E(X) = \dfrac{a + b}{2}\)
  • \(\operatorname{Var}(X) = \dfrac{(b - a)^2}{12}\)
  • \(F(x) = \dfrac{x - a}{b - a}\) on the support.

6. Function of a Continuous Random Variable

To find the distribution of \(Y = g(X)\):

  1. Write \(F_Y(y) = P(g(X) \le y)\).
  2. Manipulate the inequality to get it in terms of \(X\).
  3. Express in terms of \(F_X\).
  4. Differentiate to get \(f_Y(y)\).
\(X \sim U(0, 1)\). Find the pdf of \(Y = X^2\).
\(F_Y(y) = P(X^2 \le y) = P(X \le \sqrt{y}) = \sqrt{y}\) for \(0 \le y \le 1\).
Differentiate: \(f_Y(y) = \dfrac{1}{2\sqrt{y}}\) for \(0 < y \le 1\).

7. Exam Style Strategy

  • Always check the pdf integrates to 1 before anything else.
  • For medians of piecewise distributions, figure out which piece the median lies in first (which interval crosses \(F = 0.5\)).
  • Express answers exactly when possible; decimal approximations can mask errors.
  • When computing \(\operatorname{Var}\), use \(E(X^2) - \mu^2\), not the definitional integral of \((x - \mu)^2\).
