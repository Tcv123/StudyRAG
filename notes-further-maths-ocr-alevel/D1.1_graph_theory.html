D1.1: Graph Theory

1. Basic Definitions

A graph \(G = (V, E)\) consists of a set \(V\) of vertices (or nodes) and a set \(E\) of edges, each joining two vertices.
TermMeaning
Simple graphNo loops, no multiple edges
Order of a graphNumber of vertices \(|V|\)
Size of a graphNumber of edges \(|E|\)
Degree of a vertexNumber of edges incident with it (a loop counts twice)
Regular graphAll vertices have the same degree
Complete graph \(K_n\)Every pair of vertices joined; \(\binom{n}{2}\) edges
Bipartite graphVertices partition into two sets with edges only between sets
Complete bipartite \(K_{m,n}\)Every vertex in one set joined to every vertex in the other
SubgraphGraph formed from a subset of vertices and a subset of their edges

Handshake Lemma

\(\sum_{v \in V} \deg(v) = 2|E|\)

Sum of degrees is always even → any graph has an even number of odd-degree vertices.

2. Walks, Trails, Paths, Cycles

  • Walk: sequence of vertices with consecutive vertices joined by an edge.
  • Trail: walk with no repeated edges.
  • Path: walk with no repeated vertices.
  • Closed walk / trail / cycle: starts and ends at the same vertex.
  • Cycle: closed path (vertices distinct except first = last).

Connectedness

A graph is connected if there is a path between every pair of vertices. Maximal connected subgraphs are components.

3. Eulerian Graphs

An Eulerian trail uses every edge exactly once; an Eulerian circuit is a closed Eulerian trail.
Number of odd-degree verticesEulerian property (connected graph)
0Eulerian circuit exists
2Eulerian trail (start/end at odd-degree vertices)
Any other even numberNeither — graph is semi-Eulerian only if exactly 2 odd-degree

4. Hamiltonian Graphs

A Hamiltonian path visits every vertex exactly once; a Hamiltonian cycle is a closed Hamiltonian path. A graph with a Hamiltonian cycle is called Hamiltonian.

Unlike Eulerian tests, there is no simple criterion for Hamiltonicity — finding a Hamiltonian cycle is in general computationally hard (NP-complete).

5. Trees

A tree is a connected graph with no cycles. A graph with all of these equivalent properties is a tree:
  • Connected and acyclic.
  • Exactly \(n - 1\) edges for \(n\) vertices, and connected.
  • There is a unique path between any pair of vertices.

A forest is a graph whose components are all trees. A spanning tree of a connected graph \(G\) is a subgraph that is a tree and includes every vertex of \(G\).

6. Planar Graphs

A graph is planar if it can be drawn in the plane with no edges crossing.

Euler's Formula (Planar Connected Graph)

\(V - E + F = 2,\)

where \(F\) is the number of faces (including the unbounded outer face).

Non-Planar Graphs

\(K_5\) (complete on 5 vertices) and \(K_{3,3}\) are the smallest non-planar graphs. By Kuratowski's theorem, a graph is non-planar iff it contains a subgraph homeomorphic to \(K_5\) or \(K_{3,3}\).

7. Isomorphism

Two graphs are isomorphic if there is a one-to-one correspondence between their vertex sets that preserves adjacency.

Invariants: order, size, degree sequence, number of components, presence of cycles of particular lengths.

OCR Exam Tip: To show non-isomorphism, cite an invariant that differs (e.g. different degree sequences). To show isomorphism, give an explicit bijection.

8. Matrix Representation

Adjacency Matrix

\(A_{ij} = 1\) if there's an edge between vertex \(i\) and vertex \(j\), else 0 (for a simple graph). For multigraphs, \(A_{ij}\) = number of edges.

\((\mathbf{A}^n)_{ij}\) counts the number of walks of length \(n\) from \(i\) to \(j\).

9. Exam Style Strategy

  • Always check degrees first — Eulerian and handshake questions turn on them.
  • State definitions precisely — examiners reward correct terminology (walk vs. trail vs. path).
  • For planarity, apply \(V - E + F = 2\) and the inequality \(E \le 3V - 6\) (simple, connected, planar).
  • For isomorphism proofs, present the bijection as a table or labelled diagram.
