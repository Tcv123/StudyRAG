Measures of Central Tendency and Spread

Mean

$$\bar{x} = \frac{\sum x}{n} \quad \text{or} \quad \bar{x} = \frac{\sum fx}{\sum f} \text{ (grouped data)}$$

Variance and Standard Deviation

$$\text{Var}(X) = \sigma^2 = \frac{\sum x^2}{n} - \bar{x}^2 = \frac{\sum(x - \bar{x})^2}{n}$$
$$\text{Standard deviation} = \sigma = \sqrt{\text{Var}(X)}$$

Coding

Coding: If \(y = \dfrac{x - a}{b}\), then: $$\bar{x} = b\bar{y} + a \qquad \sigma_x = b\sigma_y$$ Coding simplifies calculations but preserves the structure of the data.

Worked Example: Using Coding

Data values are coded using \(y = \dfrac{x - 50}{10}\). The coded data has \(\bar{y} = 2.4\) and \(\sigma_y = 1.2\). Find \(\bar{x}\) and \(\sigma_x\).

Solution:

$$\bar{x} = 10(2.4) + 50 = 74 \qquad \sigma_x = 10(1.2) = 12$$

Data Presentation

Outliers are typically defined as values below \(Q_1 - 1.5 \times \text{IQR}\) or above \(Q_3 + 1.5 \times \text{IQR}\), where \(\text{IQR} = Q_3 - Q_1\).

Probability

Key Probability Rules:
  • Addition (OR): \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)
  • Mutually exclusive: \(P(A \cap B) = 0\), so \(P(A \cup B) = P(A) + P(B)\)
  • Independent: \(P(A \cap B) = P(A) \times P(B)\)
  • Conditional: \(P(A|B) = \dfrac{P(A \cap B)}{P(B)}\)
  • Complement: \(P(A') = 1 - P(A)\)

Worked Example: Conditional Probability

In a class, 60% study Maths (M), 40% study Physics (P), and 25% study both. Find \(P(M|P)\).

Solution:

$$P(M|P) = \frac{P(M \cap P)}{P(P)} = \frac{0.25}{0.40} = 0.625$$

So 62.5% of Physics students also study Maths.

Statistical Distributions

Binomial Distribution

Binomial Distribution: If \(X \sim B(n, p)\) where \(n\) is the number of independent trials and \(p\) is the probability of success: $$P(X = r) = \binom{n}{r} p^r (1-p)^{n-r}$$ $$E(X) = np \qquad \text{Var}(X) = np(1-p)$$

Worked Example: Binomial

A fair coin is tossed 8 times. Find the probability of getting exactly 5 heads.

Solution: \(X \sim B(8, 0.5)\):

$$P(X = 5) = \binom{8}{5} (0.5)^5 (0.5)^3 = 56 \times (0.5)^8 = \frac{56}{256} = 0.219$$

Normal Distribution

Normal Distribution: If \(X \sim N(\mu, \sigma^2)\), the distribution is symmetric about the mean \(\mu\) with standard deviation \(\sigma\). Standardise using: $$Z = \frac{X - \mu}{\sigma} \sim N(0, 1)$$

Worked Example: Normal Distribution

The heights of students are normally distributed with mean 170 cm and standard deviation 8 cm. Find the probability a student is taller than 182 cm.

Solution: \(X \sim N(170, 64)\).

$$P(X > 182) = P\!\left(Z > \frac{182 - 170}{8}\right) = P(Z > 1.5)$$
$$= 1 - P(Z \leq 1.5) = 1 - 0.9332 = 0.0668$$

Hypothesis Testing

Hypothesis Testing Steps:
  1. State \(H_0\) (null hypothesis) and \(H_1\) (alternative hypothesis)
  2. Choose the significance level \(\alpha\) (usually 5% or 1%)
  3. Calculate the test statistic
  4. Compare with the critical value or find the p-value
  5. Conclude: reject or do not reject \(H_0\)

Binomial Hypothesis Test

Used to test the parameter \(p\) of a binomial distribution.

Worked Example: Binomial Hypothesis Test

A coin is suspected of being biased towards heads. In 20 tosses, 14 heads are obtained. Test at the 5% significance level.

Solution:

\(H_0: p = 0.5\) (fair coin), \(H_1: p > 0.5\) (biased towards heads). One-tailed test.

Under \(H_0\), \(X \sim B(20, 0.5)\).

$$P(X \geq 14) = 1 - P(X \leq 13) = 1 - 0.9423 = 0.0577$$

Since \(0.0577 > 0.05\), we do not reject \(H_0\). There is insufficient evidence that the coin is biased.

Normal Hypothesis Test

Used to test the mean \(\mu\) of a normal distribution when \(\sigma\) is known.

$$Z = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}}$$
Exam Tip: Always state your hypotheses clearly using correct notation, state the significance level, show your working, and write a conclusion in context. Saying "reject \(H_0\)" alone is not sufficient -- you must interpret the result in the context of the question.
Exam Tip: "Do not reject \(H_0\)" is NOT the same as "accept \(H_0\)". We never "prove" the null hypothesis is true; we only conclude there is insufficient evidence to reject it.
