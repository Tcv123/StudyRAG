1.2 Algebra & Functions

Laws of Indices

Indices (powers) follow a set of rules that allow us to simplify expressions involving powers.
RuleLawExample
Multiplication\(a^m \times a^n = a^{m+n}\)\(x^3 \times x^4 = x^7\)
Division\(a^m \div a^n = a^{m-n}\)\(x^5 \div x^2 = x^3\)
Power of a power\((a^m)^n = a^{mn}\)\((x^2)^3 = x^6\)
Zero index\(a^0 = 1\)\(5^0 = 1\)
Negative index\(a^{-n} = \dfrac{1}{a^n}\)\(x^{-2} = \dfrac{1}{x^2}\)
Fractional index\(a^{1/n} = \sqrt[n]{a}\)\(8^{1/3} = 2\)
General fractional\(a^{m/n} = (\sqrt[n]{a})^m\)\(27^{2/3} = (\sqrt[3]{27})^2 = 9\)

Example: Simplify \(\ \dfrac{3x^{5/2} \times 2x^{-1}}{x^{1/2}}\)

Numerator: \(3x^{5/2} \times 2x^{-1} = 6x^{5/2 - 1} = 6x^{3/2}\)

Divide by \(x^{1/2}\): \(6x^{3/2 - 1/2} = 6x\)

Surds

A surd is an irrational root that cannot be simplified to a rational number, e.g. \(\sqrt{2}, \sqrt{5}, \sqrt[3]{7}\).
RuleFormulaExample
Multiplication\(\sqrt{a} \times \sqrt{b} = \sqrt{ab}\)\(\sqrt{3}\times\sqrt{12} = \sqrt{36} = 6\)
Division\(\dfrac{\sqrt{a}}{\sqrt{b}} = \sqrt{\dfrac{a}{b}}\)\(\dfrac{\sqrt{20}}{\sqrt{5}} = \sqrt{4} = 2\)
Simplification\(\sqrt{a^2 b} = a\sqrt{b}\)\(\sqrt{50} = 5\sqrt{2}\)
Rationalise (simple)\(\dfrac{1}{\sqrt{a}} = \dfrac{\sqrt{a}}{a}\)\(\dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}\)
Rationalise (compound)Multiply by conjugate \(\dfrac{a \mp \sqrt{b}}{a \mp \sqrt{b}}\)\(\dfrac{1}{3+\sqrt{2}} = \dfrac{3-\sqrt{2}}{7}\)

Example: Rationalise \(\dfrac{5}{2 + \sqrt{3}}\)

Multiply top and bottom by the conjugate \(2 - \sqrt{3}\):

$$\frac{5}{2 + \sqrt{3}} \times \frac{2 - \sqrt{3}}{2 - \sqrt{3}} = \frac{5(2 - \sqrt{3})}{4 - 3} = \frac{10 - 5\sqrt{3}}{1} = 10 - 5\sqrt{3}$$

Quadratics

A quadratic expression has the form \(ax^2 + bx + c\) where \(a \neq 0\). The graph is a parabola.

The Quadratic Formula

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

The Discriminant

The discriminant \(\Delta = b^2 - 4ac\) determines the nature of the roots of a quadratic equation.
DiscriminantNature of RootsGraph
\(b^2 - 4ac > 0\)Two distinct real rootsParabola crosses x-axis twice
\(b^2 - 4ac = 0\)One repeated real rootParabola touches x-axis once
\(b^2 - 4ac < 0\)No real roots (complex)Parabola does not cross x-axis

Completing the Square

$$ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2}{4a} + c$$

Example: Write \(2x^2 + 12x + 7\) in completed square form.

Factor out 2 from the \(x\) terms: \(2(x^2 + 6x) + 7\)

Complete the square inside: \(2\big((x + 3)^2 - 9\big) + 7\)

Expand: \(2(x+3)^2 - 18 + 7 = 2(x+3)^2 - 11\)

Vertex is at \((-3, -11)\). Minimum value is \(-11\).

Exam Tip: Completing the square tells you the vertex (turning point) of the parabola. If the question asks for the minimum/maximum value, this is the method to use. The vertex is \(\left(-\dfrac{b}{2a},\ c - \dfrac{b^2}{4a}\right)\).

Simultaneous Equations

Linear-Linear

Use elimination (add/subtract equations) or substitution.

Linear-Quadratic

Substitute the linear equation into the quadratic and solve the resulting quadratic.

Example: Solve \(y = x + 3\) and \(x^2 + y^2 = 29\).

Substitute: \(x^2 + (x+3)^2 = 29\)

\(x^2 + x^2 + 6x + 9 = 29\)

\(2x^2 + 6x - 20 = 0 \Rightarrow x^2 + 3x - 10 = 0\)

\((x+5)(x-2) = 0 \Rightarrow x = -5\) or \(x = 2\)

When \(x = -5\): \(y = -2\). When \(x = 2\): \(y = 5\).

Solutions: \((-5, -2)\) and \((2, 5)\).

Inequalities

Linear Inequalities

Solve like equations. Reverse the inequality sign when multiplying/dividing by a negative number.

Quadratic Inequalities

  1. Solve the corresponding equation \(ax^2 + bx + c = 0\) to find critical values.
  2. Sketch the parabola.
  3. Read off the solution from the graph.

Example: Solve \(x^2 - 5x + 6 < 0\).

Factorise: \((x-2)(x-3) = 0\), so roots at \(x = 2\) and \(x = 3\).

The parabola is U-shaped (positive \(x^2\) coefficient). It is below the x-axis between the roots.

Solution: \(2 < x < 3\).

Exam Tip: Always sketch the quadratic to decide the correct region. For \(> 0\) with a U-shaped parabola, the answer is two separate regions. For \(< 0\), the answer is between the roots.

Polynomials

Factor Theorem

If \(f(a) = 0\), then \((x - a)\) is a factor of \(f(x)\).
Conversely, if \((x - a)\) is a factor of \(f(x)\), then \(f(a) = 0\).

Example: Show that \((x - 2)\) is a factor of \(f(x) = x^3 - 6x^2 + 11x - 6\).

\(f(2) = 8 - 24 + 22 - 6 = 0\). Since \(f(2) = 0\), \((x-2)\) is a factor. Then:

\(x^3 - 6x^2 + 11x - 6 = (x-2)(x^2 - 4x + 3) = (x-1)(x-2)(x-3)\)

Algebraic Division

Use polynomial long division or compare coefficients to divide a polynomial by a linear or quadratic factor.

Example: Divide \(2x^3 + 3x^2 - 8x + 3\) by \((x + 3)\).

Using long division or inspection:

$$2x^3 + 3x^2 - 8x + 3 = (x + 3)(2x^2 - 3x + 1) = (x+3)(2x-1)(x-1)$$

Graphs of Functions

You need to recognise and sketch the following key graphs:

  • Linear: \(y = mx + c\)
  • Quadratic: \(y = ax^2 + bx + c\)
  • Cubic: \(y = ax^3 + \ldots\)
  • Reciprocal: \(y = \dfrac{a}{x}\) and \(y = \dfrac{a}{x^2}\)
  • Exponential: \(y = a^x\)
  • Trigonometric: \(y = \sin x, \cos x, \tan x\)

Exam Tip: Remember the counter-intuitive rule: changes inside the bracket (to \(x\)) do the opposite of what you expect. \(f(x - 3)\) shifts right by 3, and \(f(2x)\) squashes the graph horizontally by factor \(\frac{1}{2}\).

Partial Fractions

Partial fractions decompose a single fraction into a sum of simpler fractions. This is the reverse of adding fractions together.

Types

Denominator TypePartial Fraction Form
Linear factors: \(\dfrac{f(x)}{(ax+b)(cx+d)}\)\(\dfrac{A}{ax+b} + \dfrac{B}{cx+d}\)
Repeated factor: \(\dfrac{f(x)}{(ax+b)^2(cx+d)}\)\(\dfrac{A}{ax+b} + \dfrac{B}{(ax+b)^2} + \dfrac{C}{cx+d}\)
Improper (degree num \(\geq\) denom)Divide first, then decompose the remainder

Example: Express \(\dfrac{5x + 1}{(x-1)(x+2)}\) in partial fractions.

Let \(\dfrac{5x+1}{(x-1)(x+2)} = \dfrac{A}{x-1} + \dfrac{B}{x+2}\)

Multiply through: \(5x + 1 = A(x+2) + B(x-1)\)

Set \(x = 1\): \(6 = 3A \Rightarrow A = 2\)

Set \(x = -2\): \(-9 = -3B \Rightarrow B = 3\)

$$\frac{5x+1}{(x-1)(x+2)} = \frac{2}{x-1} + \frac{3}{x+2}$$

Modulus Function

The modulus (absolute value) of \(x\), written \(|x|\), gives the magnitude of \(x\) without regard to sign.
$$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$

Graph of \(y = |f(x)|\)

Reflect any part of \(y = f(x)\) that is below the x-axis in the x-axis (make negative y-values positive).

Graph of \(y = f(|x|)\)

Keep the graph for \(x \geq 0\) and reflect it in the y-axis (the graph becomes symmetric about the y-axis).

Solving Modulus Equations

Example: Solve \(|2x - 3| = 5\).

Case 1: \(2x - 3 = 5 \Rightarrow 2x = 8 \Rightarrow x = 4\)

Case 2: \(2x - 3 = -5 \Rightarrow 2x = -2 \Rightarrow x = -1\)

Solutions: \(x = 4\) or \(x = -1\).

Example: Solve \(|x + 1| = |2x - 3|\).

Case 1: \(x + 1 = 2x - 3 \Rightarrow 4 = x \Rightarrow x = 4\)

Case 2: \(x + 1 = -(2x - 3) \Rightarrow x + 1 = -2x + 3 \Rightarrow 3x = 2 \Rightarrow x = \frac{2}{3}\)

Solutions: \(x = 4\) or \(x = \frac{2}{3}\).

Exam Tip: When solving \(|f(x)| = |g(x)|\), you can also square both sides: \(f(x)^2 = g(x)^2\), which avoids case work. For modulus inequalities like \(|f(x)| < a\), rewrite as \(-a < f(x) < a\).
