1.10: Proof by Induction

1. The Principle of Mathematical Induction

To prove a statement \(P(n)\) holds for all integers \(n \ge n_0\):
  1. Base case: show \(P(n_0)\) is true.
  2. Inductive step: assume \(P(k)\) is true for some \(k \ge n_0\), and prove \(P(k+1)\) follows.
  3. Conclusion: by the principle of induction, \(P(n)\) holds for all \(n \ge n_0\).

Think of it as toppling a line of dominoes: (1) push the first one, (2) every domino knocks over the next.

2. Structure and Marking

The OCR mark scheme usually awards marks for each of:

  • Establishing the base case (\(n = 1\) or equivalent).
  • Assuming the result for \(n = k\) (state this explicitly).
  • Correctly manipulating the \(n = k+1\) expression.
  • Writing a clear concluding sentence.
OCR Exam Tip: The concluding sentence is worth one mark. Aim for something like: "Since \(P(1)\) is true, and \(P(k) \Rightarrow P(k+1)\), by the principle of induction \(P(n)\) is true for all \(n \ge 1\)."

3. Induction with Series

Prove that \(\displaystyle \sum_{r=1}^{n} r^2 = \tfrac{1}{6}n(n+1)(2n+1)\) for all \(n \ge 1\).

Base \(n=1\): LHS = 1. RHS = \(\tfrac{1}{6}(1)(2)(3) = 1\). Ôťô

Assume \(n = k\): \(\sum_{r=1}^{k} r^2 = \tfrac{1}{6}k(k+1)(2k+1)\).

Show \(n = k+1\):
\(\sum_{r=1}^{k+1} r^2 = \tfrac{1}{6}k(k+1)(2k+1) + (k+1)^2\)
\(= \tfrac{1}{6}(k+1)\big[k(2k+1) + 6(k+1)\big]\)
\(= \tfrac{1}{6}(k+1)(2k^2 + 7k + 6) = \tfrac{1}{6}(k+1)(k+2)(2k+3)\).
This is the RHS with \(n = k+1\). Ôťô

Conclusion: by induction, the result holds for all \(n \ge 1\). ÔłÄ

4. Induction with Divisibility

To show \(f(n)\) is divisible by \(d\), write

\(f(k+1) = f(k) + \text{(something obviously divisible by } d\text{)}\).
Prove \(4^n + 5^n - 1\) is divisible by 4 for all \(n \ge 1\).
Base \(n = 1\): \(4 + 5 - 1 = 8 = 4\cdot 2\). Ôťô
Assume \(4^k + 5^k - 1 = 4m\) for some integer \(m\).
Consider \(4^{k+1} + 5^{k+1} - 1 = 4\cdot 4^k + 5\cdot 5^k - 1\).
Write \(5\cdot 5^k = 5^k + 4\cdot 5^k\):
\(= 4\cdot 4^k + 5^k + 4\cdot 5^k - 1\)
\(= (4^k + 5^k - 1) + 3\cdot 4^k + 4\cdot 5^k\)
\(= 4m + 3\cdot 4^k + 4\cdot 5^k\)
The last two terms: \(3\cdot 4^k = 4(3\cdot 4^{k-1})\), divisible by 4 for \(k \ge 1\). And \(4\cdot 5^k\) is obviously divisible by 4.
So \(4^{k+1} + 5^{k+1} - 1\) is divisible by 4. Conclusion: result holds for all \(n \ge 1\). ÔłÄ

5. Induction with Matrices

Prove that \(\begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}^n = \begin{pmatrix} 1 & n \\ 0 & 1 \end{pmatrix}\) for all \(n \ge 1\).
Base \(n=1\): matches directly. Ôťô
Assume \(n = k\): \(\mathbf{M}^k = \begin{pmatrix} 1 & k \\ 0 & 1 \end{pmatrix}\).
Inductive step: \(\mathbf{M}^{k+1} = \mathbf{M}^k \mathbf{M} = \begin{pmatrix} 1 & k \\ 0 & 1 \end{pmatrix}\begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix} = \begin{pmatrix} 1 & k+1 \\ 0 & 1 \end{pmatrix}\). Ôťô
Conclusion: the result holds for all \(n \ge 1\). ÔłÄ

6. Induction with de Moivre

The inductive proof of \((\cos\theta + i\sin\theta)^n = \cos n\theta + i\sin n\theta\) for positive integers \(n\) is classical:

  • Base \(n = 1\) trivial.
  • Assume for \(n = k\). Multiply both sides by \((\cos\theta + i\sin\theta)\) and use the compound-angle formulae to simplify to \(\cos(k+1)\theta + i\sin(k+1)\theta\).

7. Induction with Inequalities

Prove \(2^n > n^2\) for all \(n \ge 5\).
Base \(n = 5\): \(32 > 25\). Ôťô
Assume \(2^k > k^2\) for some \(k \ge 5\).
Then \(2^{k+1} = 2\cdot 2^k > 2k^2\). We want to show \(2k^2 \ge (k+1)^2\).
\(2k^2 - (k+1)^2 = 2k^2 - k^2 - 2k - 1 = k^2 - 2k - 1\).
For \(k \ge 5\): \(k^2 - 2k - 1 \ge 25 - 10 - 1 = 14 > 0\). Ôťô
So \(2^{k+1} > 2k^2 \ge (k+1)^2\). Conclusion: true for all \(n \ge 5\). ÔłÄ

8. Common Mistakes

  • Forgetting the base case. The inductive step alone proves nothing.
  • Not stating the assumption clearly. Always write "assume \(P(k)\) is true".
  • Skipping the concluding sentence. It's worth a mark.
  • Bad algebra in the inductive step ÔÇö factor carefully.
  • Using what you're trying to prove ÔÇö use only the assumption \(P(k)\), not \(P(k+1)\).

9. Template to Copy

1. Base: for \(n = 1\), LHS = ÔÇŽ, RHS = ÔÇŽ, so \(P(1)\) is true.
2. Assume: \(P(k)\) is true, i.e. ÔÇŽ
3. Step: consider \(P(k+1)\). Using the assumption, ÔÇŽ = RHS with \(n = k+1\), so \(P(k+1)\) is true.
4. Conclusion: since \(P(1)\) is true and \(P(k) \Rightarrow P(k+1)\), by the principle of mathematical induction, \(P(n)\) is true for all \(n \ge 1\).
