2. Chain Rule

If \(y = f(g(x))\), let \(u = g(x)\), then: $$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$$ In words: differentiate the outside, keep the inside, times derivative of the inside.
Example 1: Differentiate \(y = \sin(3x^2 + 1)\).

Let \(u = 3x^2+1\), so \(y = \sin u\).
\(\dfrac{dy}{du} = \cos u\), \(\dfrac{du}{dx} = 6x\)
\(\dfrac{dy}{dx} = 6x\cos(3x^2+1)\)
Example 2: Differentiate \(y = e^{\sin x}\).

\(\dfrac{dy}{dx} = \cos x \cdot e^{\sin x}\)
Example 3: Differentiate \(y = \ln(x^2 + 4)\).

\(\dfrac{dy}{dx} = \dfrac{2x}{x^2+4}\)

3. Product Rule

If \(y = uv\) where \(u\) and \(v\) are functions of \(x\): $$\frac{d}{dx}(uv) = u\frac{dv}{dx} + v\frac{du}{dx}$$ Mnemonic: u dv + v du
Example: Differentiate \(y = x^2 e^{3x}\).

Let \(u = x^2\), \(v = e^{3x}\).
\(\dfrac{du}{dx} = 2x\), \(\dfrac{dv}{dx} = 3e^{3x}\)
\(\dfrac{dy}{dx} = x^2 \cdot 3e^{3x} + e^{3x} \cdot 2x = e^{3x}(3x^2 + 2x) = xe^{3x}(3x+2)\)
Example: Differentiate \(y = x\ln x\).

\(u = x,\; v = \ln x\)
\(\dfrac{dy}{dx} = x \cdot \dfrac{1}{x} + \ln x \cdot 1 = 1 + \ln x\)

4. Quotient Rule

If \(y = \dfrac{u}{v}\): $$\frac{d}{dx}\!\left(\frac{u}{v}\right) = \frac{v\dfrac{du}{dx} - u\dfrac{dv}{dx}}{v^2}$$ Mnemonic: v du minus u dv, all over v squared
Example: Differentiate \(y = \dfrac{\sin x}{x^2}\).

\(u = \sin x,\; v = x^2\)
\(\dfrac{du}{dx} = \cos x,\; \dfrac{dv}{dx} = 2x\)
\(\dfrac{dy}{dx} = \dfrac{x^2\cos x - 2x\sin x}{x^4} = \dfrac{x\cos x - 2\sin x}{x^3}\)
Quotient vs product: It is often easier to write \(\dfrac{u}{v} = u \cdot v^{-1}\) and use the product rule with chain rule, avoiding the quotient rule formula entirely.

5. Implicit Differentiation

When \(y\) is not expressed explicitly as a function of \(x\), differentiate both sides with respect to \(x\) and use the chain rule for terms involving \(y\):

$$\frac{d}{dx}[f(y)] = f'(y) \cdot \frac{dy}{dx}$$ Key results:
  • \(\dfrac{d}{dx}(y^2) = 2y\dfrac{dy}{dx}\)
  • \(\dfrac{d}{dx}(y^n) = ny^{n-1}\dfrac{dy}{dx}\)
  • \(\dfrac{d}{dx}(\sin y) = \cos y\dfrac{dy}{dx}\)
  • \(\dfrac{d}{dx}(e^y) = e^y\dfrac{dy}{dx}\)
  • \(\dfrac{d}{dx}(xy) = x\dfrac{dy}{dx} + y\) (product rule)
Example: Find \(\dfrac{dy}{dx}\) for \(x^2 + y^2 = 25\).

Differentiate both sides: \(2x + 2y\dfrac{dy}{dx} = 0\)
\(\dfrac{dy}{dx} = -\dfrac{x}{y}\)
Example: Find \(\dfrac{dy}{dx}\) for \(x^3 + y^3 - 3xy = 0\).

\(3x^2 + 3y^2\dfrac{dy}{dx} - 3\!\left(x\dfrac{dy}{dx} + y\right) = 0\)
\(3x^2 - 3y + (3y^2 - 3x)\dfrac{dy}{dx} = 0\)
\(\dfrac{dy}{dx} = \dfrac{y - x^2}{y^2 - x}\)

Second derivative implicitly

To find \(\dfrac{d^2y}{dx^2}\) implicitly, differentiate \(\dfrac{dy}{dx}\) again with respect to \(x\), substituting back for \(\dfrac{dy}{dx}\) as needed.

6. Parametric Differentiation

For \(x = f(t),\; y = g(t)\): $$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} \qquad \frac{d^2y}{dx^2} = \frac{\dfrac{d}{dt}\!\left(\dfrac{dy}{dx}\right)}{\dfrac{dx}{dt}}$$

See also Topic P17 for full treatment of parametric equations.

7. Connected Rates of Change

The chain rule links rates of change in real-world problems:

$$\frac{dA}{dt} = \frac{dA}{dr} \cdot \frac{dr}{dt} \qquad \text{(general form: chain rule in context)}$$
Example: A spherical balloon is inflated so its radius increases at \(2\) cm/s. Find the rate of increase of volume when \(r = 5\) cm.

\(V = \dfrac{4}{3}\pi r^3 \Rightarrow \dfrac{dV}{dr} = 4\pi r^2\)
\(\dfrac{dV}{dt} = \dfrac{dV}{dr} \cdot \dfrac{dr}{dt} = 4\pi(5)^2 \times 2 = 200\pi \text{ cm}^3/\text{s}\)
Example: Water flows into a cone at \(10\) cm\(^3\)/s. Given \(h = 3r\), find \(\dfrac{dr}{dt}\) when \(r = 4\) cm.

\(V = \dfrac{1}{3}\pi r^2 h = \dfrac{1}{3}\pi r^2(3r) = \pi r^3\)
\(\dfrac{dV}{dr} = 3\pi r^2\), so \(\dfrac{dr}{dt} = \dfrac{1}{3\pi r^2}\cdot\dfrac{dV}{dt} = \dfrac{10}{3\pi(16)} = \dfrac{5}{24\pi}\) cm/s
Exam tip: Write out the chain rule equation first, identify which rates are known and which is unknown, then solve. Never mix up \(\dfrac{dV}{dt}\) with \(\dfrac{dt}{dV}\).

8. Second Derivatives and Nature of Stationary Points

For a stationary point where \(f'(x) = 0\):

  • If \(f''(x) > 0\): local minimum
  • If \(f''(x) < 0\): local maximum
  • If \(f''(x) = 0\): inconclusive — use first derivative test

A point of inflection occurs where \(f''(x) = 0\) and \(f''\) changes sign. Note: \(f''(x)=0\) is necessary but not sufficient.

Function of a function, e.g. \(\sin(x^2)\)Chain rule
Product of two functions, e.g. \(x^2 e^x\)Product rule
Ratio of two functions, e.g. \(\dfrac{\sin x}{x}\)Quotient rule (or product + chain)
Equation mixing \(x\) and \(y\), e.g. \(x^2+y^2=1\)Implicit differentiation
\(x = f(t), y = g(t)\)Parametric: \(dy/dx = (dy/dt)/(dx/dt)\)
