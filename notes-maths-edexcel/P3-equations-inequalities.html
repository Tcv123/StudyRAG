P3: Equations and Inequalities

1. Simultaneous Equations

Both Linear — Elimination Method

Multiply equations to make coefficients of one variable equal, then add or subtract to eliminate it.

Example: Solve \(3x + 2y = 8\) and \(5x - y = 7\).

Multiply equation 2 by 2: \(10x - 2y = 14\)
Add to equation 1: \(13x = 22 \Rightarrow x = \dfrac{22}{13}\)
Substitute: \(y = 5 \cdot \dfrac{22}{13} - 7 = \dfrac{110-91}{13} = \dfrac{19}{13}\)

Both Linear — Substitution Method

Rearrange one equation for a variable, substitute into the other.

One Linear, One Quadratic (Common Exam Type)

Rearrange the linear equation and substitute into the quadratic. This typically gives a quadratic to solve.

Example: Solve \(y = x + 1\) and \(x^2 + y^2 = 25\).

Substitute \(y = x+1\): \(x^2 + (x+1)^2 = 25\)
\(x^2 + x^2 + 2x + 1 = 25\)
\(2x^2 + 2x - 24 = 0 \Rightarrow x^2 + x - 12 = 0\)
\((x+4)(x-3) = 0 \Rightarrow x = -4\) or \(x = 3\)
Corresponding \(y\): \(y = -3\) or \(y = 4\)
Solutions: \((-4,\,-3)\) and \((3,\,4)\)
Exam Tip: Always substitute from the linear equation into the quadratic — never the other way round, as this creates higher-degree equations. Give both \((x,y)\) pairs in your final answer.

2. Linear Inequalities

Solve like a linear equation, but reverse the inequality sign when multiplying or dividing by a negative number.

Example: Solve \(3 - 2x > 7\).

\(-2x > 4\)
Dividing by \(-2\) (flip sign): \(x < -2\)

Number line representation:

  • Open circle (\(\circ\)) for strict inequalities \(<\) or \(>\)
  • Filled circle (\(\bullet\)) for \(\leq\) or \(\geq\)

3. Quadratic Inequalities

Solve by finding roots of the corresponding equation, then using the shape of the parabola to determine the solution set.

Example: Solve \(x^2 - 4x - 5 \leq 0\).

Roots of \(x^2 - 4x - 5 = 0\): \((x-5)(x+1) = 0 \Rightarrow x = -1,\; x = 5\)
Parabola opens upward (positive \(a\)), so \(\leq 0\) between the roots.
Answer: \(-1 \leq x \leq 5\)
Sign diagram approach: divide the number line into regions using the roots and test each region:
For \((x-5)(x+1)\): regions \(x<-1\), \(-1 < x < 5\), \(x>5\)
  • \(x=-2\): \((-7)(-1) = +7 > 0\)
  • \(x=0\): \((-5)(1) = -5 < 0\) ✓ (satisfies \(\leq 0\))
  • \(x=6\): \((1)(7) = +7 > 0\)

4. Modulus (Absolute Value)

The modulus \(|x|\) gives the distance from \(x\) to zero: $$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$

Solving Modulus Equations

$$|f(x)| = a \implies f(x) = a \text{ or } f(x) = -a \quad (a > 0)$$
Example: Solve \(|2x - 3| = 5\).

Case 1: \(2x - 3 = 5 \Rightarrow x = 4\)
Case 2: \(2x - 3 = -5 \Rightarrow x = -1\)
Example: Solve \(|3x + 1| = |x - 2|\).

Case 1: \(3x+1 = x-2 \Rightarrow 2x = -3 \Rightarrow x = -\frac{3}{2}\)
Case 2: \(3x+1 = -(x-2) \Rightarrow 3x+1 = -x+2 \Rightarrow 4x = 1 \Rightarrow x = \frac{1}{4}\)

Modulus Inequalities

FormEquivalent condition
\(|f(x)| < a\)\(-a < f(x) < a\)
\(|f(x)| > a\)\(f(x) > a\) or \(f(x) < -a\)
\(|f(x)| \leq a\)\(-a \leq f(x) \leq a\)
\(|f(x)| \geq a\)\(f(x) \geq a\) or \(f(x) \leq -a\)
Example: Solve \(|2x - 1| < 7\).

\(-7 < 2x - 1 < 7\)
\(-6 < 2x < 8\)
\(-3 < x < 4\)
