2.2 Homogeneity of Equations

Homogeneous equation: An equation is homogeneous if every term has the same SI base units. This is a necessary (but not sufficient) condition for an equation to be correct.

To check homogeneity:

  1. Write out the SI base units of each term in the equation
  2. Simplify each term
  3. Check that all terms have identical base units

Example: Check v2 = u2 + 2as

  • v2: (m s−1)2 = m2 s−2
  • u2: (m s−1)2 = m2 s−2
  • 2as: (m s−2)(m) = m2 s−2

All terms have units m2 s−2, so the equation is homogeneous.

Exam Tip: Homogeneity cannot verify dimensionless constants (like the "2" in 2as) or confirm that an equation is physically correct — only that the units are consistent.

Adding Vectors

Vectors are added using the triangle rule or parallelogram rule.

Triangle Rule:

  1. Draw the first vector as an arrow (to scale)
  2. Draw the second vector starting from the tip of the first
  3. The resultant vector goes from the start of the first to the tip of the second

Parallelogram Rule:

  1. Draw both vectors starting from the same point (to scale)
  2. Complete the parallelogram
  3. The resultant is the diagonal from the starting point

Special cases for two vectors A and B:

  • Same direction: resultant = A + B
  • Opposite directions: resultant = |A − B|
  • At right angles: resultant = √(A2 + B2), angle θ = tan−1(B/A)
  • At angle θ between them: use the cosine rule or scale drawing

Resolving Vectors into Components

Any vector can be resolved into two perpendicular components. For a vector F at angle θ to the horizontal:

Fx = F cos θ     (horizontal component)
Fy = F sin θ     (vertical component)
  • The component adjacent to the angle uses cosine
  • The component opposite to the angle uses sine
  • The original vector can be recovered: F = √(Fx2 + Fy2)
  • The angle can be recovered: θ = tan−1(Fy / Fx)
Exam Tip: When resolving vectors, always draw a clear diagram showing the angle and label both components. A common mistake is to swap sine and cosine — always check which component is adjacent and which is opposite to the given angle.

Equilibrium

An object is in equilibrium when the resultant force is zero. This means:

  • The sum of all horizontal components = 0
  • The sum of all vertical components = 0
  • For three forces in equilibrium, they form a closed triangle when drawn tip-to-tail
Coplanar forces: Forces that all act in the same plane. For equilibrium of coplanar forces, resolve in two perpendicular directions and set each sum to zero.
