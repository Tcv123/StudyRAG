CP1.5: Volumes of Revolution

1. The Core Idea

When a region in the \(xy\)-plane is rotated through \(2\pi\) (360°) about the \(x\)-axis or \(y\)-axis, it sweeps out a solid of revolution. The volume of this solid can be found by integration.

Intuitive Derivation (Disc Method)

Imagine slicing the solid perpendicular to the axis of rotation. Each thin slice is approximately a disc (circle) with:

  • Radius \(y\) (when rotating about the \(x\)-axis)
  • Thickness \(\delta x\)
  • Volume \(\approx \pi y^2 \, \delta x\)

Summing all discs and taking the limit gives the integral formula.

2. Volume About the x-axis

Volume of revolution about the x-axis: $$V = \pi \int_a^b y^2 \, dx$$ where \(y = f(x)\) and the region is bounded by \(x = a\) and \(x = b\).
Exam tip: The formula involves \(y^2\), not \(y\). Never forget the \(\pi\) factor and the squaring. Also, make sure your limits are correct — the formula integrates with respect to \(x\), so the limits are \(x\)-values.
Example 1: Find the volume generated when \(y = x^2 + 1\) is rotated \(360°\) about the \(x\)-axis between \(x=0\) and \(x=2\).

$$V = \pi\int_0^2 (x^2+1)^2 \, dx = \pi\int_0^2 (x^4 + 2x^2 + 1) \, dx$$ $$= \pi\left[\frac{x^5}{5} + \frac{2x^3}{3} + x\right]_0^2$$ $$= \pi\left[\frac{32}{5} + \frac{16}{3} + 2 - 0\right]$$ $$= \pi\left[\frac{96}{15} + \frac{80}{15} + \frac{30}{15}\right] = \frac{206\pi}{15}$$
Example 2 (Exponential): Find the volume generated when \(y = e^x\) is rotated about the \(x\)-axis between \(x=0\) and \(x=1\).

$$V = \pi\int_0^1 e^{2x} \, dx = \pi\left[\frac{e^{2x}}{2}\right]_0^1 = \pi\left[\frac{e^2}{2} - \frac{1}{2}\right] = \frac{\pi(e^2-1)}{2}$$
Example 3 (Trigonometric): Find the volume generated when \(y = \sin x\) is rotated about the \(x\)-axis between \(x=0\) and \(x=\pi\).

$$V = \pi\int_0^\pi \sin^2 x \, dx$$ Use the identity: \(\sin^2 x = \frac{1-\cos 2x}{2}\)
$$= \pi\int_0^\pi \frac{1-\cos 2x}{2} \, dx = \frac{\pi}{2}\left[x - \frac{\sin 2x}{2}\right]_0^\pi$$ $$= \frac{\pi}{2}\left[\pi - 0 - 0 + 0\right] = \frac{\pi^2}{2}$$

3. Volume About the y-axis

Volume of revolution about the y-axis: $$V = \pi \int_c^d x^2 \, dy$$ where \(x = g(y)\) and the region is bounded by \(y = c\) and \(y = d\). You need \(x\) expressed as a function of \(y\).
Example: Find the volume generated when \(y = x^3\) is rotated \(360°\) about the \(y\)-axis between \(y=0\) and \(y=8\).

Rearrange: \(x = y^{1/3}\), so \(x^2 = y^{2/3}\).

$$V = \pi\int_0^8 y^{2/3} \, dy = \pi\left[\frac{y^{5/3}}{5/3}\right]_0^8 = \pi\left[\frac{3y^{5/3}}{5}\right]_0^8$$ $$= \pi\left[\frac{3 \cdot 32}{5} - 0\right] = \frac{96\pi}{5}$$

4. Region Between Two Curves

When the region between two curves \(y = f(x)\) (outer) and \(y = g(x)\) (inner) is rotated about the \(x\)-axis:

$$V = \pi\int_a^b \left[f(x)^2 - g(x)^2\right] dx$$ Washer method: subtract the volume of the inner solid (hole) from the outer solid.
Exam tip: The washer formula is \(\pi\int(y_{\text{outer}}^2 - y_{\text{inner}}^2)\,dx\), NOT \(\pi\int(y_{\text{outer}} - y_{\text{inner}})^2\,dx\). This is one of the most common errors in this topic.
Example: The region bounded by \(y = \sqrt{x}\) and \(y = x\) (for \(0 \leq x \leq 1\)) is rotated \(360°\) about the \(x\)-axis. Find the volume.

For \(0 \leq x \leq 1\): \(\sqrt{x} \geq x\), so outer curve is \(y = \sqrt{x}\).

$$V = \pi\int_0^1 \left[(\sqrt{x})^2 - x^2\right]dx = \pi\int_0^1(x - x^2)\,dx$$ $$= \pi\left[\frac{x^2}{2} - \frac{x^3}{3}\right]_0^1 = \pi\left[\frac{1}{2} - \frac{1}{3}\right] = \frac{\pi}{6}$$
Example (bounded region, not from 0): Find the volume when the region bounded by \(y = 4 - x^2\) and \(y = 3\) (for \(x \geq 0\)) is rotated about the \(x\)-axis.

Find intersection: \(4 - x^2 = 3 \Rightarrow x = 1\). Region is for \(0 \leq x \leq 1\).
Outer curve: \(y = 4 - x^2\); inner: \(y = 3\)

$$V = \pi\int_0^1\left[(4-x^2)^2 - 9\right]dx = \pi\int_0^1(16 - 8x^2 + x^4 - 9)\,dx$$ $$= \pi\int_0^1(7 - 8x^2 + x^4)\,dx = \pi\left[7x - \frac{8x^3}{3} + \frac{x^5}{5}\right]_0^1$$ $$= \pi\left[7 - \frac{8}{3} + \frac{1}{5}\right] = \pi\left[\frac{105}{15} - \frac{40}{15} + \frac{3}{15}\right] = \frac{68\pi}{15}$$

5. Parametric Form

If the curve is given parametrically as \(x = f(t), y = g(t)\), use the chain rule:

$$V = \pi\int y^2 \, dx = \pi\int_{t_1}^{t_2} [g(t)]^2 \cdot f'(t) \, dt$$

Be careful with the direction of integration — if \(x\) is decreasing as \(t\) increases, the integral will be negative. Ensure you account for this by checking the limits.

7. Common Errors and Exam Tips

Error 1: Writing \(V = \pi\int y\,dx\) instead of \(V = \pi\int y^2\,dx\). Always square the function.

Error 2: For the between-curves formula: \((y_1 - y_2)^2 \neq y_1^2 - y_2^2\). Use the correct washer formula.

Error 3: Forgetting to rearrange \(y = f(x)\) into \(x = g(y)\) when rotating about the \(y\)-axis.

Error 4: Using the wrong limits. When rotating about the \(x\)-axis, limits are \(x\)-values. When rotating about \(y\)-axis, limits are \(y\)-values.

Mark scheme note: Give your answer as an exact multiple of \(\pi\) unless told otherwise (e.g., \(\frac{68\pi}{15}\), not \(14.2\)).
