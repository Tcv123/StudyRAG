1. Definitions

$$\cosh x = \frac{e^x + e^{-x}}{2}, \qquad \sinh x = \frac{e^x - e^{-x}}{2}, \qquad \tanh x = \frac{\sinh x}{\cosh x} = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$ Also: \(\text{sech}\,x = \dfrac{1}{\cosh x}\), \(\text{cosech}\,x = \dfrac{1}{\sinh x}\), \(\text{coth}\,x = \dfrac{\cosh x}{\sinh x}\)

3. Hyperbolic Identities

Fundamental identity: $$\cosh^2 x - \sinh^2 x = 1$$ Proof: \(\cosh^2 x - \sinh^2 x = \dfrac{(e^x+e^{-x})^2-(e^x-e^{-x})^2}{4} = \dfrac{4}{4} = 1\)
Trig identityHyperbolic equivalent
\(\cos^2\theta+\sin^2\theta=1\)\(\cosh^2 x - \sinh^2 x = 1\)
\(\sin 2\theta = 2\sin\theta\cos\theta\)\(\sinh 2x = 2\sinh x\cosh x\)
\(\cos 2\theta = \cos^2\theta - \sin^2\theta\)\(\cosh 2x = \cosh^2 x + \sinh^2 x\)
\(\cos 2\theta = 2\cos^2\theta - 1\)\(\cosh 2x = 2\cosh^2 x - 1\)
\(\cos 2\theta = 1 - 2\sin^2\theta\)\(\cosh 2x = 1 + 2\sinh^2 x\)
\(1+\tan^2\theta = \sec^2\theta\)\(1 - \tanh^2 x = \text{sech}^2 x\)
\(\cot^2\theta+1=\csc^2\theta\)\(\coth^2 x - 1 = \text{cosech}^2 x\)
Osborne's Rule: To convert a trig identity to a hyperbolic identity, replace:
  • \(\cos\theta \to \cosh x\)
  • \(\sin\theta \to i\sinh x\) (i.e., replace \(\sin^2\) with \(-\sinh^2\))
  • Every product of two sines (or \(\sin^2\)) changes sign
This works for identities but does NOT work for inequalities or equations.
Exam tip: Osborne's rule is a memory aid, not a proof. In proofs, start from definitions (exponential form) and manipulate algebraically.

6. Inverse Hyperbolic Functions

$$\text{arsinh}\,x = \ln\!\left(x+\sqrt{x^2+1}\right), \quad x \in \mathbb{R}$$ $$\text{arcosh}\,x = \ln\!\left(x+\sqrt{x^2-1}\right), \quad x \geq 1$$ $$\text{artanh}\,x = \frac{1}{2}\ln\!\left(\frac{1+x}{1-x}\right), \quad |x| < 1$$

6.1 Proof for arsinh x

Let \(y = \text{arsinh}\,x\), so \(\sinh y = x\), i.e. \(\dfrac{e^y-e^{-y}}{2} = x\).

Multiply by \(2e^y\): \(e^{2y} - 1 = 2xe^y\), so \(e^{2y} - 2xe^y - 1 = 0\).

Quadratic in \(e^y\): \(e^y = \dfrac{2x\pm\sqrt{4x^2+4}}{2} = x\pm\sqrt{x^2+1}\).

Since \(e^y > 0\) and \(x - \sqrt{x^2+1} < 0\), we take the positive root: \(e^y = x+\sqrt{x^2+1}\), so \(y = \ln(x+\sqrt{x^2+1})\). âœ“

7. Derivatives of Inverse Hyperbolic Functions

$$\frac{d}{dx}(\text{arsinh}\,x) = \frac{1}{\sqrt{x^2+1}}$$ $$\frac{d}{dx}(\text{arcosh}\,x) = \frac{1}{\sqrt{x^2-1}}, \quad x > 1$$ $$\frac{d}{dx}(\text{artanh}\,x) = \frac{1}{1-x^2}, \quad |x| < 1$$

Proof for arsinh: Let \(y = \text{arsinh}\,x\), so \(\sinh y = x\). Differentiate implicitly: \(\cosh y \cdot \dfrac{dy}{dx} = 1\). So \(\dfrac{dy}{dx} = \dfrac{1}{\cosh y} = \dfrac{1}{\sqrt{\sinh^2 y+1}} = \dfrac{1}{\sqrt{x^2+1}}\).

8. Integration Using Inverse Hyperbolic Functions

$$\int\frac{1}{\sqrt{x^2+a^2}}\,dx = \text{arsinh}\!\left(\frac{x}{a}\right)+C = \ln\!\left(x+\sqrt{x^2+a^2}\right)+C$$ $$\int\frac{1}{\sqrt{x^2-a^2}}\,dx = \text{arcosh}\!\left(\frac{x}{a}\right)+C = \ln\!\left(x+\sqrt{x^2-a^2}\right)+C, \quad x > a > 0$$
Example: Find \(\displaystyle\int\frac{1}{\sqrt{x^2+9}}\,dx\).

\(a = 3\). \(\text{arsinh}\!\left(\dfrac{x}{3}\right)+C = \ln\!\left(x+\sqrt{x^2+9}\right)+C\).
Example: Find \(\displaystyle\int\frac{dx}{\sqrt{x^2+4x+8}}\).

Complete the square: \(x^2+4x+8 = (x+2)^2+4\).
Let \(u = x+2\): \(\displaystyle\int\frac{du}{\sqrt{u^2+4}} = \text{arsinh}\!\left(\dfrac{u}{2}\right)+C = \text{arsinh}\!\left(\dfrac{x+2}{2}\right)+C\).
Example: Evaluate \(\displaystyle\int_2^3\frac{1}{\sqrt{x^2-1}}\,dx\).

\(a = 1\). \(\left[\text{arcosh}\,x\right]_2^3 = \text{arcosh}\,3 - \text{arcosh}\,2 = \ln(3+\sqrt{8})-\ln(2+\sqrt{3})\).
\(= \ln(3+2\sqrt{2}) - \ln(2+\sqrt{3})\).
Exam tip: When integrating with inverse hyperbolics, you can write either the \(\text{arsinh}\) form or the \(\ln\) form â€” both are accepted. The ln form is sometimes required if the question says "in the form \(\ln(\ldots)\)".

11. Exam-Style Practice Questions

  1. [Proof] Starting from the definitions of \(\sinh x\) and \(\cosh x\), prove that \(\cosh 2x = 2\cosh^2 x - 1\).
  2. [Proof] Show that \(\text{arcosh}\,x = \ln(x+\sqrt{x^2-1})\) for \(x \geq 1\).
  3. [Differentiation] Differentiate (a) \(y = \sinh(x^2)\), (b) \(y = x\cosh 3x\), (c) \(y = \text{arsinh}(2x+1)\).
  4. [Integration] Find \(\displaystyle\int_0^2\cosh^2 x\,dx\).
  5. [Completing the square] Find \(\displaystyle\int\frac{dx}{\sqrt{2x^2+4x+6}}\).
  6. [Solving equations] Solve \(2\cosh x - \sinh x = 2\) giving your answer in logarithmic form.
  7. [Identity] Prove that \(\tanh^2 x + \text{sech}^2 x = 1\).
General exam tips for CP2.6:
  • Key difference from circular trig: \(\dfrac{d}{dx}\cosh x = +\sinh x\) (not negative!)
  • arsinh is defined for all \(x\in\mathbb{R}\); arcosh is defined only for \(x \geq 1\)
  • For "solve \(a\cosh x + b\sinh x = c\)" â€” substitute \(e^x = u\) and solve the resulting quadratic
  • For inverse hyperbolic proofs: set \(y = \text{arsinh}\,x\) (or arcosh), get quadratic in \(e^y\), take the valid root
