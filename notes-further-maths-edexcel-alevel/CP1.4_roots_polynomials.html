CP1.4: Roots of Polynomials

1. Vieta's Formulae — Quadratics

For the quadratic \(ax^2 + bx + c = 0\) with roots \(\alpha\) and \(\beta\):

$$\alpha + \beta = -\frac{b}{a} \qquad \alpha\beta = \frac{c}{a}$$

Derivation: \(ax^2 + bx + c = a(x-\alpha)(x-\beta) = a[x^2 - (\alpha+\beta)x + \alpha\beta]\). Comparing coefficients of \(x\) and the constant.

2. Vieta's Formulae — Cubics

For the cubic \(ax^3 + bx^2 + cx + d = 0\) with roots \(\alpha, \beta, \gamma\):

$$\alpha + \beta + \gamma = -\frac{b}{a}$$ $$\alpha\beta + \beta\gamma + \alpha\gamma = \frac{c}{a}$$ $$\alpha\beta\gamma = -\frac{d}{a}$$

Memory aid: The coefficient pattern for the symmetric functions is \(-b/a\), \(+c/a\), \(-d/a\), \(+e/a\), ... (alternating signs).

Example: The roots of \(x^3 - 6x^2 + 11x - 6 = 0\) are \(\alpha, \beta, \gamma\). Find \(\alpha^2+\beta^2+\gamma^2\).

\(\alpha+\beta+\gamma = 6\), \(\alpha\beta+\beta\gamma+\alpha\gamma = 11\), \(\alpha\beta\gamma = 6\)

\(\alpha^2+\beta^2+\gamma^2 = 6^2 - 2(11) = 36 - 22 = 14\)

3. Vieta's Formulae — Quartics

For the quartic \(ax^4 + bx^3 + cx^2 + dx + e = 0\) with roots \(\alpha, \beta, \gamma, \delta\):

$$\alpha+\beta+\gamma+\delta = -\frac{b}{a}$$ $$\alpha\beta+\alpha\gamma+\alpha\delta+\beta\gamma+\beta\delta+\gamma\delta = \frac{c}{a}$$ $$\alpha\beta\gamma+\alpha\beta\delta+\alpha\gamma\delta+\beta\gamma\delta = -\frac{d}{a}$$ $$\alpha\beta\gamma\delta = \frac{e}{a}$$

The second symmetric function for the quartic sums all \(\binom{4}{2} = 6\) products of pairs of roots. The third sums all products of triples.

4. Forming New Equations with Transformed Roots

Substitution Method

Given an equation with roots \(\alpha, \beta, \gamma\) and asked for the equation with roots \(f(\alpha), f(\beta), f(\gamma)\):

  1. Let \(w = f(\alpha)\) (the transformation applied to a general root)
  2. Rearrange to express \(\alpha\) in terms of \(w\)
  3. Substitute into the original equation
  4. Simplify to get the new equation in \(w\)
  5. Replace \(w\) with \(x\) (or \(z\)) for the final answer
Example: The equation \(x^3 - 3x^2 + 2x + 1 = 0\) has roots \(\alpha, \beta, \gamma\). Find the equation with roots \(\alpha+2, \beta+2, \gamma+2\).

Let \(w = \alpha + 2\), so \(\alpha = w - 2\).
Substitute \(x = w-2\) into the original equation:

\((w-2)^3 - 3(w-2)^2 + 2(w-2) + 1 = 0\)

Expand \((w-2)^3 = w^3 - 6w^2 + 12w - 8\)
Expand \(-3(w-2)^2 = -3(w^2 - 4w + 4) = -3w^2 + 12w - 12\)
Expand \(2(w-2) = 2w - 4\)

Total: \(w^3 - 6w^2 + 12w - 8 - 3w^2 + 12w - 12 + 2w - 4 + 1 = 0\)
\(w^3 - 9w^2 + 26w - 23 = 0\)

The new equation is \(x^3 - 9x^2 + 26x - 23 = 0\).
Example: The equation \(2x^3 - x^2 + 3x - 5 = 0\) has roots \(\alpha, \beta, \gamma\). Find the equation with roots \(\frac{1}{\alpha}, \frac{1}{\beta}, \frac{1}{\gamma}\).

Let \(w = \frac{1}{\alpha}\), so \(\alpha = \frac{1}{w}\). Substitute \(x = \frac{1}{w}\):

\(\frac{2}{w^3} - \frac{1}{w^2} + \frac{3}{w} - 5 = 0\)

Multiply through by \(w^3\):
\(2 - w + 3w^2 - 5w^3 = 0\)
\(5w^3 - 3w^2 + w - 2 = 0\)

The new equation is \(5x^3 - 3x^2 + x - 2 = 0\).
Example: The equation \(x^3 - 4x + 7 = 0\) has roots \(\alpha, \beta, \gamma\). Find the equation with roots \(2\alpha-1, 2\beta-1, 2\gamma-1\).

Let \(w = 2\alpha - 1\), so \(\alpha = \frac{w+1}{2}\). Substitute \(x = \frac{w+1}{2}\):

\(\left(\frac{w+1}{2}\right)^3 - 4\left(\frac{w+1}{2}\right) + 7 = 0\)

\(\frac{(w+1)^3}{8} - 2(w+1) + 7 = 0\)

Multiply by 8:
\((w+1)^3 - 16(w+1) + 56 = 0\)
\(w^3 + 3w^2 + 3w + 1 - 16w - 16 + 56 = 0\)
\(w^3 + 3w^2 - 13w + 41 = 0\)

New equation: \(x^3 + 3x^2 - 13x + 41 = 0\)
Exam tip: There are two methods: (1) substitution (cleanest for most transformations), (2) computing the new symmetric functions directly using Vieta's formulae. For transformations like \(\alpha+k\), \(k\alpha\), or \(1/\alpha\), substitution is almost always faster. For \(\alpha^2\) or more complex transformations, computing new symmetric functions directly may be easier.

5. New Equations via Symmetric Functions

An alternative to substitution: compute the three symmetric functions of the new roots using the known values from the original equation, then write the new equation as \(x^3 - (\text{sum})x^2 + (\text{sum of pairs})x - (\text{product}) = 0\).

Example: The equation \(x^3 + 2x^2 - 5x + 3 = 0\) has roots \(\alpha, \beta, \gamma\). Find the equation with roots \(\alpha^2, \beta^2, \gamma^2\).

From Vieta's: \(\alpha+\beta+\gamma = -2\), \(\alpha\beta+\beta\gamma+\alpha\gamma = -5\), \(\alpha\beta\gamma = -3\).

New roots: \(\alpha^2, \beta^2, \gamma^2\).
Sum: \(\alpha^2+\beta^2+\gamma^2 = (\alpha+\beta+\gamma)^2 - 2(\alpha\beta+\beta\gamma+\alpha\gamma) = 4 + 10 = 14\)
Sum of pairs: \(\alpha^2\beta^2+\beta^2\gamma^2+\alpha^2\gamma^2 = (\alpha\beta+\beta\gamma+\alpha\gamma)^2 - 2\alpha\beta\gamma(\alpha+\beta+\gamma) = 25 - 2(-3)(-2) = 25 - 12 = 13\)
Product: \(\alpha^2\beta^2\gamma^2 = (\alpha\beta\gamma)^2 = 9\)

New equation: \(x^3 - 14x^2 + 13x - 9 = 0\)
