1.7: Polar Coordinates

1. Definition and Conversion

Polar coordinates: A point \(P\) is described by \((r, \theta)\) where:
  • \(r \geq 0\) is the distance from the origin (pole)
  • \(\theta\) is the angle measured anticlockwise from the initial line (positive \(x\)-axis)
Conversion formulae: $$x = r\cos\theta, \quad y = r\sin\theta$$ $$r^2 = x^2 + y^2, \quad \tan\theta = \frac{y}{x}$$

3. Sketching Polar Curves

Method:

  1. Find where \(r = 0\) (curve passes through pole)
  2. Find maximum and minimum values of \(r\)
  3. Check symmetry: curve is symmetric about initial line if \(r(-\theta) = r(\theta)\); about \(\theta = \pi/2\) if \(r(\pi-\theta)=r(\theta)\)
  4. Build a table of values for key angles: \(0, \pi/6, \pi/4, \pi/3, \pi/2, \pi, \ldots\)
  5. Sketch, noting the direction of tracing as \(\theta\) increases
AQA Exam Tip: AQA mark schemes for sketching award marks for: shape (1), correct intercepts on initial line (1), correct behaviour at pole/origin (1), and any labelled maximum values. Always label the maximum value of \(r\) on your sketch.

4. Area in Polar Coordinates

Area swept out by the radius vector: $$A = \frac{1}{2}\int_{\alpha}^{\beta} r^2\,d\theta$$ (the area of the sector from angle \(\alpha\) to \(\beta\)).

Area between two polar curves: $$A = \frac{1}{2}\int_{\alpha}^{\beta} (r_1^2 - r_2^2)\,d\theta$$ where \(r_1 > r_2\) in \([\alpha, \beta]\).
Example: Find the total area enclosed by the cardioid \(r = a(1+\cos\theta)\).

The cardioid is traced out for \(\theta \in [0, 2\pi]\), but using symmetry about the initial line:
\(A = 2 \times \dfrac{1}{2}\int_0^{\pi} a^2(1+\cos\theta)^2\,d\theta = a^2\int_0^{\pi}(1+2\cos\theta+\cos^2\theta)\,d\theta\)

Using \(\cos^2\theta = \dfrac{1+\cos2\theta}{2}\):
\(= a^2\int_0^{\pi}\left(\dfrac{3}{2} + 2\cos\theta + \dfrac{\cos2\theta}{2}\right)d\theta\)
\(= a^2\left[\dfrac{3\theta}{2} + 2\sin\theta + \dfrac{\sin2\theta}{4}\right]_0^{\pi} = a^2\!\left(\dfrac{3\pi}{2}\right) = \dfrac{3\pi a^2}{2}\)
Example: Find the area of one loop of the rose curve \(r = 2\cos(2\theta)\).

One petal exists where \(r \geq 0\), e.g. \(-\pi/4 \leq \theta \leq \pi/4\):
\(A = \dfrac{1}{2}\int_{-\pi/4}^{\pi/4} 4\cos^2(2\theta)\,d\theta = 2\int_0^{\pi/4}(1+\cos4\theta)\,d\theta\)
\(= 2\left[\theta + \dfrac{\sin4\theta}{4}\right]_0^{\pi/4} = 2\left(\dfrac{\pi}{4} + 0\right) = \dfrac{\pi}{2}\)
AQA Exam Tip: Always identify the correct limits carefully. For a cardioid or lima\c{c}on, the full curve is \([0, 2\pi]\). For one loop of a rose, find where \(r = 0\) to determine limits. Write down your limits explicitly — AQA marks this.

5. Tangents to Polar Curves

To find \(\dfrac{dy}{dx}\) for a polar curve:

$$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta} \quad \text{where } x = r\cos\theta,\ y = r\sin\theta$$
$$\frac{dy}{d\theta} = \frac{dr}{d\theta}\sin\theta + r\cos\theta, \qquad \frac{dx}{d\theta} = \frac{dr}{d\theta}\cos\theta - r\sin\theta$$
Example: Find the tangent at the pole for \(r = a(1+\cos\theta)\).

At the pole, \(r = 0 \Rightarrow 1 + \cos\theta = 0 \Rightarrow \theta = \pi\).

\(\dfrac{dr}{d\theta} = -a\sin\theta\)
At \(\theta = \pi\): \(\dfrac{dy}{d\theta} = (-a\sin\pi)(\sin\pi) + 0\cdot\cos\pi = 0\)
\(\dfrac{dx}{d\theta} = (-a\sin\pi)(\cos\pi) - 0 = 0\)

Indeterminate — use L'Hopital or note the tangent direction is \(\theta = \pi\), i.e. horizontal.

7. Practice Checklist

  • Can you convert between Cartesian and polar coordinates?
  • Can you sketch standard polar curves (circle, cardioid, rose, lemniscate)?
  • Can you find the area enclosed by a polar curve?
  • Can you find the area between two polar curves?
  • Can you find the gradient \(dy/dx\) at a given point on a polar curve?
