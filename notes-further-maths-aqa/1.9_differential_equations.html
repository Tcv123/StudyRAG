1.9: Differential Equations

1. First Order Separable Differential Equations

A differential equation is separable if it can be written as: $$\frac{dy}{dx} = f(x)\,g(y)$$ Rearranging: \(\displaystyle\int \frac{1}{g(y)}\,dy = \int f(x)\,dx\)
Example: Solve \(\dfrac{dy}{dx} = xy\), with \(y(0) = 2\).

\(\int\dfrac{1}{y}\,dy = \int x\,dx \Rightarrow \ln|y| = \dfrac{x^2}{2} + C\)
\(|y| = Ae^{x^2/2}\), so \(y = Ae^{x^2/2}\) (where \(A\) can be positive or negative).

Applying \(y(0) = 2\): \(2 = A\). Particular solution: \(y = 2e^{x^2/2}\).
Example: Solve \(\dfrac{dy}{dx} = \dfrac{1+y^2}{1+x^2}\).

\(\int\dfrac{dy}{1+y^2} = \int\dfrac{dx}{1+x^2}\)
\(\arctan y = \arctan x + C\)
\(y = \tan(\arctan x + C)\)

2. First Order Linear Differential Equations

Standard form: \(\dfrac{dy}{dx} + P(x)\,y = Q(x)\)

Integrating factor: \(\mu = e^{\int P(x)\,dx}\)

Multiply through by \(\mu\): the left side becomes \(\dfrac{d}{dx}(\mu y)\), giving: $$\mu y = \int \mu\,Q(x)\,dx$$
Example: Solve \(\dfrac{dy}{dx} + \dfrac{2y}{x} = x^2\).

\(P(x) = 2/x\). Integrating factor: \(\mu = e^{\int 2/x\,dx} = e^{2\ln x} = x^2\).

Multiply: \(\dfrac{d}{dx}(x^2 y) = x^4\)
Integrate: \(x^2 y = \dfrac{x^5}{5} + C\)
General solution: \(y = \dfrac{x^3}{5} + \dfrac{C}{x^2}\)
AQA Exam Tip: Always simplify the integrating factor \(\mu = e^{\int P\,dx}\) before multiplying through. A common error is forgetting to multiply \(Q(x)\) on the right-hand side.

3. Second Order Homogeneous Differential Equations

Form: \(a\dfrac{d^2y}{dx^2} + b\dfrac{dy}{dx} + cy = 0\)

Auxiliary equation: \(am^2 + bm + c = 0\)

The general solution depends on the roots of the auxiliary equation:
Roots of auxiliary equationGeneral solution
Two distinct real roots \(m_1, m_2\)\(y = Ae^{m_1 x} + Be^{m_2 x}\)
Repeated real root \(m\)\(y = (A + Bx)e^{mx}\)
Complex roots \(\alpha \pm \beta i\)\(y = e^{\alpha x}(A\cos\beta x + B\sin\beta x)\)
Example (distinct real roots): Solve \(\dfrac{d^2y}{dx^2} - 5\dfrac{dy}{dx} + 6y = 0\).

Auxiliary: \(m^2 - 5m + 6 = 0 \Rightarrow (m-2)(m-3) = 0 \Rightarrow m = 2, 3\)
General solution: \(y = Ae^{2x} + Be^{3x}\)
Example (repeated root): Solve \(\dfrac{d^2y}{dx^2} - 6\dfrac{dy}{dx} + 9y = 0\).

Auxiliary: \(m^2 - 6m + 9 = (m-3)^2 = 0 \Rightarrow m = 3\) (repeated)
General solution: \(y = (A + Bx)e^{3x}\)
Example (complex roots): Solve \(\dfrac{d^2y}{dx^2} - 4\dfrac{dy}{dx} + 13y = 0\).

Auxiliary: \(m^2 - 4m + 13 = 0 \Rightarrow m = \dfrac{4\pm\sqrt{16-52}}{2} = 2\pm 3i\)
So \(\alpha = 2\), \(\beta = 3\).
General solution: \(y = e^{2x}(A\cos 3x + B\sin 3x)\)

General solution: \(y = y_{CF} + y_{PI}\)
where \(y_{CF}\) is the complementary function (solution of the homogeneous equation) and \(y_{PI}\) is any particular integral satisfying the full equation.

Example: Find the general solution of \(\dfrac{d^2y}{dx^2} - 3\dfrac{dy}{dx} + 2y = 4e^{3x}\).

CF: Auxiliary: \(m^2-3m+2=(m-1)(m-2)=0\), so \(y_{CF} = Ae^x + Be^{2x}\).

Substitute: \(9\lambda e^{3x} - 9\lambda e^{3x} + 2\lambda e^{3x} = 4e^{3x}\)

Example: Find the general solution of \(y'' - 3y' + 2y = \sin x\).

CF: \(y_{CF} = Ae^x + Be^{2x}\) (as before).

PI: Try \(y_{PI} = \lambda\cos x + \mu\sin x\).
\(y' = -\lambda\sin x + \mu\cos x\), \(y'' = -\lambda\cos x - \mu\sin x\)
Substitute:
\((-\lambda\cos x - \mu\sin x) - 3(-\lambda\sin x + \mu\cos x) + 2(\lambda\cos x+\mu\sin x) = \sin x\)
Cosine: \(-\lambda - 3\mu + 2\lambda = \lambda - 3\mu = 0\)
Sine: \(-\mu + 3\lambda + 2\mu = \mu + 3\lambda = 1\)
From first: \(\lambda = 3\mu\). Substitute: \(\mu + 9\mu = 1 \Rightarrow \mu = \frac{1}{10}\), \(\lambda = \frac{3}{10}\).

\(y = Ae^x + Be^{2x} + \dfrac{3}{10}\cos x + \dfrac{1}{10}\sin x\)
AQA Exam Tip: If your trial PI is part of the complementary function, you must multiply the trial by \(x\) (or \(x^2\) for repeated roots). Forgetting this gives a zero coefficient and no PI.

5. Applying Initial/Boundary Conditions

Example: Solve \(y'' + 4y = 8\) with \(y(0) = 3\), \(y'(0) = 0\).

CF: Auxiliary: \(m^2+4=0 \Rightarrow m = \pm 2i\). \(y_{CF} = A\cos 2x + B\sin 2x\)

PI: Try \(y_{PI} = \lambda\) (constant). Then \(y'' = 0\): \(0 + 4\lambda = 8 \Rightarrow \lambda = 2\).

General solution: \(y = A\cos 2x + B\sin 2x + 2\)

Apply \(y(0)=3\): \(A + 2 = 3 \Rightarrow A = 1\)
\(y' = -2A\sin 2x + 2B\cos 2x\); \(y'(0)=0\): \(2B = 0 \Rightarrow B = 0\)

Particular solution: \(y = \cos 2x + 2\)

6. Coupled Differential Equations

A system of coupled first-order equations can often be reduced to a single second-order equation.

Method for coupled ODEs: Given \(\dfrac{dx}{dt} = ax + by\) and \(\dfrac{dy}{dt} = cx + dy\):
  1. Differentiate one equation to get \(\ddot{x}\)
  2. Substitute the other equation to eliminate \(y\) (and \(\dot{y}\))
  3. Solve the resulting second-order ODE for \(x(t)\)
  4. Back-substitute to find \(y(t)\)
Example: Solve \(\dot{x} = 3x + 2y\), \(\dot{y} = x + 2y\), with \(x(0) = 1\), \(y(0) = 0\).

Differentiate first equation: \(\ddot{x} = 3\dot{x} + 2\dot{y} = 3\dot{x} + 2(x + 2y)\)
From first equation: \(y = \dfrac{\dot{x}-3x}{2}\)
Substitute: \(\ddot{x} = 3\dot{x} + 2x + 2\cdot2\cdot\dfrac{\dot{x}-3x}{2} = 3\dot{x}+2x+2\dot{x}-6x = 5\dot{x}-4x\)
\(\ddot{x} - 5\dot{x} + 4x = 0\)
Auxiliary: \(m^2-5m+4=(m-1)(m-4)=0\)
\(x = Ae^t + Be^{4t}\)
\(y = \dfrac{\dot{x}-3x}{2} = \dfrac{Ae^t+4Be^{4t}-3Ae^t-3Be^{4t}}{2} = -Ae^t + \dfrac{Be^{4t}}{2}\)

Applying IC: \(A+B=1\), \(-A+B/2=0 \Rightarrow B=2A\), so \(3A=1 \Rightarrow A=1/3\), \(B=2/3\).
\(x = \dfrac{1}{3}e^t + \dfrac{2}{3}e^{4t}\), \(y = -\dfrac{1}{3}e^t + \dfrac{1}{3}e^{4t}\)

7. Equilibrium Points and Stability

For a first-order autonomous DE \(\dfrac{dy}{dx} = f(y)\), equilibrium points (or fixed points) satisfy \(f(y^*) = 0\).

  • Stable equilibrium: If \(f'(y^*) < 0\), small perturbations decay back to \(y^*\)
  • Unstable equilibrium: If \(f'(y^*) > 0\), perturbations grow away from \(y^*\)

Coupled Differential Equations — Worked Example

To solve a pair of coupled first-order ODEs, use the elimination method:

  1. Express one variable in terms of the other and its derivative from the first equation.
  2. Differentiate and substitute into the second equation to get a single second-order ODE.
  3. Solve the second-order ODE, then back-substitute for the second variable.
Example: Solve \(\dot{x}=3x-y\), \(\dot{y}=x+y\). Step 1: From the first equation, \(y=3x-\dot{x}\). Step 2: Differentiate: \(\dot{y}=3\dot{x}-\ddot{x}\). Substitute into second: \(3\dot{x}-\ddot{x}=x+(3x-\dot{x})=4x-\dot{x}\). $$\ddot{x}-4\dot{x}+4x=0$$ Step 3: Auxiliary equation: \((m-2)^2=0\), so \(m=2\) (repeated root). $$x=(At+B)e^{2t}$$ Step 4: \(y=3x-\dot{x}\). Compute \(\dot{x}=(A+2At+2B)e^{2t}\). Then: $$y=3(At+B)e^{2t}-(A+2At+2B)e^{2t}=(At-A+B)e^{2t}$$
Check: Always verify by substituting back into both original equations.

Second Order ODE — Physical Interpretation

The equation \(m\ddot{x}+c\dot{x}+kx=F(t)\) models a mass-spring-damper system:

  • \(m\): mass, \(c\): damping coefficient, \(k\): spring constant, \(F(t)\): forcing function.
  • Dividing by \(m\): \(\ddot{x}+2\beta\dot{x}+\omega_0^2 x = F(t)/m\) where \(\omega_0=\sqrt{k/m}\) is the natural frequency and \(\beta=c/(2m)\) is the damping ratio.
ConditionRoot typeMotion type
\(c^2<4mk\) (i.e.\ \(\beta<\omega_0\))Complex conjugates \(-\beta\pm i\omega_d\)Underdamped: damped oscillation \(e^{-\beta t}\sin(\omega_d t+\phi)\)
\(c^2=4mk\)Repeated real root \(-\beta\)Critically damped: fastest decay without oscillation \((A+Bt)e^{-\beta t}\)
\(c^2>4mk\)Two distinct real negative rootsOverdamped: slow exponential decay \(Ae^{-\alpha_1 t}+Be^{-\alpha_2 t}\)
\(c=0\)Pure imaginary \(\pm i\omega_0\)Simple harmonic motion: \(A\cos\omega_0 t+B\sin\omega_0 t\)
Resonance: When the forcing frequency equals the natural frequency (\(F(t)=F_0\cos\omega_0 t\), \(c=0\)), the response grows without bound: \(x_p = \dfrac{F_0 t}{2m\omega_0}\sin\omega_0 t\). This is why resonance is physically dangerous.

First-Order ODEs — Full Methods

Method 1 — Separable equations: \(\dfrac{dy}{dx} = f(x)g(y)\).
Separate: \(\dfrac{1}{g(y)}\,dy = f(x)\,dx\). Integrate both sides.
Example: \(\dfrac{dy}{dx} = xy\). Separate: \(\dfrac{1}{y}\,dy = x\,dx\).
\(\ln|y| = \dfrac{x^2}{2}+C \Rightarrow y = Ae^{x^2/2}\) where \(A=\pm e^C\).
Method 2 — Integrating factor (linear first-order): \(\dfrac{dy}{dx}+P(x)y = Q(x)\).
Multiply by integrating factor \(\mu = e^{\int P\,dx}\):
\(\dfrac{d}{dx}(\mu y) = \mu Q \Rightarrow \mu y = \int\mu Q\,dx\).
Example: \(\dfrac{dy}{dx} + 2y = e^{-x}\).
\(\mu = e^{\int 2\,dx} = e^{2x}\). Multiply: \(\dfrac{d}{dx}(e^{2x}y) = e^{2x}\cdot e^{-x} = e^x\).
Integrate: \(e^{2x}y = e^x + C \Rightarrow y = e^{-x}+Ce^{-2x}\).
Example 2 (non-constant \(P\)): \(x\dfrac{dy}{dx}+y = x^2\), i.e. \(\dfrac{dy}{dx}+\dfrac{1}{x}y=x\).
\(\mu = e^{\int(1/x)dx} = e^{\ln x} = x\). Multiply: \(\dfrac{d}{dx}(xy) = x^2\).
Integrate: \(xy = \dfrac{x^3}{3}+C \Rightarrow y = \dfrac{x^2}{3}+\dfrac{C}{x}\).

Particular Integrals — Worked Examples

Example 1 — Polynomial RHS: \(\dfrac{d^2y}{dx^2}+3\dfrac{dy}{dx}+2y = 4x+1\).
Try PI: \(y_p = ax+b\). \(y_p''=0,\;y_p'=a\).
Sub: \(0+3a+2(ax+b)=4x+1 \Rightarrow 2ax+(3a+2b)=4x+1\).
\(2a=4 \Rightarrow a=2\). \(3(2)+2b=1 \Rightarrow b=-\tfrac{5}{2}\). PI: \(y_p = 2x-\tfrac{5}{2}\).
Example 2 — Exponential RHS: \(\dfrac{d^2y}{dx^2}-3\dfrac{dy}{dx}+2y = e^{4x}\).
Try \(y_p=\lambda e^{4x}\). Sub: \(16\lambda-12\lambda+2\lambda=1 \Rightarrow 6\lambda=1 \Rightarrow \lambda=\tfrac{1}{6}\). PI: \(y_p=\tfrac{1}{6}e^{4x}\).
Example 3 — Trigonometric RHS: \(\dfrac{d^2y}{dx^2}+4y = \sin 3x\).
Try \(y_p = A\cos 3x+B\sin 3x\). \(y_p''=-9A\cos 3x-9B\sin 3x\).
Sub: \(-9A\cos 3x-9B\sin 3x + 4A\cos 3x+4B\sin 3x = \sin 3x\).
\(-5A=0 \Rightarrow A=0\). \(-5B=1 \Rightarrow B=-\tfrac{1}{5}\). PI: \(y_p = -\tfrac{1}{5}\sin 3x\).

Boundary Conditions and Initial Value Problems

Full Example: Solve \(\dfrac{d^2y}{dx^2}-4\dfrac{dy}{dx}+3y=6\) with \(y(0)=2,\; y'(0)=3\).
AE: \(m^2-4m+3=0 \Rightarrow (m-1)(m-3)=0\). CF: \(y_c = Ae^x+Be^{3x}\).
PI: Try \(y_p=k\) (constant). \(0-0+3k=6 \Rightarrow k=2\). PI: \(y_p=2\).
GS: \(y = Ae^x+Be^{3x}+2\).
\(y(0)=A+B+2=2 \Rightarrow A+B=0\).
\(y'=Ae^x+3Be^{3x}\). \(y'(0)=A+3B=3\).
Solving: \(B=3/2,\; A=-3/2\).
\(y = \dfrac{3}{2}e^{3x}-\dfrac{3}{2}e^x+2\)

Substitution Methods for ODEs

Some ODEs are not standard form; a substitution reduces them.
Bernoulli's equation: \(\dfrac{dy}{dx}+P(x)y = Q(x)y^n\).
Substitute \(v=y^{1-n}\). Then \(\dfrac{dv}{dx}=(1-n)y^{-n}\dfrac{dy}{dx}\), giving a linear equation in \(v\).
Example: \(\dfrac{dy}{dx}-y = y^2\). Here \(n=2\). Let \(v=y^{-1}\).
\(\dfrac{dv}{dx}=-y^{-2}\dfrac{dy}{dx}\). Divide original by \(y^2\): \(y^{-2}\dfrac{dy}{dx}-y^{-1}=1\).
So \(-\dfrac{dv}{dx}-v=1 \Rightarrow \dfrac{dv}{dx}+v=-1\). Integrating factor \(e^x\):
\(\dfrac{d}{dx}(ve^x)=-e^x \Rightarrow ve^x=-e^x+C \Rightarrow v=-1+Ce^{-x}\).
\(y = \dfrac{1}{v} = \dfrac{1}{Ce^{-x}-1}\).

AQA Differential Equations — Exam Technique

  • Always write GS = CF + PI clearly. Examiners look for this structure.
  • Resonance: If the PI trial form already appears in the CF, multiply by \(x\). If it appears as a double root, multiply by \(x^2\).
  • First-order linear: Always multiply through by IF before integrating; do not try to separate unless it genuinely is separable.
  • CF check: Verify your CF satisfies the homogeneous equation by substituting back.
  • PI check: Verify your PI satisfies the full equation by substituting back.
  • IVP: Apply both initial conditions to GS to get two equations for \(A\) and \(B\); solve simultaneously.
  • Coupled ODEs: Eliminate one variable by differentiating one equation and substituting the other; this gives a second-order ODE in one variable.

Modelling with Differential Equations

Common models:
  • Exponential growth/decay: \(\dfrac{dN}{dt} = kN\) gives \(N = N_0 e^{kt}\). If \(k>0\): growth; \(k<0\): decay.
  • Newton's law of cooling: \(\dfrac{d\theta}{dt} = -k(\theta-\theta_{\rm amb})\). Solution: \(\theta - \theta_{\rm amb} = Ae^{-kt}\).
  • Logistic growth: \(\dfrac{dN}{dt} = kN\!\left(1-\dfrac{N}{L}\right)\) where \(L\) is the carrying capacity.
  • Harmonic oscillator: \(\dfrac{d^2x}{dt^2}+\omega^2 x=0\) gives \(x=A\cos\omega t+B\sin\omega t\).
  • Damped oscillator: \(\dfrac{d^2x}{dt^2}+2k\dfrac{dx}{dt}+\omega^2 x=0\).
Example — Newton's cooling: A body at \(80°C\) cools in a room at \(20°C\). After 10 minutes it is \(60°C\). Find its temperature after 20 minutes.
\(\theta-20 = Ae^{-kt}\). At \(t=0\): \(A=60\). At \(t=10\): \(40=60e^{-10k} \Rightarrow e^{-10k}=\tfrac{2}{3}\).
At \(t=20\): \(\theta-20 = 60\cdot\left(\tfrac{2}{3}\right)^2 = \tfrac{160}{3}\). \(\theta = 20+\tfrac{160}{3} \approx 73.3°C\).

Resonance — Full Worked Example

Example: Solve \(\dfrac{d^2y}{dx^2}+4y = \sin 2x\) (resonance case).
CF: AE \(m^2+4=0 \Rightarrow m=\pm 2i\). CF: \(y_c=A\cos 2x+B\sin 2x\).
PI: Since \(\sin 2x\) is in CF, try \(y_p = x(C\cos 2x+D\sin 2x)\).
\(y_p' = C\cos 2x+D\sin 2x+x(-2C\sin 2x+2D\cos 2x)\)
\(y_p'' = -2C\sin 2x+2D\cos 2x + (-2C\sin 2x+2D\cos 2x) + x(-4C\cos 2x-4D\sin 2x)\)
\(= -4C\sin 2x+4D\cos 2x - 4x(C\cos 2x+D\sin 2x)\)
Sub into \(y_p''+4y_p\): \(-4C\sin 2x+4D\cos 2x = \sin 2x\).
\(4D=0 \Rightarrow D=0\). \(-4C=1 \Rightarrow C=-\tfrac{1}{4}\).
PI: \(y_p = -\tfrac{x}{4}\cos 2x\). GS: \(y = A\cos 2x+B\sin 2x - \dfrac{x}{4}\cos 2x\).

Coupled ODEs — Full Worked Example

Solve: \(\dfrac{dx}{dt} = x+2y,\quad\dfrac{dy}{dt} = 3x+2y\) with \(x(0)=1,\; y(0)=0\).
Differentiate first equation: \(\ddot{x} = \dot{x}+2\dot{y} = \dot{x}+2(3x+2y) = \dot{x}+6x+4y\).
From first equation: \(2y = \dot{x}-x \Rightarrow y = \tfrac{1}{2}(\dot{x}-x)\).
Sub: \(\ddot{x} = \dot{x}+6x+2(\dot{x}-x) = 3\dot{x}+4x\).
\(\ddot{x}-3\dot{x}-4x=0\). AE: \(m^2-3m-4=0 \Rightarrow (m-4)(m+1)=0\). \(m=4,-1\).
\(x = Ae^{4t}+Be^{-t}\). Then \(y = \tfrac{1}{2}(\dot{x}-x) = \tfrac{1}{2}(4Ae^{4t}-Be^{-t}-Ae^{4t}-Be^{-t}) = \tfrac{3A}{2}e^{4t}-Be^{-t}\).
Conditions: \(x(0)=A+B=1\); \(y(0)=\tfrac{3A}{2}-B=0 \Rightarrow B=\tfrac{3A}{2}\).
\(A+\tfrac{3A}{2}=1 \Rightarrow \tfrac{5A}{2}=1 \Rightarrow A=\tfrac{2}{5},\; B=\tfrac{3}{5}\).
\(x=\tfrac{2}{5}e^{4t}+\tfrac{3}{5}e^{-t},\quad y=\tfrac{3}{5}e^{4t}-\tfrac{3}{5}e^{-t}\).

Second-Order ODE Applications — SHM and Damping

A mass-spring system with damping satisfies: \[m\ddot{x} + c\dot{x} + kx = F(t)\] where \(m\) = mass, \(c\) = damping coefficient, \(k\) = spring constant, \(F(t)\) = external force.
  • Underdamped: \(c^2 < 4mk\) — complex roots, oscillatory decay.
  • Critically damped: \(c^2 = 4mk\) — repeated real roots, fastest return to equilibrium.
  • Overdamped: \(c^2 > 4mk\) — real distinct roots, exponential decay, no oscillations.
Example (underdamped): \(\ddot{x}+2\dot{x}+5x=0\), \(x(0)=1,\;\dot{x}(0)=0\).
AE: \(m^2+2m+5=0 \Rightarrow m=-1\pm 2i\). CF: \(x = e^{-t}(A\cos 2t+B\sin 2t)\).
\(x(0)=A=1\). \(\dot{x}=-e^{-t}(A\cos 2t+B\sin 2t)+e^{-t}(-2A\sin 2t+2B\cos 2t)\).
\(\dot{x}(0)=-A+2B=0 \Rightarrow B=\tfrac{1}{2}\).
\(x = e^{-t}\!\left(\cos 2t+\tfrac{1}{2}\sin 2t\right)\).

Euler–Cauchy Equations

The Euler–Cauchy equation \(x^2\dfrac{d^2y}{dx^2}+px\dfrac{dy}{dx}+qy = 0\) is solved by the substitution \(x = e^t\) (i.e. \(t=\ln x\)), which transforms it into a constant-coefficient ODE in \(t\).
Example: Solve \(x^2y''-2y=0\).
With \(x=e^t\): \(x^2\dfrac{d^2y}{dx^2} = \dfrac{d^2y}{dt^2}-\dfrac{dy}{dt}\).
Equation becomes: \(\dfrac{d^2y}{dt^2}-\dfrac{dy}{dt}-2y=0\).
AE: \(m^2-m-2=0 \Rightarrow (m-2)(m+1)=0 \Rightarrow m=2,-1\).
GS in \(t\): \(y=Ae^{2t}+Be^{-t}\). Back-substitute \(t=\ln x\): \(y=Ax^2+Bx^{-1}\).
Note for AQA: Euler–Cauchy equations may appear in AQA Further Pure papers. The substitution \(x=e^t\) reduces them to constant-coefficient form. Try a power-law solution \(y=x^m\) directly: substituting gives the indicial equation \(m(m-1)+pm+q=0\).

Superposition Principle

For a linear ODE \(Ly = f(x)\), if \(y_1\) is a solution when \(f=f_1\) and \(y_2\) when \(f=f_2\), then \(y_1+y_2\) is a solution when \(f=f_1+f_2\).
This allows complex RHS functions to be split into simpler parts, each solved separately.
Example: To solve \(y''+y = \sin x + e^x\), find PI for \(\sin x\) and PI for \(e^x\) separately, then add.
PI\(_1\) for \(\sin x\): since \(m=\pm i\) are roots of AE, there is resonance. Try \(y_{p1}=x(A\cos x+B\sin x)\).
PI\(_2\) for \(e^x\): try \(y_{p2}=\lambda e^x\). \(\lambda e^x+\lambda e^x = e^x \Rightarrow \lambda=\tfrac{1}{2}\).
GS: \(y = A\cos x+B\sin x + y_{p1} + \tfrac{1}{2}e^x\).
