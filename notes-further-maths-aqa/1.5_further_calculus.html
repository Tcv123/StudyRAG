1.5: Further Calculus

1. Arc Length

Arc length (Cartesian): The length of the curve \(y = f(x)\) from \(x = a\) to \(x = b\): $$L = \int_a^b \sqrt{1 + \left(\frac{dy}{dx}\right)^2}\,dx$$ Arc length (parametric): If \(x = x(t)\), \(y = y(t)\), from \(t = t_1\) to \(t = t_2\): $$L = \int_{t_1}^{t_2} \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2}\,dt$$
Example: Find the arc length of \(y = \dfrac{2}{3}x^{3/2}\) from \(x = 0\) to \(x = 3\).

\(\dfrac{dy}{dx} = x^{1/2}\), so \(1 + \left(\dfrac{dy}{dx}\right)^2 = 1 + x\)

\(L = \int_0^3 \sqrt{1+x}\,dx = \left[\dfrac{2}{3}(1+x)^{3/2}\right]_0^3 = \dfrac{2}{3}(4^{3/2}) - \dfrac{2}{3}(1) = \dfrac{2}{3}(8) - \dfrac{2}{3} = \dfrac{14}{3}\)
AQA Exam Tip: AQA often chooses curves where \(1 + (dy/dx)^2\) simplifies to a perfect square. If your integrand doesn't simplify nicely, recheck your differentiation.

2. Surface Area of Revolution

Surface area of revolution about the \(x\)-axis: $$S = 2\pi \int_a^b y\sqrt{1+\left(\frac{dy}{dx}\right)^2}\,dx$$ Parametric form: $$S = 2\pi \int_{t_1}^{t_2} y\sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2}\,dt$$
Example: Find the surface area generated when the curve \(y = x^2\) from \(x = 0\) to \(x = 1\) is rotated \(2\pi\) about the \(x\)-axis.

\(\dfrac{dy}{dx} = 2x\), so \(1 + (2x)^2 = 1 + 4x^2\)

\(S = 2\pi\int_0^1 x^2\sqrt{1+4x^2}\,dx\)
(This integral requires substitution \(x = \frac{1}{2}\tan\theta\) — see integration section for technique.)

Result after full working: \(S = \dfrac{\pi}{6}\!\left(5\sqrt{5}-1\right)\)

3. Mean Value of a Function

Mean value: The mean value of \(f(x)\) over \([a, b]\) is: $$\bar{f} = \frac{1}{b-a}\int_a^b f(x)\,dx$$
Example: Find the mean value of \(f(x) = x^2 + 1\) over \([0, 3]\).

\(\bar{f} = \dfrac{1}{3}\int_0^3 (x^2+1)\,dx = \dfrac{1}{3}\left[\dfrac{x^3}{3}+x\right]_0^3 = \dfrac{1}{3}(9+3) = 4\)

4. Reduction Formulae

A reduction formula expresses an integral \(I_n\) in terms of \(I_{n-1}\) (or \(I_{n-2}\)), allowing evaluation by repeatedly reducing the index.

Example: Derive a reduction formula for \(I_n = \displaystyle\int_0^{\pi/2} \sin^n x\,dx\).

Write \(\sin^n x = \sin^{n-1}x \cdot \sin x\) and integrate by parts:
Let \(u = \sin^{n-1}x\), \(dv = \sin x\,dx \Rightarrow v = -\cos x\)

\(I_n = \left[-\cos x\,\sin^{n-1}x\right]_0^{\pi/2} + (n-1)\int_0^{\pi/2}\cos^2 x\,\sin^{n-2}x\,dx\)
\(= 0 + (n-1)\int_0^{\pi/2}(1-\sin^2 x)\sin^{n-2}x\,dx = (n-1)(I_{n-2} - I_n)\)

\(I_n + (n-1)I_n = (n-1)I_{n-2} \Rightarrow \boxed{I_n = \dfrac{n-1}{n}I_{n-2}}\)

With: \(I_0 = \dfrac{\pi}{2}\), \(I_1 = 1\).

So: \(I_2 = \dfrac{1}{2}\cdot\dfrac{\pi}{2} = \dfrac{\pi}{4}\), \(I_4 = \dfrac{3}{4}\cdot\dfrac{\pi}{4} = \dfrac{3\pi}{16}\), \(I_3 = \dfrac{2}{3}\), \(I_5 = \dfrac{8}{15}\), etc.
Example: Derive a reduction formula for \(I_n = \displaystyle\int x^n e^x\,dx\).

Integrate by parts: \(u = x^n\), \(dv = e^x dx\)
\(I_n = x^n e^x - n\int x^{n-1}e^x\,dx = x^n e^x - n\,I_{n-1}\)

Reduction formula: \(I_n = x^n e^x - n\,I_{n-1}\)
Starting from \(I_0 = e^x + C\).
AQA Exam Tip: When asked to "prove" a reduction formula, show integration by parts in full. When asked to "use" it, apply it repeatedly, citing the formula each time. AQA often asks for a specific value like \(I_4\) or \(I_5\) — make sure you track the base cases.

5. Improper Integrals

An integral is improper if:
  • The interval of integration is infinite: \(\displaystyle\int_a^{\infty} f(x)\,dx = \lim_{t\to\infty}\int_a^t f(x)\,dx\)
  • The integrand has a discontinuity in \([a,b]\): \(\displaystyle\int_a^b f(x)\,dx = \lim_{\epsilon\to 0^+}\int_{a+\epsilon}^b f(x)\,dx\)
The integral converges if the limit exists and is finite; otherwise it diverges.
Example: Determine whether \(\displaystyle\int_1^{\infty} \dfrac{1}{x^2}\,dx\) converges.

\(\int_1^t x^{-2}\,dx = \left[-x^{-1}\right]_1^t = -\dfrac{1}{t} + 1\)

As \(t \to \infty\): \(1 - \dfrac{1}{t} \to 1\). Converges to 1.
Example: Show that \(\displaystyle\int_1^{\infty} \dfrac{1}{x}\,dx\) diverges.

\(\int_1^t \dfrac{1}{x}\,dx = [\ln x]_1^t = \ln t \to \infty\) as \(t \to \infty\). Diverges.
p-test: \(\displaystyle\int_1^{\infty} x^{-p}\,dx\) converges if and only if \(p > 1\). Similarly, \(\displaystyle\int_0^1 x^{-p}\,dx\) converges if and only if \(p < 1\).

6. Integration Techniques

Integration by Partial Fractions

Decompose the rational function into partial fractions, then integrate each term.

Example: Find \(\displaystyle\int \dfrac{3}{(x-1)(x+2)}\,dx\).

\(\dfrac{3}{(x-1)(x+2)} = \dfrac{1}{x-1} - \dfrac{1}{x+2}\)

\(\int = \ln|x-1| - \ln|x+2| + C = \ln\left|\dfrac{x-1}{x+2}\right| + C\)

Inverse Trigonometric Integrals

$$\int \frac{1}{\sqrt{a^2-x^2}}\,dx = \arcsin\!\left(\frac{x}{a}\right) + C, \quad |x| < a$$ $$\int \frac{1}{a^2+x^2}\,dx = \frac{1}{a}\arctan\!\left(\frac{x}{a}\right) + C$$
Example: Find \(\displaystyle\int \dfrac{1}{\sqrt{9-x^2}}\,dx\).

Using the formula with \(a = 3\): \(\arcsin\!\left(\dfrac{x}{3}\right) + C\)
Example: Find \(\displaystyle\int_0^2 \dfrac{1}{4+x^2}\,dx\).

\(= \left[\dfrac{1}{2}\arctan\!\left(\dfrac{x}{2}\right)\right]_0^2 = \dfrac{1}{2}\arctan(1) - 0 = \dfrac{1}{2}\cdot\dfrac{\pi}{4} = \dfrac{\pi}{8}\)

Completing the Square for Integration

Integrals of the form \(\dfrac{1}{ax^2+bx+c}\) or \(\dfrac{1}{\sqrt{ax^2+bx+c}}\) require completing the square to match standard forms.

Example: Find \(\displaystyle\int \dfrac{1}{x^2+6x+13}\,dx\).

Complete the square: \(x^2+6x+13 = (x+3)^2 + 4\)

\(\int\dfrac{1}{(x+3)^2+4}\,dx = \dfrac{1}{2}\arctan\!\left(\dfrac{x+3}{2}\right) + C\)

8. Practice Checklist

  • Can you calculate arc length and surface area for both Cartesian and parametric curves?
  • Can you find the mean value of a function over an interval?
  • Can you derive and apply reduction formulae for \(\int \sin^n x\,dx\), \(\int \cos^n x\,dx\), and \(\int x^n e^x\,dx\)?
  • Can you evaluate improper integrals and determine convergence/divergence?
  • Can you integrate using inverse trigonometric functions and completing the square?
