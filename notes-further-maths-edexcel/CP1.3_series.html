CP1.3: Series

1. Sigma Notation

Sigma notation: \(\displaystyle\sum_{r=1}^{n} f(r)\) means the sum of \(f(r)\) for \(r = 1, 2, 3, \ldots, n\).
  • \(r\) is the index of summation
  • \(1\) is the lower limit; \(n\) is the upper limit
  • \(n\) may be a specific number or a general variable

Basic Properties

$$\sum_{r=1}^{n}[f(r) + g(r)] = \sum_{r=1}^{n}f(r) + \sum_{r=1}^{n}g(r)$$
$$\sum_{r=1}^{n} c \cdot f(r) = c\sum_{r=1}^{n}f(r) \quad \text{(constant factor)}$$
$$\sum_{r=1}^{n} k = kn \quad \text{(sum of } n \text{ copies of constant } k\text{)}$$

Partial sums: to find \(\displaystyle\sum_{r=m}^{n} f(r)\), compute \(\displaystyle\sum_{r=1}^{n} f(r) - \sum_{r=1}^{m-1} f(r)\).

2. Standard Results

The four standard summation formulae (must be memorised):
SumFormula
\(\displaystyle\sum_{r=1}^{n} 1 = n\)Trivial
\(\displaystyle\sum_{r=1}^{n} r\)\(\dfrac{n(n+1)}{2}\)
\(\displaystyle\sum_{r=1}^{n} r^2\)\(\dfrac{n(n+1)(2n+1)}{6}\)
\(\displaystyle\sum_{r=1}^{n} r^3\)\(\left[\dfrac{n(n+1)}{2}\right]^2\)
Exam tip: These formulae ARE given in the Edexcel formula booklet, but you should recognise them instantly and know how to apply them. Note that \(\sum r^3 = \left(\sum r\right)^2\) — a beautiful identity worth remembering.

Using the Standard Results

Example: Find \(\displaystyle\sum_{r=1}^{n}(3r^2 - 2r + 5)\).

Split the sum: $$\sum_{r=1}^{n}(3r^2 - 2r + 5) = 3\sum_{r=1}^{n}r^2 - 2\sum_{r=1}^{n}r + 5\sum_{r=1}^{n}1$$ $$= 3 \cdot \frac{n(n+1)(2n+1)}{6} - 2 \cdot \frac{n(n+1)}{2} + 5n$$ $$= \frac{n(n+1)(2n+1)}{2} - n(n+1) + 5n$$ $$= \frac{n}{2}\Big[(n+1)(2n+1) - 2(n+1) + 10\Big]$$ $$= \frac{n}{2}\Big[2n^2 + 3n + 1 - 2n - 2 + 10\Big]$$ $$= \frac{n}{2}(2n^2 + n + 9)$$ $$= \frac{n(2n^2 + n + 9)}{2}$$
Example (partial sum): Find \(\displaystyle\sum_{r=10}^{20} r^2\).

$$\sum_{r=10}^{20} r^2 = \sum_{r=1}^{20} r^2 - \sum_{r=1}^{9} r^2$$ $$= \frac{20 \cdot 21 \cdot 41}{6} - \frac{9 \cdot 10 \cdot 19}{6}$$ $$= \frac{17220}{6} - \frac{1710}{6} = 2870 - 285 = 2585$$

3. Method of Differences

Method of differences (telescoping): If the general term \(f(r)\) can be written as \(g(r) - g(r+1)\) (or \(g(r+1) - g(r)\)) for some function \(g\), then the sum telescopes: $$\sum_{r=1}^{n}[g(r) - g(r+1)] = g(1) - g(n+1)$$ Most terms cancel when the sum is expanded, leaving only the first few and last few terms.

Setting Up Telescoping via Partial Fractions

When the general term involves products of consecutive integers in the denominator, use partial fractions to express it as a difference, then sum by telescoping.

Example: Find \(\displaystyle\sum_{r=1}^{n} \frac{1}{r(r+1)}\).

Step 1: Partial fractions.
\(\dfrac{1}{r(r+1)} = \dfrac{1}{r} - \dfrac{1}{r+1}\)

Step 2: Write out the telescoping sum.
$$\sum_{r=1}^{n}\left(\frac{1}{r} - \frac{1}{r+1}\right) = \left(\frac{1}{1} - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \cdots + \left(\frac{1}{n} - \frac{1}{n+1}\right)$$
Step 3: Cancel telescoping terms.
All intermediate terms cancel, leaving: $$= 1 - \frac{1}{n+1} = \frac{n}{n+1}$$
Example: Find \(\displaystyle\sum_{r=1}^{n} \frac{1}{r(r+2)}\).

Step 1: \(\dfrac{1}{r(r+2)} = \dfrac{A}{r} + \dfrac{B}{r+2}\)
\(1 = A(r+2) + Br\)
\(r=0: A = \frac{1}{2}\); \(r=-2: B = -\frac{1}{2}\)
So \(\dfrac{1}{r(r+2)} = \dfrac{1}{2}\left(\dfrac{1}{r} - \dfrac{1}{r+2}\right)\)

Step 2: Telescoping table.
$$\sum_{r=1}^{n} = \frac{1}{2}\left[\left(\frac{1}{1} - \frac{1}{3}\right) + \left(\frac{1}{2} - \frac{1}{4}\right) + \left(\frac{1}{3} - \frac{1}{5}\right) + \cdots\right]$$
The pattern leaves (writing the last two surviving terms): $$= \frac{1}{2}\left[1 + \frac{1}{2} - \frac{1}{n+1} - \frac{1}{n+2}\right] = \frac{1}{2}\left[\frac{3}{2} - \frac{1}{n+1} - \frac{1}{n+2}\right]$$ $$= \frac{3}{4} - \frac{1}{2(n+1)} - \frac{1}{2(n+2)}$$
Exam tip: Write out at least the first 3–4 rows AND the last 2–3 rows of the telescoping sum clearly. Show the cancellation explicitly. Examiners award method marks for this clear layout. Common error: forgetting to include the \(\frac{1}{2}\) factor from partial fractions.

Telescoping with Polynomial Terms

Example: Use the fact that \(r(r+1)(r+2) - (r-1)r(r+1) = 3r(r+1)\) to find \(\displaystyle\sum_{r=1}^{n} r(r+1)\).

Let \(g(r) = r(r+1)(r+2)\). Then:
\(g(r) - g(r-1) = r(r+1)(r+2) - (r-1)r(r+1) = r(r+1)\big[(r+2)-(r-1)\big] = 3r(r+1)\)

So: \(r(r+1) = \frac{1}{3}[g(r) - g(r-1)]\)

Summing from \(r=1\) to \(n\): $$\sum_{r=1}^{n} r(r+1) = \frac{1}{3}[g(n) - g(0)] = \frac{1}{3}[n(n+1)(n+2) - 0] = \frac{n(n+1)(n+2)}{3}$$

4. Further Manipulation of Sigma Notation

Changing Limits

Example: Find \(\displaystyle\sum_{r=3}^{n}(r^2+1)\) in terms of \(n\).

$$\sum_{r=3}^{n}(r^2+1) = \sum_{r=1}^{n}(r^2+1) - \sum_{r=1}^{2}(r^2+1)$$ $$= \left[\frac{n(n+1)(2n+1)}{6} + n\right] - \left[(1+1)+(4+1)\right]$$ $$= \frac{n(n+1)(2n+1)}{6} + n - 7$$

Recognising Standard Forms

Always expand brackets and group powers of \(r\) before applying standard results:

Example: Find \(\displaystyle\sum_{r=1}^{n}(2r-1)^2\).

Expand: \((2r-1)^2 = 4r^2 - 4r + 1\)

$$\sum_{r=1}^{n}(4r^2 - 4r + 1) = 4 \cdot \frac{n(n+1)(2n+1)}{6} - 4 \cdot \frac{n(n+1)}{2} + n$$ $$= \frac{2n(n+1)(2n+1)}{3} - 2n(n+1) + n$$ $$= \frac{n}{3}\Big[2(n+1)(2n+1) - 6(n+1) + 3\Big]$$ $$= \frac{n}{3}(4n^2 + 6n + 2 - 6n - 6 + 3) = \frac{n(4n^2-1)}{3} = \frac{n(2n-1)(2n+1)}{3}$$

6. Common Exam Question Types

  1. Express and evaluate: Write an expression in sigma notation, use standard results to find a closed form, then evaluate for a specific \(n\).
  2. Partial sum: Use \(\sum_{r=m}^{n} = \sum_{r=1}^{n} - \sum_{r=1}^{m-1}\).
  3. Method of differences: Given partial fractions form, write telescoping expansion, cancel terms, state final sum.
  4. Proof-style: Show that \(\sum_{r=1}^{n} f(r) = \frac{1}{6}n(n+1)(2n+1)\) etc. by expanding and collecting.
Exam tip: When using method of differences in an exam, always write the sum as a specific expansion (show rows \(r=1,2,3,\ldots, n-1, n\)) with at least 3–4 rows at the start and end. Never just write the final answer — show the cancellation. The mark scheme specifically looks for this layout.
