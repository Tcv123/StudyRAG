P14: Sequences and Series

1. Arithmetic Sequences

An arithmetic sequence has a constant common difference \(d\) between consecutive terms.
First term: \(a\), common difference: \(d\).
$$u_n = a + (n-1)d$$
$$S_n = \frac{n}{2}(2a + (n-1)d) = \frac{n}{2}(a + l)$$

where \(l\) is the last term.

Example: An arithmetic sequence has first term 5 and common difference 3. Find \(u_{20}\) and \(S_{20}\).

\(u_{20} = 5 + 19 \times 3 = 62\)
\(S_{20} = \dfrac{20}{2}(5 + 62) = 10 \times 67 = 670\)
Exam Tip: For problems involving the sum, remember \(S_n - S_{n-1} = u_n\). This is useful for finding terms from given sums.

2. Geometric Sequences

A geometric sequence has a constant common ratio \(r\) between consecutive terms.
$$u_n = ar^{n-1}$$
$$S_n = \frac{a(r^n - 1)}{r - 1} = \frac{a(1 - r^n)}{1 - r} \quad (r \neq 1)$$
$$S_\infty = \frac{a}{1-r} \quad \text{only when } |r| < 1$$

Convergence Condition

A geometric series converges (has a finite sum to infinity) if and only if \(|r| < 1\), i.e. \(-1 < r < 1\).
Example: A geometric series has first term 12 and common ratio \(\frac{1}{3}\). Find \(S_\infty\).

Since \(\left|\frac{1}{3}\right| < 1\), the series converges.
\(S_\infty = \dfrac{12}{1 - \frac{1}{3}} = \dfrac{12}{\frac{2}{3}} = 18\)

3. Recurrence Relations

A recurrence relation defines each term in terms of previous terms:
$$u_{n+1} = f(u_n), \quad u_1 = k$$
Example: \(u_{n+1} = 2u_n - 3\), \(u_1 = 5\). Find the first four terms.

\(u_1 = 5\), \(u_2 = 2(5)-3 = 7\), \(u_3 = 2(7)-3 = 11\), \(u_4 = 2(11)-3 = 19\)

A sequence is periodic if it repeats a cycle of values. A sequence is increasing/decreasing if each term is greater/less than the previous.

4. Sigma Notation and Standard Results

$$\sum_{r=1}^{n} r = \frac{n(n+1)}{2}$$
$$\sum_{r=1}^{n} r^2 = \frac{n(n+1)(2n+1)}{6}$$
$$\sum_{r=1}^{n} r^3 = \left[\frac{n(n+1)}{2}\right]^2$$
$$\sum_{r=1}^{n} c = nc \quad \text{(constant } c\text{)}$$
Example: Find \(\displaystyle\sum_{r=1}^{20}(3r^2 - r)\).

\(= 3\sum_{r=1}^{20} r^2 - \sum_{r=1}^{20} r = 3 \cdot \dfrac{20 \cdot 21 \cdot 41}{6} - \dfrac{20 \cdot 21}{2} = 3 \times 2870 - 210 = 8610 - 210 = 8400\)

5. Binomial Expansion (Positive Integer \(n\))

For positive integer \(n\): $$(a + b)^n = \sum_{r=0}^{n} \binom{n}{r} a^{n-r} b^r$$ where \(\dbinom{n}{r} = \dfrac{n!}{r!(n-r)!}\)

Pascal's Triangle (first 6 rows)

Example: Expand \((2 + x)^4\).

Using row 4 of Pascal's triangle: coefficients 1, 4, 6, 4, 1.
\((2+x)^4 = 1(2)^4 + 4(2)^3 x + 6(2)^2 x^2 + 4(2)x^3 + x^4\)
\(= 16 + 32x + 24x^2 + 8x^3 + x^4\)

6. Binomial Expansion for Any Rational \(n\)

For any rational \(n\) and \(|x| < 1\): $$(1+x)^n = 1 + nx + \frac{n(n-1)}{2!}x^2 + \frac{n(n-1)(n-2)}{3!}x^3 + \cdots$$ This is an infinite series valid only for \(|x| < 1\).

For \((a + bx)^n\), write as \(a^n\!\left(1 + \dfrac{b}{a}x\right)^n\), valid for \(\left|\dfrac{b}{a}x\right| < 1\).

Example: Find the first three terms of \((1-2x)^{-1}\), stating the range of validity.

\((1-2x)^{-1} = 1 + (-1)(-2x) + \dfrac{(-1)(-2)}{2}(-2x)^2 + \cdots\)
\(= 1 + 2x + 4x^2 + \cdots\)
Valid for \(|{-2x}| < 1 \Rightarrow |x| < \dfrac{1}{2}\)
Finding a specific term: Find the coefficient of \(x^3\) in \((2+3x)^7\).

General term: \(\dbinom{7}{3}(2)^4(3x)^3 = 35 \times 16 \times 27x^3 = 15120x^3\)
Coefficient: \(15120\)
Exam Tip: Always state the range of validity for any binomial expansion with rational \(n\). The series only approximates the function within this range. Do NOT use Pascal's triangle for rational (non-integer positive) \(n\).
