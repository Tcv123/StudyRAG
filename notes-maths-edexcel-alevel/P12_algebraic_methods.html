P12: Algebraic Methods

1. Algebraic Division

Polynomial long division allows us to divide one polynomial by another. The result takes the form:

$$\frac{f(x)}{d(x)} = q(x) + \frac{r(x)}{d(x)}$$

where \(q(x)\) is the quotient and \(r(x)\) is the remainder (with degree less than the divisor).

Method: Long Division Layout

To divide \(2x^3 + 3x^2 - 4x + 1\) by \((x + 2)\):

Worked Example: Divide \(2x^3 + 3x^2 - 4x + 1\) by \((x + 2)\).

Step 1: \(2x^3 \div x = 2x^2\). Multiply: \(2x^2(x+2) = 2x^3 + 4x^2\). Subtract: \(-x^2 - 4x + 1\).
Step 2: \(-x^2 \div x = -x\). Multiply: \(-x(x+2) = -x^2 - 2x\). Subtract: \(-2x + 1\).
Step 3: \(-2x \div x = -2\). Multiply: \(-2(x+2) = -2x - 4\). Subtract: \(5\).

Result: \(2x^3 + 3x^2 - 4x + 1 = (x+2)(2x^2 - x - 2) + 5\)

2. Factor Theorem and Remainder Theorem

Factor Theorem: If \(f(a) = 0\) then \((x - a)\) is a factor of \(f(x)\).
Conversely, if \((x - a)\) is a factor, then \(f(a) = 0\).
Remainder Theorem: When \(f(x)\) is divided by \((x - a)\), the remainder is \(f(a)\).
More generally, when divided by \((ax - b)\), the remainder is \(f\!\left(\dfrac{b}{a}\right)\).
Example: Show \((x - 2)\) is a factor of \(f(x) = x^3 - 3x^2 + 4\).

\(f(2) = 8 - 12 + 4 = 0\) ✓ So \((x-2)\) is a factor.
Exam Tip: When finding factors by trial, test \(x = \pm 1, \pm 2, \pm 3, \ldots\) systematically. Once one factor is found, use long division to find the remaining quadratic, then factorise further if possible.

3. Factorising Cubics

The general method to fully factorise a cubic \(f(x)\):

  1. Find a value \(a\) such that \(f(a) = 0\) by inspection/trial
  2. Divide \(f(x)\) by \((x - a)\) to obtain a quadratic quotient \(q(x)\)
  3. Factorise \(q(x)\) by inspection or the quadratic formula
  4. Write \(f(x) = (x - a) \cdot q(x)\) fully factorised
Example: Fully factorise \(f(x) = x^3 - x^2 - 4x + 4\).

Try \(x = 1\): \(f(1) = 1 - 1 - 4 + 4 = 0\) ✓
Divide by \((x-1)\): \(f(x) = (x-1)(x^2 - 4)\)
Factorise: \(f(x) = (x-1)(x-2)(x+2)\)

4. Partial Fractions

Partial fractions decompose a rational expression into simpler fractions. Three main cases:

Case 1: Distinct Linear Factors

$$\frac{px + q}{(ax+b)(cx+d)} \equiv \frac{A}{ax+b} + \frac{B}{cx+d}$$

Case 2: Repeated Linear Factor

$$\frac{px + q}{(ax+b)^2} \equiv \frac{A}{ax+b} + \frac{B}{(ax+b)^2}$$

For three factors with one repeated: \(\dfrac{f(x)}{(ax+b)^2(cx+d)} \equiv \dfrac{A}{ax+b} + \dfrac{B}{(ax+b)^2} + \dfrac{C}{cx+d}\)

Case 3: Improper Fractions

When the degree of the numerator is greater than or equal to the degree of the denominator, perform polynomial division first, then apply partial fractions to the remainder.

Worked Example — Partial Fractions Method:

Express \(\dfrac{5x+1}{(x+1)(x-2)}\) in partial fractions.

Let \(\dfrac{5x+1}{(x+1)(x-2)} \equiv \dfrac{A}{x+1} + \dfrac{B}{x-2}\)

Multiply both sides by \((x+1)(x-2)\):
\(5x + 1 \equiv A(x-2) + B(x+1)\)

Set \(x = 2\): \(11 = 3B\), so \(B = \dfrac{11}{3}\)
Set \(x = -1\): \(-4 = -3A\), so \(A = \dfrac{4}{3}\)

Answer: \(\dfrac{4/3}{x+1} + \dfrac{11/3}{x-2}\)

5. Proof by Contradiction

Proof by Contradiction: Assume the negation of what you want to prove. Show this assumption leads to a logical contradiction. Therefore the original statement must be true.

Structure of a Proof by Contradiction

  1. State: "Assume for contradiction that [negation of statement] is true."
  2. Use valid logical steps from this assumption.
  3. Arrive at a contradiction (e.g. a number is both even and odd, or \(p\) and \(\neg p\)).
  4. Conclude: "This is a contradiction, so the original statement must be true."

Classic Example 1: \(\sqrt{2}\) is Irrational

Proof: Assume \(\sqrt{2}\) is rational. Then \(\sqrt{2} = \dfrac{p}{q}\) where \(p, q \in \mathbb{Z}\), \(q \neq 0\), and \(\gcd(p,q) = 1\) (fully reduced).

Then \(2 = \dfrac{p^2}{q^2}\), so \(p^2 = 2q^2\).
This means \(p^2\) is even, so \(p\) is even. Write \(p = 2m\).
Then \((2m)^2 = 2q^2 \Rightarrow 4m^2 = 2q^2 \Rightarrow q^2 = 2m^2\).
So \(q\) is also even. But then \(p\) and \(q\) share factor 2 — contradiction with \(\gcd(p,q)=1\).
Therefore \(\sqrt{2}\) is irrational. \(\square\)

Classic Example 2: Infinitely Many Primes

Proof: Assume there are finitely many primes: \(p_1, p_2, \ldots, p_n\).

Consider \(N = p_1 \cdot p_2 \cdots p_n + 1\).
Dividing \(N\) by any \(p_i\) gives remainder 1, so no \(p_i\) divides \(N\).
But every integer \(> 1\) has a prime factor, so \(N\) must have a prime factor not in our list — contradiction.
Therefore there are infinitely many primes. \(\square\)
Exam Tip: You must explicitly state the assumption, reach a clear contradiction, and conclude. Simply finding something "odd" is not sufficient — name the contradiction precisely.
