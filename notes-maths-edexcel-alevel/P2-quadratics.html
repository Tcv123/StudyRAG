P2: Quadratics

1. Solving Quadratic Equations

A quadratic equation has the form \(ax^2 + bx + c = 0\) where \(a \neq 0\). It has at most two solutions (roots).

Method 1: Factorising

If the quadratic factorises neatly, this is the quickest method.

Example: Solve \(x^2 - 5x + 6 = 0\).

Factorise: \((x-2)(x-3) = 0\)
So \(x = 2\) or \(x = 3\)

Method 2: Quadratic Formula

Always works, even when factorising is impossible:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
Example: Solve \(2x^2 + 3x - 1 = 0\).

\(a=2,\; b=3,\; c=-1\)
\(x = \dfrac{-3 \pm \sqrt{9 + 8}}{4} = \dfrac{-3 \pm \sqrt{17}}{4}\)
\(x = \dfrac{-3 + \sqrt{17}}{4} \approx 0.28\) or \(x = \dfrac{-3 - \sqrt{17}}{4} \approx -1.78\)

Method 3: Completing the Square

Rewrite \(ax^2 + bx + c\) in the form \(a(x+p)^2 + q\):

$$ax^2 + bx + c = a\!\left(x + \frac{b}{2a}\right)^2 + c - \frac{b^2}{4a}$$
Example: Solve \(x^2 - 6x + 4 = 0\) by completing the square.

\((x-3)^2 - 9 + 4 = 0\)
\((x-3)^2 = 5\)
\(x - 3 = \pm\sqrt{5}\)
\(x = 3 \pm \sqrt{5}\)
Exam Tip: Completing the square is often asked explicitly. Also use it to find the vertex of a parabola: \(a(x+p)^2 + q\) has vertex \((-p,\, q)\).

2. The Discriminant

The discriminant \(\Delta = b^2 - 4ac\) determines how many real roots the quadratic has.
DiscriminantNumber of real rootsGraph crosses x-axis
\(\Delta > 0\)Two distinct real rootsAt two points
\(\Delta = 0\)One repeated (equal) rootTouches (tangent) at one point
\(\Delta < 0\)No real rootsDoes not cross x-axis
Example: Find the values of \(k\) for which \(x^2 + kx + 9 = 0\) has two distinct real roots.

Require \(\Delta > 0\): \(k^2 - 36 > 0\)
\((k-6)(k+6) > 0\)
\(k < -6\) or \(k > 6\)

3. Quadratic Graphs

Key features of \(y = ax^2 + bx + c\):

  • Vertex: \(\left(-\dfrac{b}{2a},\; c - \dfrac{b^2}{4a}\right)\) — minimum if \(a>0\), maximum if \(a<0\)
  • Axis of symmetry: \(x = -\dfrac{b}{2a}\)
  • y-intercept: \((0, c)\)
  • x-intercepts (roots): found using the quadratic formula
  • Parabola opens upward if \(a > 0\), downward if \(a < 0\)

4. Quadratic Inequalities

To solve a quadratic inequality, always sketch the parabola first and identify where it is above/below the x-axis.

Example: Solve \(x^2 - x - 6 < 0\).

Step 1: Factorise: \((x-3)(x+2) = 0\), roots \(x = -2\) and \(x = 3\)
Step 2: Parabola opens upward (\(a=1>0\)), so it is below the x-axis between the roots.
Answer: \(-2 < x < 3\)
Example: Solve \(2x^2 - 5x + 2 \geq 0\).

Factorise: \((2x-1)(x-2) = 0\), roots \(x = \frac{1}{2}\) and \(x = 2\)
Parabola opens upward, so it is above or on the x-axis outside the roots.
Answer: \(x \leq \dfrac{1}{2}\) or \(x \geq 2\)
Exam Tip: "Between the roots" for \(< 0\) when \(a > 0\). "Outside the roots" for \(> 0\) when \(a > 0\). A quick sketch prevents sign errors.

5. Disguised Quadratics

Some equations look non-quadratic but become quadratic after a substitution.

Example: Solve \(x^4 - 5x^2 + 4 = 0\).

Let \(u = x^2\): \(u^2 - 5u + 4 = 0\)
\((u-1)(u-4) = 0\) → \(u = 1\) or \(u = 4\)
Back-substitute: \(x^2 = 1 \Rightarrow x = \pm 1\); \(x^2 = 4 \Rightarrow x = \pm 2\)
Example: Solve \(e^{2x} - 7e^x + 10 = 0\).

Let \(u = e^x\): \(u^2 - 7u + 10 = 0\) → \((u-2)(u-5) = 0\)
\(e^x = 2 \Rightarrow x = \ln 2\); \(e^x = 5 \Rightarrow x = \ln 5\)
Exam Tip: Common substitutions: \(u = x^2\), \(u = e^x\), \(u = \sin x\), \(u = \sqrt{x}\). Always state your substitution clearly and back-substitute at the end.
