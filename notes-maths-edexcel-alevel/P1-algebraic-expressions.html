P1: Algebraic Expressions

1. Index Laws

Index Laws — rules that govern how powers (indices/exponents) combine when multiplying, dividing, or raising to a power.
LawRuleExample
Multiplication\(a^m \times a^n = a^{m+n}\)\(x^3 \times x^5 = x^8\)
Division\(a^m \div a^n = a^{m-n}\)\(y^7 \div y^2 = y^5\)
Power of a power\((a^m)^n = a^{mn}\)\((z^4)^3 = z^{12}\)
Zero index\(a^0 = 1\) (for \(a \neq 0\))\(5^0 = 1\)
Negative index\(a^{-n} = \dfrac{1}{a^n}\)\(x^{-3} = \dfrac{1}{x^3}\)
Fractional index\(a^{1/n} = \sqrt[n]{a}\)\(8^{1/3} = \sqrt[3]{8} = 2\)
Fractional index (general)\(a^{m/n} = \left(\sqrt[n]{a}\right)^m\)\(27^{2/3} = (\sqrt[3]{27})^2 = 9\)
Exam Tip: When evaluating \(a^{m/n}\), always take the root first then raise to the power — this keeps the numbers smaller and avoids errors. So \(8^{2/3} = (\sqrt[3]{8})^2 = 4\), not \(\sqrt[3]{64} = 4\) (same answer, but the first method is safer for large numbers).
Example: Simplify \(\dfrac{x^5 \times x^{-2}}{x^3}\).

Numerator: \(x^5 \times x^{-2} = x^{5+(-2)} = x^3\)
Then: \(x^3 \div x^3 = x^{3-3} = x^0 = 1\)
Example: Evaluate \(16^{3/4}\).

\(16^{3/4} = \left(\sqrt[4]{16}\right)^3 = 2^3 = 8\)

2. Expanding Brackets

Single Brackets

Multiply each term inside by the factor outside:

$$a(b + c) = ab + ac$$

Double Brackets (FOIL)

$$(a+b)(c+d) = ac + ad + bc + bd$$
Example: Expand \((2x+3)(x-5)\).

\((2x)(x) + (2x)(-5) + (3)(x) + (3)(-5)\) \(= 2x^2 - 10x + 3x - 15\) \(= 2x^2 - 7x - 15\)

Special Identities

IdentityExpansion
\((a+b)^2\)\(a^2 + 2ab + b^2\)
\((a-b)^2\)\(a^2 - 2ab + b^2\)
\((a+b)(a-b)\)\(a^2 - b^2\) (difference of two squares)

Triple Brackets

Expand two brackets first, then multiply the result by the third:

Example: Expand \((x+1)(x-2)(x+3)\).

Step 1: \((x+1)(x-2) = x^2 - x - 2\)
Step 2: \((x^2 - x - 2)(x+3)\) \(= x^3 + 3x^2 - x^2 - 3x - 2x - 6\) \(= x^3 + 2x^2 - 5x - 6\)
Exam Tip: When expanding triple brackets, write out every term systematically and collect like terms carefully. A sign error is the most common source of lost marks.

3. Factorising

Common Factor

Identify the highest common factor (HCF) of all terms and take it outside:

Example: Factorise \(6x^3 - 9x^2 + 3x\).

HCF is \(3x\), so: \(3x(2x^2 - 3x + 1)\)
Then factorise further: \(3x(2x-1)(x-1)\)

Difference of Two Squares

$$a^2 - b^2 = (a+b)(a-b)$$
Example: Factorise \(4x^2 - 25\).

\(4x^2 - 25 = (2x)^2 - 5^2 = (2x+5)(2x-5)\)

Quadratic Trinomials

For \(x^2 + bx + c\): find two numbers that multiply to \(c\) and add to \(b\).

For \(ax^2 + bx + c\) where \(a \neq 1\): find two numbers that multiply to \(ac\) and add to \(b\), then split the middle term.

Example: Factorise \(6x^2 + 7x - 3\).

Need two numbers that multiply to \(6 \times (-3) = -18\) and add to \(7\): these are \(9\) and \(-2\).
\(6x^2 + 9x - 2x - 3 = 3x(2x+3) - 1(2x+3) = (3x-1)(2x+3)\)

4. Surds

A surd is an irrational number expressed as a root that cannot be simplified to a rational number. For example, \(\sqrt{2}\), \(\sqrt{3}\), \(\sqrt[3]{5}\).

Simplifying Surds

$$\sqrt{ab} = \sqrt{a}\,\sqrt{b}$$

To simplify \(\sqrt{n}\), find the largest perfect-square factor:

Example: Simplify \(\sqrt{72}\).

\(\sqrt{72} = \sqrt{36 \times 2} = \sqrt{36}\,\sqrt{2} = 6\sqrt{2}\)

Arithmetic with Surds

OperationRuleExample
Addition/SubtractionOnly like surds combine\(3\sqrt{2} + 5\sqrt{2} = 8\sqrt{2}\)
Multiplication\(\sqrt{a} \times \sqrt{b} = \sqrt{ab}\)\(\sqrt{3} \times \sqrt{12} = \sqrt{36} = 6\)
Square of a surd\((\sqrt{a})^2 = a\)\((\sqrt{5})^2 = 5\)

Rationalising the Denominator

Monomial denominator: Multiply numerator and denominator by the surd.

Example: Rationalise \(\dfrac{3}{\sqrt{5}}\).

\(\dfrac{3}{\sqrt{5}} \times \dfrac{\sqrt{5}}{\sqrt{5}} = \dfrac{3\sqrt{5}}{5}\)

Binomial denominator: Multiply by the conjugate \((a \mp b\sqrt{c})\).

Example: Rationalise \(\dfrac{4}{3 + \sqrt{2}}\).

Conjugate: \(3 - \sqrt{2}\)
\(\dfrac{4}{3+\sqrt{2}} \times \dfrac{3-\sqrt{2}}{3-\sqrt{2}} = \dfrac{4(3-\sqrt{2})}{9-2} = \dfrac{4(3-\sqrt{2})}{7} = \dfrac{12-4\sqrt{2}}{7}\)
Exam Tip: The conjugate of \(a + b\sqrt{c}\) is \(a - b\sqrt{c}\). When you multiply them, the denominator becomes \(a^2 - b^2 c\) — always a rational number.
