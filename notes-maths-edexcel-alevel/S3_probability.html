S3: Probability

1. Basic Probability

$$P(A) = \frac{\text{number of favourable outcomes}}{\text{total number of equally likely outcomes}}$$
  • \(0 \leq P(A) \leq 1\) for any event \(A\).
  • \(P(\text{certain event}) = 1\), \(P(\text{impossible event}) = 0\).
  • The sum of all probabilities in a sample space = 1.

Complement Rule

$$P(A') = 1 - P(A)$$

where \(A'\) is the event "A does not happen".

2. Addition Rule

For any two events A and B: $$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

Mutually Exclusive Events

Two events are mutually exclusive if they cannot both happen at the same time: \(P(A \cap B) = 0\). Therefore:

$$P(A \cup B) = P(A) + P(B)$$
Exam Tip: Do NOT confuse mutually exclusive with independent. Two events can be independent but not mutually exclusive (and vice versa). Mutually exclusive events with non-zero probability are NEVER independent.

3. Independent Events

Events A and B are independent if knowing one occurred gives no information about the other: $$P(A \cap B) = P(A) \times P(B)$$

Equivalently, A and B are independent if and only if \(P(A|B) = P(A)\).

4. Conditional Probability

The probability of A given B has occurred: $$P(A|B) = \frac{P(A \cap B)}{P(B)} \qquad \text{where } P(B) > 0$$
Example: In a class of 30 students: 18 study Maths (M), 12 study Physics (P), 8 study both. Find \(P(M|P)\).
\(P(M \cap P) = \frac{8}{30}\), \(P(P) = \frac{12}{30}\)
\(P(M|P) = \frac{8/30}{12/30} = \frac{8}{12} = \frac{2}{3}\)

5. Venn Diagrams

Venn diagrams represent events as overlapping circles within a rectangle (sample space).

Key regions to identify:

  • \(A \cap B\): inside both circles (intersection)
  • \(A \cup B\): inside at least one circle (union)
  • \(A \cap B'\): inside A only
  • \((A \cup B)'\): outside both circles (neither)

6. Tree Diagrams

Tree diagrams show all possible outcomes of sequential events. Multiply probabilities along branches; add probabilities for different paths to the same outcome.

7. Discrete Probability Distributions

A probability distribution lists each value of a discrete random variable \(X\) with its probability \(P(X = x)\). All probabilities must sum to 1:

$$\sum_{\text{all } x} P(X = x) = 1$$

Expected Value (Mean)

$$E(X) = \sum x \cdot P(X = x)$$

Variance

$$\text{Var}(X) = E(X^2) - [E(X)]^2 = \sum x^2 P(X=x) - \left[\sum x P(X=x)\right]^2$$
Example:
\(x\)1234
\(P(X=x)\)0.10.30.40.2
\(E(X) = 1(0.1) + 2(0.3) + 3(0.4) + 4(0.2) = 0.1 + 0.6 + 1.2 + 0.8 = 2.7\)
\(E(X^2) = 1(0.1) + 4(0.3) + 9(0.4) + 16(0.2) = 0.1 + 1.2 + 3.6 + 3.2 = 8.1\)
\(\text{Var}(X) = 8.1 - 2.7^2 = 8.1 - 7.29 = 0.81\)
