P16: Trigonometric Functions

1. Reciprocal Trigonometric Functions

$$\sec x = \frac{1}{\cos x} \qquad \cosec x = \frac{1}{\sin x} \qquad \cot x = \frac{1}{\tan x} = \frac{\cos x}{\sin x}$$ These are defined wherever the denominator is non-zero.

2. Pythagorean Identities

Starting from \(\sin^2 x + \cos^2 x \equiv 1\):

$$1 + \tan^2 x \equiv \sec^2 x$$
$$1 + \cot^2 x \equiv \cosec^2 x$$

Derived by dividing \(\sin^2 x + \cos^2 x = 1\) by \(\cos^2 x\) and \(\sin^2 x\) respectively.

Example — Simplify: \(\dfrac{\sec^2 x - 1}{\tan x}\)

Since \(\sec^2 x - 1 = \tan^2 x\):
\(\dfrac{\tan^2 x}{\tan x} = \tan x\)

4. Addition Formulae

$$\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B$$
$$\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B$$
$$\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A \tan B}$$
Example: Find the exact value of \(\sin 75°\).

\(\sin 75° = \sin(45° + 30°) = \sin 45°\cos 30° + \cos 45°\sin 30°\)
\(= \dfrac{\sqrt{2}}{2} \cdot \dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2} \cdot \dfrac{1}{2} = \dfrac{\sqrt{6} + \sqrt{2}}{4}\)
Example — Prove: \(\cos(A - B) - \cos(A + B) \equiv 2\sin A \sin B\)

LHS \(= (\cos A\cos B + \sin A\sin B) - (\cos A\cos B - \sin A\sin B)\)
\(= 2\sin A\sin B =\) RHS \(\square\)

5. Double Angle Formulae

$$\sin 2A = 2\sin A\cos A$$
$$\cos 2A = \cos^2 A - \sin^2 A = 2\cos^2 A - 1 = 1 - 2\sin^2 A$$
$$\tan 2A = \frac{2\tan A}{1 - \tan^2 A}$$
Key rearrangements of double angle:
\(\cos^2 A = \dfrac{1 + \cos 2A}{2}\)   |   \(\sin^2 A = \dfrac{1 - \cos 2A}{2}\)
These are essential for integration of \(\sin^2 x\) and \(\cos^2 x\).
Example — Solve: \(\sin 2x = \cos x\) for \(0 \leq x \leq 2\pi\).

\(2\sin x\cos x = \cos x \Rightarrow \cos x(2\sin x - 1) = 0\)
Either \(\cos x = 0 \Rightarrow x = \dfrac{\pi}{2}, \dfrac{3\pi}{2}\)
Or \(\sin x = \dfrac{1}{2} \Rightarrow x = \dfrac{\pi}{6}, \dfrac{5\pi}{6}\)
Solutions: \(x = \dfrac{\pi}{6}, \dfrac{\pi}{2}, \dfrac{5\pi}{6}, \dfrac{3\pi}{2}\)

6. The R-form (Harmonic Form)

Any expression of the form \(a\sin x + b\cos x\) can be written as: $$a\sin x + b\cos x \equiv R\sin(x + \alpha)$$ where: $$R = \sqrt{a^2 + b^2} \qquad \tan\alpha = \frac{b}{a}$$ Similarly: \(a\sin x - b\cos x \equiv R\sin(x - \alpha)\), and \(a\cos x \pm b\sin x \equiv R\cos(x \mp \alpha)\).

Method

  1. Expand \(R\sin(x + \alpha) = R\sin x\cos\alpha + R\cos x\sin\alpha\)
  2. Match coefficients: \(R\cos\alpha = a\) and \(R\sin\alpha = b\)
  3. Solve: \(R = \sqrt{a^2+b^2}\) and \(\alpha = \arctan\!\left(\dfrac{b}{a}\right)\)
Example: Write \(3\sin x + 4\cos x\) in the form \(R\sin(x+\alpha)\).

\(R = \sqrt{9 + 16} = 5\)
\(\tan\alpha = \dfrac{4}{3} \Rightarrow \alpha = \arctan\!\left(\dfrac{4}{3}\right) \approx 0.927\) rad
Answer: \(5\sin(x + 0.927)\)

Maximum value: \(5\), when \(x + 0.927 = \dfrac{\pi}{2}\).
Exam Tip: Always state \(R > 0\) and \(0 < \alpha < \frac{\pi}{2}\). The R-form is used to find maximum/minimum values and to solve equations of the type \(a\sin x + b\cos x = k\).

7. Using Identities to Prove and Simplify

Key strategies for proving identities:

  • Work on one side only (usually the more complex side)
  • Convert everything to \(\sin\) and \(\cos\) if stuck
  • Use Pythagorean identities to substitute \(\sin^2 + \cos^2 = 1\), or equivalents
  • Look for double angle or addition formula opportunities
  • Multiply numerator and denominator by a conjugate if needed
Prove: \(\dfrac{\cosec x}{1 + \cot^2 x} \equiv \sin x\)

LHS: \(\dfrac{\cosec x}{\cosec^2 x} = \dfrac{1}{\cosec x} = \sin x\) = RHS \(\square\)
