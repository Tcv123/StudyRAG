P13: Functions and Graphs

1. Function Notation, Domain and Range

Function: A mapping where every element of the domain maps to exactly one element of the range.

Domain: The set of allowable input values (the \(x\)-values).
Range: The set of output values (the \(f(x)\)-values).

Functions can be described using function notation: \(f : x \mapsto 2x + 1\) or \(f(x) = 2x + 1\).

When stating a domain, write it explicitly, e.g. \(f(x) = \sqrt{x}\), \(x \geq 0\).

Exam Tip: A mapping diagram with one-to-one or many-to-one arrows represents a valid function. A one-to-many mapping is NOT a function.

2. Composite Functions

Composite function \(fg(x)\) means "apply \(g\) first, then apply \(f\) to the result":
$$fg(x) = f(g(x))$$ Note: order matters — \(fg(x) \neq gf(x)\) in general.
Example: Given \(f(x) = x^2 + 1\) and \(g(x) = 3x - 2\):

\(fg(x) = f(3x-2) = (3x-2)^2 + 1 = 9x^2 - 12x + 5\)
\(gf(x) = g(x^2+1) = 3(x^2+1) - 2 = 3x^2 + 1\)

These are clearly different — order matters!

3. Inverse Functions

The inverse function \(f^{-1}(x)\) reverses the effect of \(f\):
$$f^{-1}(f(x)) = x \quad \text{and} \quad f(f^{-1}(x)) = x$$ The graph of \(y = f^{-1}(x)\) is the reflection of \(y = f(x)\) in the line \(y = x\).
Domain of \(f^{-1}\) = Range of \(f\)   |   Range of \(f^{-1}\) = Domain of \(f\)

Method to Find an Inverse

  1. Write \(y = f(x)\)
  2. Swap \(x\) and \(y\): write \(x = f(y)\)
  3. Rearrange to make \(y\) the subject
  4. Write \(f^{-1}(x) = \ldots\) with the domain stated as the range of \(f\)
Example: Find \(f^{-1}(x)\) where \(f(x) = \dfrac{2x+1}{x-3}\), \(x \neq 3\).

Let \(y = \dfrac{2x+1}{x-3}\). Swap: \(x = \dfrac{2y+1}{y-3}\).
\(x(y-3) = 2y + 1 \Rightarrow xy - 3x = 2y + 1 \Rightarrow y(x-2) = 3x + 1\)
\(f^{-1}(x) = \dfrac{3x+1}{x-2}\), \(x \neq 2\)

Graph of \(f\) and its inverse, reflected in \(y=x\)

4. The Modulus Function

$$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$ The graph of \(y = |f(x)|\) reflects all parts of \(y = f(x)\) that lie below the \(x\)-axis up above it.

Solving Modulus Equations

To solve \(|f(x)| = g(x)\), consider two cases:

  • Case 1: \(f(x) = g(x)\) (where \(f(x) \geq 0\))
  • Case 2: \(-f(x) = g(x)\), i.e. \(f(x) = -g(x)\)

Always check solutions in the original equation — extraneous solutions can arise.

Modulus Inequalities

Best solved graphically: sketch both sides and identify where one is above/below the other.

  • \(|f(x)| < k \Rightarrow -k < f(x) < k\)
  • \(|f(x)| > k \Rightarrow f(x) < -k\) or \(f(x) > k\)

Graph of \(y = |x^2 - 4|\): the section between \(x = -2\) and \(x = 2\) is reflected upward.

Exam Tip: When solving modulus equations, always draw a sketch first. This helps identify how many solutions exist and avoids missing cases.

Order of Transformations

When combining transformations, order matters. Changes inside the function (affecting \(x\)) act in the opposite direction and in reverse order to what you might expect.

Example: Describe the transformation \(y = 2f(3x - 1)\).

Rewrite as \(y = 2f\!\left(3\left(x - \tfrac{1}{3}\right)\right)\).
Order: (1) horizontal stretch by factor \(\frac{1}{3}\), (2) translation by \(\frac{1}{3}\) right, (3) vertical stretch by factor 2.
Exam Tip: Inside brackets — horizontal effects, but reversed in direction. Outside brackets — vertical effects, as expected.

6. Piecewise Functions

A piecewise function is defined by different rules on different intervals:

$$f(x) = \begin{cases} x^2 & x < 0 \\ 2x + 1 & 0 \leq x \leq 3 \\ 7 & x > 3 \end{cases}$$

When sketching, plot each piece on its interval. Use open circles \(\circ\) for excluded endpoints and closed circles \(\bullet\) for included endpoints.
