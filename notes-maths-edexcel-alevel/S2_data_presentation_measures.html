S2: Data Presentation and Measures

1. Types of Data

TypeDescriptionExamples
Qualitative (Categorical)Non-numerical; describes categoriesColour, gender, blood type
Quantitative — DiscreteNumerical; only specific values (often integers)Number of siblings, shoe size, goals scored
Quantitative — ContinuousNumerical; can take any value in a rangeHeight, weight, time, temperature

2. Frequency Tables and Grouped Data

A frequency table lists each value (or class) and the number of times it occurs.

For grouped continuous data, use class intervals. The midpoint of each class is used to estimate the mean:

$$\bar{x} = \frac{\sum f x}{\sum f}$$

where \(x\) is the midpoint and \(f\) is the frequency.

Example: Heights (cm):

\(\bar{x} = 4155 / 25 = 166.2\) cm

3. Histograms

Histogram rule: In a histogram, the area of each bar is proportional to the frequency, NOT the height. The y-axis shows frequency density: $$\text{Frequency Density} = \frac{\text{Frequency}}{\text{Class Width}}$$
Exam Tip: When class widths are unequal, you MUST use frequency density. A common error is plotting frequency on the y-axis — this is only correct when all class widths are equal.

4. Cumulative Frequency and Box Plots

A cumulative frequency graph (ogive) plots cumulative frequency against the upper class boundary. From it, read off:

  • Median (Q2): at the \(\frac{n}{2}\)th value
  • Lower quartile (Q1): at the \(\frac{n}{4}\)th value
  • Upper quartile (Q3): at the \(\frac{3n}{4}\)th value
  • IQR: \(Q_3 - Q_1\)

5. Measures of Central Tendency

MeasureDefinitionWhen to use
Mean \(\bar{x}\)Sum of all values ÷ number of valuesSymmetric data; no extreme outliers
MedianMiddle value when orderedSkewed data; outliers present
ModeMost frequent valueCategorical data; finding most common

Mean from a frequency table:

$$\bar{x} = \frac{\sum fx}{\sum f}$$

Mean from grouped data: use midpoints for \(x\) in the formula above.

6. Measures of Spread

Range and Interquartile Range

$$\text{Range} = x_{\max} - x_{\min} \qquad \text{IQR} = Q_3 - Q_1$$

Variance and Standard Deviation

$$\sigma^2 = \frac{\sum fx^2}{\sum f} - \bar{x}^2$$
$$\sigma = \sqrt{\frac{\sum fx^2}{\sum f} - \bar{x}^2}$$
Exam Tip: On your calculator, use the statistics mode to find \(\bar{x}\) and \(\sigma_x\) directly. Make sure you use \(\sigma_x\) (population standard deviation), not \(S_x\) (sample standard deviation), unless the question specifies otherwise.
Example: Find the variance of: 2, 4, 4, 6, 8, 10, 10, 12 (n = 8)
\(\bar{x} = \frac{2+4+4+6+8+10+10+12}{8} = \frac{56}{8} = 7\)
\(\sum x^2 = 4+16+16+36+64+100+100+144 = 480\)
\(\sigma^2 = \frac{480}{8} - 7^2 = 60 - 49 = 11\)
\(\sigma = \sqrt{11} \approx 3.317\)

7. Coding

Coding simplifies calculations by transforming the data. If \(y = \frac{x - a}{b}\) then:

$$\bar{y} = \frac{\bar{x} - a}{b} \qquad \sigma_y = \frac{\sigma_x}{b}$$

Note: subtracting a constant \(a\) shifts the mean but does NOT affect the standard deviation. Dividing by \(b\) scales both the mean and standard deviation.

Example: Data is coded using \(y = \frac{x - 50}{10}\). Given \(\bar{y} = 3.2\) and \(\sigma_y = 1.4\):
\(\bar{x} = 10 \times 3.2 + 50 = 82\)
\(\sigma_x = 10 \times 1.4 = 14\)

8. Outliers

A data point is an outlier if it lies more than \(1.5 \times \text{IQR}\) above \(Q_3\) or below \(Q_1\): $$\text{Outlier if } x > Q_3 + 1.5 \times \text{IQR} \quad \text{or} \quad x < Q_1 - 1.5 \times \text{IQR}$$
Example: \(Q_1 = 20,\; Q_3 = 38,\; \text{IQR} = 18\)
Upper fence: \(38 + 1.5 \times 18 = 38 + 27 = 65\)
Lower fence: \(20 - 27 = -7\)
Any value above 65 or below −7 is an outlier.
