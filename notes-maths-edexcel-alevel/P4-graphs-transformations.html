1. Common Curve Shapes

FunctionKey features
\(y = x^2\)Upward parabola, vertex at origin, symmetric about y-axis
\(y = x^3\)S-shaped cubic, passes through origin, no symmetry axis (point symmetry about origin)
\(y = \frac{1}{x}\) (reciprocal)Two branches in quadrants 1 and 3; asymptotes: \(x=0\) and \(y=0\)
\(y = \frac{1}{x^2}\)Always positive; asymptotes \(x=0\), \(y=0\); symmetric about y-axis
\(y = \sqrt{x}\)Defined for \(x \geq 0\); passes through origin; increases, concave down
\(y = |x|\)V-shape, vertex at origin; equal arms with gradient \(\pm 1\)
Exam Tip: For any sketch, always label axes, key coordinates (intercepts, vertex, asymptotes), and the equation of any asymptote. Marks are awarded for these labels.

2. Graph Transformations

A transformation maps each point \((x,y)\) on \(y = f(x)\) to a new point according to specific rules. You must know all four types for the exam.

Translations

TransformationEquationEffect
Vertical translation up by \(a\)\(y = f(x) + a\)Every point moves up \(a\) units
Vertical translation down by \(a\)\(y = f(x) - a\)Every point moves down \(a\) units
Horizontal translation right by \(a\)\(y = f(x - a)\)Every point moves right \(a\) units (counter-intuitive — note the sign)
Horizontal translation left by \(a\)\(y = f(x + a)\)Every point moves left \(a\) units
Counter-intuitive: \(y = f(x-3)\) moves the graph right by 3. Think: to get the same y-value, you need a larger x-input.

Stretches

TransformationEquationEffect
Vertical stretch by scale factor \(a\)\(y = a\,f(x)\)y-coordinates multiplied by \(a\); x-intercepts unchanged
Horizontal stretch by scale factor \(\frac{1}{a}\)\(y = f(ax)\)x-coordinates divided by \(a\); y-intercept unchanged
A horizontal stretch by factor \(\frac{1}{a}\) means the graph is compressed if \(a > 1\) and stretched if \(0 < a < 1\). For example, \(y = f(2x)\) halves all x-coordinates.

Reflections

TransformationEquationEffect
Reflection in the x-axis\(y = -f(x)\)Negate all y-coordinates
Reflection in the y-axis\(y = f(-x)\)Negate all x-coordinates

4. Combined Transformations

When applying multiple transformations, the order matters. As a general rule:
  1. First apply transformations inside the brackets (horizontal changes)
  2. Then apply transformations outside the brackets (vertical changes)
Example: Describe the transformations that map \(y = f(x)\) to \(y = 3f(2x - 1) + 4\).

Rewrite as \(y = 3f\!\left(2\!\left(x - \tfrac{1}{2}\right)\right) + 4\)

Order of transformations:
  1. Horizontal stretch by factor \(\frac{1}{2}\) (i.e., \(y = f(2x)\))
  2. Horizontal translation right by \(\frac{1}{2}\) (i.e., \(y = f(2x-1)\))
  3. Vertical stretch by factor 3 (i.e., \(y = 3f(2x-1)\))
  4. Vertical translation up by 4 (i.e., \(y = 3f(2x-1)+4\))

5. Using Intersections to Solve Equations

The solutions of \(f(x) = g(x)\) are the x-coordinates of the intersections of \(y = f(x)\) and \(y = g(x)\).

Example: Use a sketch to find the number of solutions of \(x^2 = \dfrac{1}{x}\) for \(x > 0\).

\(y = x^2\) (upward parabola) and \(y = \frac{1}{x}\) (reciprocal curve, upper branch for \(x>0\)) cross exactly once in the first quadrant.
Setting equal: \(x^3 = 1 \Rightarrow x = 1\). One solution.
