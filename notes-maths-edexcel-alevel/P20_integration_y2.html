2. Integration by Substitution

Use substitution when the integrand contains a composite function. Let \(u = g(x)\), then \(du = g'(x)\,dx\).

Procedure:
  1. Choose a substitution \(u = g(x)\)
  2. Find \(\dfrac{du}{dx}\) and write \(dx = \dfrac{du}{g'(x)}\)
  3. Rewrite the integral entirely in terms of \(u\)
  4. Integrate with respect to \(u\)
  5. Substitute back to get the answer in terms of \(x\)
For definite integrals: change the limits to \(u\)-values.
Example 1: Find \(\displaystyle\int 2x(x^2+1)^4\,dx\).

Let \(u = x^2 + 1\), so \(du = 2x\,dx\).
\(\displaystyle\int u^4\,du = \dfrac{u^5}{5} + C = \dfrac{(x^2+1)^5}{5} + C\)
Example 2 (definite): Find \(\displaystyle\int_0^1 x\sqrt{x^2+1}\,dx\).

Let \(u = x^2+1\), \(du = 2x\,dx\).
When \(x=0\): \(u=1\). When \(x=1\): \(u=2\).
\(\displaystyle\int_1^2 \dfrac{\sqrt{u}}{2}\,du = \dfrac{1}{2}\cdot\dfrac{2}{3}[u^{3/2}]_1^2 = \dfrac{1}{3}(2\sqrt{2}-1)\)

3. Integration by Parts

$$\int u\,\frac{dv}{dx}\,dx = uv - \int v\,\frac{du}{dx}\,dx$$ Or in shorthand: \(\displaystyle\int u\,dv = uv - \int v\,du\)

LIATE: Choosing \(u\)

Choose \(u\) as whichever function comes first in LIATE; let \(dv\) be everything else.

LetterTypeExamples
LLogarithmic\(\ln x,\;\log x\)
IInverse trig\(\sin^{-1}x,\;\tan^{-1}x\)
AAlgebraic\(x^n,\;x^2+1\)
TTrigonometric\(\sin x,\;\cos x\)
EExponential\(e^x,\;2^x\)
Example 1: Find \(\displaystyle\int x e^x\,dx\).

\(u = x \Rightarrow \dfrac{du}{dx} = 1\); \(\dfrac{dv}{dx} = e^x \Rightarrow v = e^x\)
\(\displaystyle\int xe^x\,dx = xe^x - \int e^x\,dx = xe^x - e^x + C = e^x(x-1) + C\)
Example 2: Find \(\displaystyle\int x^2\sin x\,dx\).

\(u = x^2,\; dv = \sin x\,dx \Rightarrow v = -\cos x\)
\(= -x^2\cos x + 2\displaystyle\int x\cos x\,dx\)
Apply IBP again: \(u=x,\;dv=\cos x\,dx\Rightarrow v=\sin x\)
\(= -x^2\cos x + 2(x\sin x - \int\sin x\,dx) = -x^2\cos x + 2x\sin x + 2\cos x + C\)
Example 3: Find \(\displaystyle\int \ln x\,dx\).

\(u = \ln x,\; dv = 1\,dx \Rightarrow v = x\)
\(= x\ln x - \displaystyle\int x \cdot \dfrac{1}{x}\,dx = x\ln x - x + C\)
Cyclic integration: For \(\displaystyle\int e^x\sin x\,dx\), apply IBP twice. The original integral reappears — collect it on one side and solve algebraically.

4. Integration by Partial Fractions

Used when integrating rational functions. Decompose into partial fractions first, then integrate term by term using \(\displaystyle\int \dfrac{1}{ax+b}\,dx = \dfrac{1}{a}\ln|ax+b| + C\).

Example: Find \(\displaystyle\int \dfrac{5x+3}{(x+1)(2x-1)}\,dx\).

Partial fractions: \(\dfrac{5x+3}{(x+1)(2x-1)} = \dfrac{A}{x+1} + \dfrac{B}{2x-1}\)
\(5x+3 = A(2x-1) + B(x+1)\)
\(x = -1\): \(-2 = -3A \Rightarrow A = \tfrac{2}{3}\). \(\quad x = \tfrac{1}{2}\): \(\tfrac{11}{2} = \tfrac{3}{2}B \Rightarrow B = \tfrac{11}{3}\)
\(\displaystyle\int \left(\dfrac{2/3}{x+1} + \dfrac{11/3}{2x-1}\right)dx = \dfrac{2}{3}\ln|x+1| + \dfrac{11}{6}\ln|2x-1| + C\)

5. Differential Equations

Separable Differential Equations

A first-order ODE of the form \(\dfrac{dy}{dx} = f(x)g(y)\) is solved by separating variables: $$\int \frac{1}{g(y)}\,dy = \int f(x)\,dx$$ Then integrate both sides and include a constant of integration \(C\).
Example (General solution): Solve \(\dfrac{dy}{dx} = \dfrac{x}{y}\).

Separate: \(y\,dy = x\,dx\)
Integrate: \(\dfrac{y^2}{2} = \dfrac{x^2}{2} + C\)
General solution: \(y^2 - x^2 = k\) (where \(k = 2C\))
Example (Particular solution with boundary condition): Solve \(\dfrac{dy}{dx} = 2xy\), given \(y = 3\) when \(x = 0\).

Separate: \(\dfrac{1}{y}\,dy = 2x\,dx\)
Integrate: \(\ln|y| = x^2 + C\)
So \(y = Ae^{x^2}\) where \(A = e^C\).
Apply BC: \(3 = Ae^0 = A\).
Particular solution: \(y = 3e^{x^2}\)

General vs Particular Solutions

  • General solution: includes arbitrary constant \(C\); represents a family of curves
  • Particular solution: found by applying an initial/boundary condition to determine \(C\)

6. Modelling with Differential Equations

Exponential Growth and Decay

$$\frac{dN}{dt} = kN \quad \Rightarrow \quad N = N_0 e^{kt}$$ If \(k > 0\): growth. If \(k < 0\): decay.

Newton's Law of Cooling

$$\frac{d\theta}{dt} = -k(\theta - \theta_s)$$ where \(\theta_s\) is the surrounding temperature, \(k > 0\).
Solution: \(\theta - \theta_s = A e^{-kt}\), so \(\theta = \theta_s + Ae^{-kt}\).
Example: A cup of tea at 90°C cools in a room at 20°C. After 5 minutes, its temperature is 70°C. Find the temperature after 15 minutes.

\(\theta = 20 + Ae^{-kt}\). At \(t=0\): \(90 = 20 + A \Rightarrow A = 70\).
At \(t=5\): \(70 = 20 + 70e^{-5k} \Rightarrow e^{-5k} = \dfrac{50}{70} = \dfrac{5}{7}\)
\(-5k = \ln\dfrac{5}{7} \Rightarrow k = \dfrac{1}{5}\ln\dfrac{7}{5}\)
At \(t=15\): \(\theta = 20 + 70\!\left(\dfrac{5}{7}\right)^3 = 20 + 70 \cdot \dfrac{125}{343} \approx 20 + 25.5 \approx 45.5\)°C

Logistic Growth (context)

Population growth may be modelled by \(\dfrac{dP}{dt} = kP\!\left(1 - \dfrac{P}{M}\right)\) where \(M\) is the carrying capacity. This is separable.
