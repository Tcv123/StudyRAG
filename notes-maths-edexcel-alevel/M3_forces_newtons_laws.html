M3: Forces and Newton's Laws

1. Forces as Vectors

Forces are vector quantities — they have both magnitude and direction. The resultant force is the single force equivalent to all forces acting, found by vector addition.

Resultant Force: \(\mathbf{F}_{net} = \mathbf{F}_1 + \mathbf{F}_2 + \cdots + \mathbf{F}_n\)
In component form: \(F_x = \sum F_i \cos\theta_i\), \quad \(F_y = \sum F_i \sin\theta_i\)

The magnitude of the resultant: \(|\mathbf{F}_{net}| = \sqrt{F_x^2 + F_y^2}\), and its direction: \(\theta = \arctan\!\left(\tfrac{F_y}{F_x}\right)\).

2. Newton's Laws of Motion

  • First Law: A particle remains at rest or moves with constant velocity unless acted on by a resultant force.
  • Second Law: \(\mathbf{F}_{net} = m\mathbf{a}\) — the net force equals mass times acceleration (vector equation).
  • Third Law: Every action has an equal and opposite reaction.
$$\mathbf{F}_{net} = m\mathbf{a}$$

For problems in a single direction, resolve forces along that direction and apply \(F = ma\) as a scalar equation.

4. Resolving Forces

Any force \(F\) at angle \(\theta\) to the horizontal can be split into:

$$F_x = F\cos\theta \qquad F_y = F\sin\theta$$

To solve problems: choose axes (usually horizontal/vertical or along/perpendicular to slope), resolve all forces, apply \(F = ma\) in each direction.

Exam Tip: Always draw a clear force diagram before writing equations. Label every force with its magnitude and direction. Never omit the weight \(mg\).

5. Equilibrium

A particle is in equilibrium when the resultant force is zero:
$$\sum F_x = 0 \qquad \text{and} \qquad \sum F_y = 0$$ Equivalently, \(\mathbf{F}_{net} = \mathbf{0}\), so \(m\mathbf{a} = \mathbf{0}\) and the particle is at rest or moving at constant velocity.

6. Connected Particles

When two particles are connected by a light inextensible string over a smooth pulley (Atwood machine):

  • The string has the same tension \(T\) throughout (light string, smooth pulley).
  • Both particles have the same magnitude of acceleration \(a\) (inextensible string).
  • Write a separate \(F = ma\) equation for each particle, then solve simultaneously.

Atwood Machine Setup

Atwood machine: \(m_1 > m_2\), so \(m_1\) descends with acceleration \(a\)

Atwood Machine Equations:
For \(m_1\) (descending): \(\quad m_1 g - T = m_1 a\)
For \(m_2\) (ascending): \(\quad T - m_2 g = m_2 a\)
Adding: \(\quad (m_1 - m_2)g = (m_1 + m_2)a\)
$$a = \frac{(m_1 - m_2)g}{m_1 + m_2}, \qquad T = \frac{2m_1 m_2 g}{m_1 + m_2}$$

Particle on a Table Connected to a Hanging Particle

Particle A (mass \(m_1\)) on a smooth horizontal table, connected by a string over a pulley at the edge to particle B (mass \(m_2\)) hanging freely.

  • For A (horizontal, no friction if smooth): \(T = m_1 a\)
  • For B (vertical): \(m_2 g - T = m_2 a\)
  • Solve: \(a = \dfrac{m_2 g}{m_1 + m_2}\), \quad \(T = \dfrac{m_1 m_2 g}{m_1 + m_2}\)

7. Particle in a Lift

A particle of mass \(m\) stands on the floor of a lift. The normal reaction \(N\) (apparent weight) depends on the lift's acceleration \(a\):

$$N = m(g + a) \quad \text{(accelerating upward)}$$ $$N = m(g - a) \quad \text{(accelerating downward)}$$
  • If \(a = 0\) (constant velocity): \(N = mg\) (normal weight).
  • If \(a = g\) downward (free fall): \(N = 0\) (weightlessness).
Exam Tip: In a lift problem, always define positive direction clearly (e.g., upward positive). Apply \(F = ma\) in that direction. If lift accelerates upward, net upward force = \(N - mg = ma\), giving \(N = m(g+a)\).

8. Worked Example

Example: Two particles, A (\(3\ \text{kg}\)) and B (\(5\ \text{kg}\)), are connected by a light inextensible string over a smooth pulley. Find the acceleration and tension.

Take downward as positive for each particle in its direction of motion (B descends).
For B: \(5(9.8) - T = 5a \Rightarrow 49 - T = 5a\) ...(1)
For A: \(T - 3(9.8) = 3a \Rightarrow T - 29.4 = 3a\) ...(2)
Add (1)+(2): \(19.6 = 8a \Rightarrow a = 2.45\ \text{m/s}^2\)
Sub into (2): \(T = 29.4 + 3(2.45) = 36.75\ \text{N}\)

Common Errors to Avoid:
  • Using \(g = 10\) instead of \(g = 9.8\) — Edexcel uses 9.8 unless told otherwise.
  • Forgetting to write separate \(F = ma\) equations for each particle in a connected particle problem.
  • Wrong sign for tension direction — tension always pulls the particle toward the string/pulley.
  • Forgetting that the string tension is the same throughout (light string, smooth pulley).
