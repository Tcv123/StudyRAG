P9: Differentiation (Year 1)

1. Differentiation from First Principles

Definition of the Derivative: $$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$ The derivative measures the instantaneous rate of change of \(f(x)\) — the gradient of the tangent to the curve at a given point.
First Principles — Differentiate \(f(x) = x^2\):

\(f'(x) = \lim_{h \to 0} \dfrac{(x+h)^2 - x^2}{h}\)

\(= \lim_{h \to 0} \dfrac{x^2 + 2xh + h^2 - x^2}{h}\)

\(= \lim_{h \to 0} \dfrac{2xh + h^2}{h}\)

\(= \lim_{h \to 0} (2x + h) = 2x\)
Exam Tip: First principles questions appear regularly. Always expand \(f(x+h)\), subtract \(f(x)\), divide by \(h\), cancel the \(h\) factor, then take the limit. Edexcel may ask this for \(x^2\), \(x^3\), \(\frac{1}{x}\), or \(\sqrt{x}\).

2. Standard Differentiation Rules

Power Rule (the most important rule)

$$\frac{d}{dx}(x^n) = nx^{n-1}$$ Works for all rational \(n\) — positive, negative, fractional.
FunctionDerivativeNotes
\(y = x^n\)\(\dfrac{dy}{dx} = nx^{n-1}\)All values of \(n\)
\(y = c\) (constant)\(\dfrac{dy}{dx} = 0\)Gradient of horizontal line is 0
\(y = cx^n\)\(\dfrac{dy}{dx} = cnx^{n-1}\)Multiply by coefficient
\(y = e^x\)\(\dfrac{dy}{dx} = e^x\)Unique self-derivative property
\(y = \ln x\)\(\dfrac{dy}{dx} = \dfrac{1}{x}\)Defined for \(x > 0\)
\(y = \sin x\)\(\dfrac{dy}{dx} = \cos x\)\(x\) in radians
\(y = \cos x\)\(\dfrac{dy}{dx} = -\sin x\)\(x\) in radians
Example — Differentiate each:

(a) \(y = 3x^4 - 2x^3 + 5x - 7\)
\(\dfrac{dy}{dx} = 12x^3 - 6x^2 + 5\)

(b) \(y = \dfrac{4}{x^2} = 4x^{-2}\)
\(\dfrac{dy}{dx} = -8x^{-3} = -\dfrac{8}{x^3}\)

(c) \(y = 5\sqrt{x} = 5x^{1/2}\)
\(\dfrac{dy}{dx} = \dfrac{5}{2}x^{-1/2} = \dfrac{5}{2\sqrt{x}}\)

(d) \(y = \dfrac{3x^2 + 2x}{x} = 3x + 2\) (simplify first!)
\(\dfrac{dy}{dx} = 3\)
Exam Tip: Always simplify before differentiating — rewrite fractions and roots as powers of \(x\). Never differentiate a fraction like \(\frac{f(x)}{g(x)}\) without the quotient rule (Year 2), unless it simplifies.

3. Tangents and Normals

At a point \(P(a, f(a))\) on a curve:

  • Gradient of tangent = \(f'(a)\)
  • Gradient of normal = \(-\dfrac{1}{f'(a)}\) (negative reciprocal, since tangent ⊥ normal)

Equation of tangent at \((a, f(a))\): \(y - f(a) = f'(a)(x - a)\)

Equation of normal at \((a, f(a))\): \(y - f(a) = -\dfrac{1}{f'(a)}(x - a)\)

Example — Find the equation of the tangent and normal to \(y = x^3 - 2x\) at \(x = 2\):

\(y\)-value: \(y = 8 - 4 = 4\), so point is \((2, 4)\)
\(\dfrac{dy}{dx} = 3x^2 - 2\), at \(x=2\): gradient \(= 12 - 2 = 10\)

Tangent: \(y - 4 = 10(x - 2) \Rightarrow y = 10x - 16\)
Normal: \(y - 4 = -\frac{1}{10}(x - 2) \Rightarrow 10y + x = 42\)

4. Second Derivative

Second Derivative \(f''(x)\) or \(\dfrac{d^2y}{dx^2}\) is the derivative of the derivative.

It measures the rate of change of the gradient:
  • \(f''(x) > 0\): gradient is increasing — curve is concave up
  • \(f''(x) < 0\): gradient is decreasing — curve is concave down

5. Stationary Points

A stationary point occurs where \(f'(x) = 0\) (the tangent is horizontal).

To classify the stationary point at \(x = a\):
  • If \(f''(a) > 0\): local minimum
  • If \(f''(a) < 0\): local maximum
  • If \(f''(a) = 0\): test is inconclusive — use sign change of \(f'(x)\)
Example — Find and classify stationary points of \(y = 2x^3 - 9x^2 + 12x - 4\):

\(\dfrac{dy}{dx} = 6x^2 - 18x + 12 = 6(x^2 - 3x + 2) = 6(x-1)(x-2)\)

Stationary points where \(\dfrac{dy}{dx} = 0\): \(x = 1\) and \(x = 2\)

\(\dfrac{d^2y}{dx^2} = 12x - 18\)

At \(x = 1\): \(f''(1) = 12 - 18 = -6 < 0\) → local maximum; \(y = 2 - 9 + 12 - 4 = 1\)
At \(x = 2\): \(f''(2) = 24 - 18 = 6 > 0\) → local minimum; \(y = 16 - 36 + 24 - 4 = 0\)

Local max at \((1, 1)\), local min at \((2, 0)\).

6. Increasing and Decreasing Functions

  • \(f'(x) > 0\) for all \(x\) in an interval \(\Rightarrow f\) is increasing on that interval
  • \(f'(x) < 0\) for all \(x\) in an interval \(\Rightarrow f\) is decreasing on that interval
Example — Find where \(f(x) = x^3 - 3x^2 + 1\) is increasing:

\(f'(x) = 3x^2 - 6x = 3x(x-2)\)
\(f'(x) > 0\) when \(x < 0\) or \(x > 2\)
So \(f\) is increasing for \(x < 0\) and \(x > 2\).

7. Optimisation

Real-world problems asking to maximise area, minimise cost, etc. Follow this method:

  1. Define a variable and express the quantity to optimise as a function of that variable
  2. Use a constraint to reduce to one variable if necessary
  3. Differentiate and set equal to 0
  4. Verify it is a maximum/minimum using the second derivative
  5. Check endpoints of the domain if required
Example — A farmer has 100 m of fencing to enclose a rectangular pen against a wall (no fence needed on wall side). Find the dimensions for maximum area:

Let width = \(x\) m (two sides), length = \(100 - 2x\) m
Area: \(A = x(100 - 2x) = 100x - 2x^2\)
\(\dfrac{dA}{dx} = 100 - 4x = 0 \Rightarrow x = 25\)
\(\dfrac{d^2A}{dx^2} = -4 < 0\) — confirms maximum
Dimensions: 25 m × 50 m, Area = 1250 m²
Key Exam Points for P9:
  • First principles: expand \(f(x+h)\), simplify, cancel \(h\), take limit — show all steps
  • Rewrite before differentiating: \(\frac{1}{x^2} = x^{-2}\), \(\sqrt[3]{x} = x^{1/3}\), \(\frac{3}{x} = 3x^{-1}\)
  • For tangent/normal: find \(y\)-value at the point, find gradient, substitute into \(y - y_1 = m(x - x_1)\)
  • Second derivative test: if \(f''=0\) at a stationary point, check sign change of \(f'\) instead
  • In optimisation, always verify whether you have a max or min and check the context makes sense
