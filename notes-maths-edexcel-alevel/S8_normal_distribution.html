S8: The Normal Distribution

1. The Normal Distribution

If \(X\) follows a normal distribution with mean \(\mu\) and variance \(\sigma^2\), we write \(X \sim N(\mu, \sigma^2)\).
  • The distribution is bell-shaped and symmetric about \(\mu\)
  • Mean = Median = Mode = \(\mu\)
  • The curve extends to \(\pm\infty\) but the total area = 1
  • \(\sigma\) controls the spread — larger \(\sigma\) gives a wider, flatter curve

2. The Standard Normal Distribution

The standard normal distribution has mean 0 and variance 1: \(Z \sim N(0, 1)\). Any normal variable can be standardised: $$Z = \frac{X - \mu}{\sigma}$$ This converts X to a Z-score, measuring how many standard deviations above/below the mean.

The cumulative distribution function is \(\Phi(z) = P(Z \leq z)\), tabulated in the Edexcel formula booklet or found via calculator.

3. Calculating Probabilities

ProbabilityHow to find it
\(P(X < a)\)Standardise: \(P\!\left(Z < \dfrac{a-\mu}{\sigma}\right) = \Phi(z)\)
\(P(X > a)\)\(= 1 - P(X < a)\)
\(P(a < X < b)\)\(= P(X < b) - P(X < a)\)
\(P(X < -z)\)\(= 1 - \Phi(z)\) (by symmetry)
Example: \(X \sim N(50, 16)\). Find \(P(X < 54)\) and \(P(46 < X < 58)\).

\(\sigma = \sqrt{16} = 4\)

Part 1: \(P(X < 54) = P\!\left(Z < \dfrac{54-50}{4}\right) = P(Z < 1) = \Phi(1) = 0.8413\)

Part 2: \(P(46 < X < 58) = P\!\left(\dfrac{46-50}{4} < Z < \dfrac{58-50}{4}\right) = P(-1 < Z < 2)\)
\(= \Phi(2) - \Phi(-1) = 0.9772 - (1 - 0.8413) = 0.9772 - 0.1587 = 0.8185\)

4. Inverse Normal — Finding Values from Probabilities

Given \(P(X < a) = p\), find \(a\):

  1. Find \(z\) such that \(\Phi(z) = p\) (using tables or calculator).
  2. Rearrange the standardisation: \(a = \mu + z\sigma\).
Example: \(X \sim N(50, 16)\). Find \(a\) such that \(P(X < a) = 0.9\).

From tables: \(\Phi(1.282) = 0.9\), so \(z = 1.282\).
\(a = 50 + 1.282 \times 4 = 50 + 5.128 = 55.1\) (3 s.f.)

5. Finding Unknown \(\mu\) or \(\sigma\)

When given probability statements, standardise to form equations and solve simultaneously if needed.

Example — Finding both \(\mu\) and \(\sigma\):
\(X \sim N(\mu, \sigma^2)\), \(P(X < 20) = 0.1\) and \(P(X < 30) = 0.75\). Find \(\mu\) and \(\sigma\).

From \(P(X < 20) = 0.1\): \(\dfrac{20 - \mu}{\sigma} = -1.282\) → \(20 - \mu = -1.282\sigma\)   (1)

From \(P(X < 30) = 0.75\): \(\dfrac{30 - \mu}{\sigma} = 0.6745\) → \(30 - \mu = 0.6745\sigma\)   (2)

Subtract (1) from (2): \(10 = (0.6745 + 1.282)\sigma = 1.9565\sigma\)
\(\sigma = \dfrac{10}{1.9565} \approx 5.11\)
From (2): \(\mu = 30 - 0.6745 \times 5.11 \approx 30 - 3.45 = 26.6\)

6. Normal Approximation to the Binomial

If \(X \sim B(n, p)\) where \(n\) is large and \(p\) is not too close to 0 or 1, then: $$X \approx N(\mu, \sigma^2) \quad \text{where } \mu = np \text{ and } \sigma^2 = np(1-p)$$ Rule of thumb: both \(np > 5\) and \(n(1-p) > 5\) should hold.
Exam tip: A normal approximation is most accurate when \(p\) is close to 0.5. Always state the conditions for the approximation being valid.

Continuity Correction

Because the binomial is discrete and the normal is continuous, a continuity correction of 0.5 is applied:

Binomial (discrete)Normal (continuous) with correction
\(P(X \leq k)\)\(P(Y < k + 0.5)\)
\(P(X < k)\)\(P(Y < k - 0.5)\)
\(P(X \geq k)\)\(P(Y > k - 0.5)\)
\(P(X > k)\)\(P(Y > k + 0.5)\)
\(P(X = k)\)\(P(k - 0.5 < Y < k + 0.5)\)
Example: \(X \sim B(100, 0.4)\). Use a normal approximation to find \(P(X \leq 35)\).

Check: \(np = 40 > 5\), \(n(1-p) = 60 > 5\). Normal approximation valid.
\(\mu = 100 \times 0.4 = 40\), \(\sigma^2 = 100 \times 0.4 \times 0.6 = 24\), \(\sigma = \sqrt{24} \approx 4.899\)
\(Y \sim N(40, 24)\)
Continuity correction: \(P(X \leq 35) \approx P(Y < 35.5)\)
\(z = \dfrac{35.5 - 40}{4.899} = \dfrac{-4.5}{4.899} \approx -0.918\)
\(P(Y < 35.5) = \Phi(-0.918) = 1 - \Phi(0.918) = 1 - 0.8206 = 0.1794\)

7. Hypothesis Testing with the Normal Distribution

To test a claim about the population mean \(\mu\) when \(X \sim N(\mu, \sigma^2)\) and \(\sigma\) is known:

  1. State \(H_0: \mu = \mu_0\) and \(H_1\) (one- or two-tailed).
  2. Identify the significance level \(\alpha\).
  3. Calculate the test statistic: \(z = \dfrac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}\)
  4. Compare with the critical z-value (e.g., \(\pm 1.96\) for two-tailed 5%) or find p-value.
  5. State conclusion in context.
Exam tip: Standard critical z-values to memorise: one-tailed 5%: 1.645; one-tailed 1%: 2.326; two-tailed 5%: 1.960; two-tailed 1%: 2.576.
