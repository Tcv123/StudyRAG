S1.4: Chi-Squared Tests

1. The Test Statistic

The \(\chi^2\) goodness-of-fit test compares observed counts \(O_i\) with expected counts \(E_i\) under some hypothesis. Test statistic: \[X^2 = \sum_i \dfrac{(O_i - E_i)^2}{E_i}.\] Under \(H_0\) (and large \(E_i\)), \(X^2\) is approximately \(\chi^2_{\nu}\) with \(\nu\) degrees of freedom.

2. Degrees of Freedom

  • Start with the number of categories \(k\).
  • Subtract 1 for the constraint \(\sum O_i = n\).
  • Subtract 1 more for each parameter of the hypothesised distribution that is estimated from the data.

So for goodness-of-fit to a Poisson with \(\lambda\) estimated from the data, \(\nu = k - 1 - 1 = k - 2\).

3. Goodness-of-Fit Test Procedure

  1. State \(H_0\) (distribution fits) and \(H_1\) (does not fit).
  2. Compute expected frequencies under \(H_0\): \(E_i = n\cdot P(X = x_i)\).
  3. Pool any \(E_i < 5\) with adjacent cells.
  4. Compute \(X^2 = \sum (O - E)^2/E\).
  5. Find \(\nu\) and the critical value \(\chi^2_{\nu, \alpha}\).
  6. Compare: if \(X^2 > \chi^2_{\nu, \alpha}\), reject \(H_0\).
  7. State conclusion in context.
The \(\chi^2\) approximation is only valid if each \(E_i \ge 5\) (a common rule of thumb). Pool cells if needed — but then count the pooled cell as one category, reducing \(k\) and hence \(\nu\).
A die is rolled 120 times with results: 1→15, 2→22, 3→24, 4→18, 5→21, 6→20. Test at 5% whether the die is fair.
\(H_0\): die is fair (each \(E_i = 20\)). \(H_1\): not fair.
\(X^2 = \tfrac{25 + 4 + 16 + 4 + 1 + 0}{20} = \tfrac{50}{20} = 2.5\).
\(\nu = 6 - 1 = 5\). Critical value \(\chi^2_{5, 0.05} = 11.07\).
Since \(2.5 < 11.07\), do not reject \(H_0\). No evidence of bias.

4. Fitting a Poisson or Binomial to Data

Poisson Fit

Estimate \(\hat{\lambda} = \bar{x}\) from the data. Compute \(E_i = n\cdot P(X = i)\) using \(\operatorname{Po}(\hat{\lambda})\). Degrees of freedom: \(\nu = k - 2\) (one less because \(\lambda\) was estimated).

Binomial Fit

If \(n\) (number of trials) is known and \(p\) is estimated, \(\nu = k - 2\). If \(p\) is also given by theory, \(\nu = k - 1\).

5. Contingency Tables

An \(r \times c\) contingency table tests whether two categorical variables are independent.

Expected Frequencies

\(E_{ij} = \dfrac{(\text{row}_i \text{ total}) \times (\text{col}_j \text{ total})}{n}\)

Degrees of Freedom

\(\nu = (r - 1)(c - 1)\)

\(\nu = (2-1)(3-1) = 2\). \(\chi^2_{2, 0.05} = 5.99\).
Since \(9.52 > 5.99\), reject \(H_0\): evidence of association between gender and subject.

6. Yates' Continuity Correction

For a \(2 \times 2\) contingency table (with \(\nu = 1\)), the discrete nature of counts can be corrected by: \[X^2_{\text{Yates}} = \sum \dfrac{(|O - E| - 0.5)^2}{E}.\] This gives a slightly smaller (less significant) statistic.

Use Yates' correction whenever \(\nu = 1\) and cell counts are small.

7. Writing up a Hypothesis Test

A complete answer includes:

  1. Hypotheses in words and symbols.
  2. Expected frequencies (show the working for at least one).
  3. The test statistic \(X^2\), to 2-3 s.f.
  4. Degrees of freedom and the critical value.
  5. Decision to reject or not reject.
  6. A contextual conclusion.

8. Exam Style Strategy

  • Compute \(X^2\) in a table — columns for \(O\), \(E\), \(O - E\), \((O - E)^2 / E\).
  • If any \(E_i < 5\), pool before computing and reduce \(\nu\) accordingly.
  • State explicitly any parameters estimated from the data, as they reduce \(\nu\).
  • \(\chi^2\) is one-tailed (always upper-tail): we reject when the statistic is too large.
