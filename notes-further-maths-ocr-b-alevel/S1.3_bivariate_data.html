S1.3: Bivariate Data and Correlation

1. Scatter Diagram

Visualise bivariate data \((x_i,y_i)\) to spot positive/negative/no correlation and check for linearity before computing \(r\).

2. Pearson Product-Moment Correlation Coefficient (PMCC)

\[r=\dfrac{S_{xy}}{\sqrt{S_{xx}S_{yy}}}\] where \(S_{xy}=\sum(x-\bar{x})(y-\bar{y})=\sum xy-n\bar{x}\bar{y}\); similarly \(S_{xx},S_{yy}\).

Range: \(-1\le r\le1\). \(r=\pm1\) means perfect linear. \(r=0\) means no linear correlation.

3. Spearman Rank Correlation

Rank each variable (1 to \(n\)). With \(d_i=\text{rank}(x_i)-\text{rank}(y_i)\): \[\rho_s=1-\dfrac{6\sum d_i^2}{n^3-n}\] (no ties). Detects monotonic (possibly non-linear) relationships; robust to outliers.

4. Hypothesis Tests for Correlation

  • PMCC test: \(H_0:\rho=0\); use tables of critical values of \(r\) for given \(n\) and \(\alpha\).
  • Spearman test: \(H_0\): no (monotonic) correlation; use Spearman tables.

5. Regression Line

The least-squares regression line of \(y\) on \(x\) is \(y=a+bx\), where \[b=\dfrac{S_{xy}}{S_{xx}},\quad a=\bar{y}-b\bar{x}.\] Minimises \(\sum(y_i-\hat{y}_i)^2\).

Line passes through \((\bar{x},\bar{y})\). Used for prediction, but valid only within the data range (interpolation safe; extrapolation risky).

6. Coefficient of Determination \(r^2\)

\(r^2\) = proportion of variance in \(y\) explained by linear regression on \(x\).

7. Correlation vs. Causation

Strong correlation does not imply causation. Possible explanations: common cause, reverse causation, chance.

8. Exam Strategy

  • Always draw/inspect scatter plot first; decide if linear analysis appropriate.
  • PMCC for linear; Spearman for monotonic / robust analysis / ordinal data.
  • Report test results in context: "There is evidence of correlation between \(x\) and \(y\) at the 5% level, but this does not establish causation."
  • For regression, state the equation and interpret slope and intercept in context.
