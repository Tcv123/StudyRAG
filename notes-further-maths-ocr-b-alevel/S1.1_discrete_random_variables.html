S1.1: Discrete Random Variables

1. Probability Distributions

A discrete random variable \(X\) takes a countable set of values \(x_1, x_2, \ldots\). Its probability mass function (pmf) is \(P(X = x_i) = p_i\), with \[\sum_i p_i = 1, \qquad 0 \le p_i \le 1.\]
\(X\) has pmf given by \(P(X = x) = kx\) for \(x = 1, 2, 3, 4\). Find \(k\).
\(k(1+2+3+4) = 1 \Rightarrow 10k = 1 \Rightarrow k = 0.1\).

2. Expectation and Variance

\(E(X) = \mu = \sum_i x_i p_i\)
\(\operatorname{Var}(X) = \sigma^2 = E(X^2) - \mu^2 = \sum_i x_i^2 p_i - \mu^2\)
\(E[g(X)] = \sum_i g(x_i) p_i\)

Linear Transformations

\(E(aX + b) = aE(X) + b\)
\(\operatorname{Var}(aX + b) = a^2 \operatorname{Var}(X)\)

Note: a constant shift doesn't affect variance; a scaling factor is squared.

3. Sums of Independent Discrete RVs

For independent \(X\) and \(Y\):

\(E(X + Y) = E(X) + E(Y)\)
\(\operatorname{Var}(X + Y) = \operatorname{Var}(X) + \operatorname{Var}(Y)\)
\(\operatorname{Var}(X - Y) = \operatorname{Var}(X) + \operatorname{Var}(Y)\) — add variances even when subtracting (for independent variables). A very common slip.

General Linear Combination

\(\operatorname{Var}(aX + bY) = a^2\operatorname{Var}(X) + b^2\operatorname{Var}(Y)\)  (independent)

Sum of \(n\) IID Copies vs. Scaling

If \(X_1, \ldots, X_n\) are IID copies of \(X\), and \(S = X_1 + \cdots + X_n\):

  • \(E(S) = nE(X)\), \(\operatorname{Var}(S) = n\operatorname{Var}(X)\).

Contrast: \(Y = nX\) (a single \(X\) scaled by \(n\)):

  • \(E(Y) = nE(X)\), \(\operatorname{Var}(Y) = n^2\operatorname{Var}(X)\).
OCR Exam Tip: Distinguish between "the total from \(n\) rolls" (sum) and "10 × (single roll)" (scaling). Totals from independent trials have \(n\) variance; scaling has \(n^2\).

4. The Discrete Uniform Distribution

\(X \sim U\{1, 2, \ldots, n\}\): each outcome equally likely with probability \(1/n\).

  • \(E(X) = \dfrac{n+1}{2}\)
  • \(\operatorname{Var}(X) = \dfrac{n^2 - 1}{12}\)

5. Joint Discrete Distributions

Two discrete random variables \(X, Y\) have a joint pmf \(P(X = x, Y = y)\), often displayed as a table.

Marginals

\(P(X = x) = \sum_y P(X = x, Y = y), \quad P(Y = y) = \sum_x P(X = x, Y = y)\)

Independence

X \perp Y \iff P(X = x, Y = y) = P(X = x) \, P(Y = y) \text{ for all } x, y.

Conditional Distribution

\(P(Y = y \mid X = x) = \dfrac{P(X = x, Y = y)}{P(X = x)}\)

6. Covariance (Support)

\(\operatorname{Cov}(X, Y) = E(XY) - E(X)E(Y)\)

For general (not necessarily independent) \(X, Y\):

\(\operatorname{Var}(X + Y) = \operatorname{Var}(X) + \operatorname{Var}(Y) + 2\operatorname{Cov}(X, Y)\)

\(X \perp Y \Rightarrow \operatorname{Cov}(X, Y) = 0\) (the converse is not generally true).

7. Exam Style Strategy

  • Use the formula \(\operatorname{Var}(X) = E(X^2) - \mu^2\) — computing \(E[(X - \mu)^2]\) directly is error-prone.
  • For distributions with a parameter \(k\), use \(\sum p_i = 1\) to find \(k\) first.
  • Drawing a joint table is very helpful; always check marginals sum correctly.
  • When working with \(aX + bY\), track signs of \(a, b\) carefully; in variance they are squared and so always non-negative.
