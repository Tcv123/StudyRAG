M1.1: Dimensional Analysis

1. Base Dimensions

In mechanics we use three base dimensions:
  • Mass — \(M\)
  • Length — \(L\)
  • Time — \(T\)
Every mechanical quantity has dimensions expressible as \(M^a L^b T^c\).

We write \([X]\) for the dimensions of a quantity \(X\).

3. Principle of Dimensional Homogeneity

In a valid physical equation, every term must have the same dimensions. Arguments of functions like \(\sin, \cos, \ln, e^x\) must be dimensionless.

This gives a quick sanity check on any derived formula.

Check that \(s = ut + \tfrac{1}{2}at^2\) is dimensionally consistent.
\([s] = L\). \([ut] = (LT^{-1})(T) = L\). \([at^2] = (LT^{-2})(T^2) = L\). All terms are \(L\). ✓

4. Finding Unknown Powers in a Formula

Assume a physical quantity depends as a product of powers of other quantities:

\(X = k\,A^a B^b C^c\ldots\)  (where \(k\) is dimensionless).

Equate dimensions of each side, giving a linear system for the powers.

The period \(T\) of a simple pendulum depends on length \(\ell\), mass \(m\), and gravity \(g\). Find the form of the dependence.
Assume \(T = k\,\ell^a m^b g^c\). Dimensions: \(T = L^a \cdot M^b \cdot (LT^{-2})^c = M^b L^{a+c} T^{-2c}\).
Match \(T^1 M^0 L^0\):
\(-2c = 1 \Rightarrow c = -\tfrac{1}{2}\). \(b = 0\). \(a + c = 0 \Rightarrow a = \tfrac{1}{2}\).
So \(T = k\sqrt{\ell/g}\) — the mass does not appear.

5. Limitations

  • Dimensional analysis cannot determine dimensionless constants (like \(k\) in the example above, which is actually \(2\pi\)).
  • It cannot distinguish between quantities of the same dimensions (torque and energy both have \(ML^2T^{-2}\)).
  • It works only for equations built from products of powers — it misses additive terms.

6. Dimensional Analysis and Constants

Many physical laws contain constants with specific dimensions.

  • Universal gravitational constant \(G\): \([G] = M^{-1}L^3 T^{-2}\).
  • Coefficient of restitution \(e\): dimensionless.
  • Spring constant \(k\) (Hooke's law): \([k] = MT^{-2}\).

7. Scaling and Model Similarity

Dimensional analysis underpins scaling arguments: if a quantity depends only on certain others, models at different scales should have the same dimensionless groups. (Beyond OCR scope, but worth knowing.)

8. Exam Style Strategy

  • Always check dimensions before accepting a derived formula.
  • Work in fundamental units (\(M, L, T\)) — don't mix SI units in with dimensions.
  • Remember trig-argument rule: anything inside \(\sin, \cos, \exp, \ln\) must be dimensionless.
  • Set up simultaneous equations carefully for each base dimension.
