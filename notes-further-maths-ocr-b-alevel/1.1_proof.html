1.1: Proof

1. Types of Proof

A proof is a sequence of valid deductions from known premises, leading to the conclusion of a theorem.
Main strategies in the A-Level syllabus:
  • Direct proof (deduction): a chain of implications.
  • Proof by contradiction: assume the negation of what you want to prove, reach a contradiction.
  • Proof by exhaustion: split the domain into finitely many cases and check each.
  • Counter-example: disprove "\(\forall x, P(x)\)" by exhibiting one \(x\) with \(\neg P(x)\).
  • Mathematical induction: prove \(P(1)\) then \(P(k)\Rightarrow P(k+1)\) (see §3).

2. Direct Proof and Contrapositive

To prove \(P\Rightarrow Q\) directly, assume \(P\), deduce \(Q\) using known facts.

The contrapositive of "\(P\Rightarrow Q\)" is "\(\neg Q\Rightarrow\neg P\)". They are logically equivalent, so one can prove the contrapositive instead.

Prove: if \(n^2\) is even, then \(n\) is even.
Use contrapositive: if \(n\) is odd, then \(n^2\) is odd.
Let \(n=2k+1\). Then \(n^2=4k^2+4k+1=2(2k^2+2k)+1\), odd ∎

3. Proof by Induction

To prove \(P(n)\) for all integers \(n\ge n_0\):
  1. Base case: show \(P(n_0)\) holds.
  2. Inductive step: assume \(P(k)\) (inductive hypothesis), prove \(P(k+1)\).
  3. Conclusion: by induction, \(P(n)\) holds for all \(n\ge n_0\).

Series Induction

Prove \(\displaystyle\sum_{r=1}^{n}r=\tfrac{n(n+1)}{2}\).
Base \(n=1\): LHS=1, RHS=1 ✓.
Assume true for \(n=k\): \(\sum_{r=1}^{k}r=\tfrac{k(k+1)}{2}\).
Add \((k+1)\): \(\tfrac{k(k+1)}{2}+(k+1)=\tfrac{(k+1)(k+2)}{2}\), i.e. \(P(k+1)\) ✓.
By induction, \(P(n)\) true for all \(n\ge1\) ∎

Divisibility Induction

Pattern: write \(f(k+1)=f(k)+(\text{term obviously divisible by }d)\) to show \(d\mid f(k+1)\).

Prove \(7^n-1\) is divisible by 6 for all \(n\ge1\).
Base \(n=1\): \(7-1=6\) ✓.
Assume \(7^k-1=6m\). Then \(7^{k+1}-1=7\cdot7^k-1=7(6m+1)-1=42m+6=6(7m+1)\), divisible by 6 ✓ ∎

Matrix Induction

Prove \(\begin{pmatrix}1&1\\0&1\end{pmatrix}^n=\begin{pmatrix}1&n\\0&1\end{pmatrix}\).
Base \(n=1\) ✓. Assume \(M^k=\begin{pmatrix}1&k\\0&1\end{pmatrix}\). Then \(M^{k+1}=M^k\cdot M=\begin{pmatrix}1&k+1\\0&1\end{pmatrix}\) ✓ ∎

Inequality Induction

Prove \(2^n>n\) for all \(n\ge1\).
Base \(n=1\): \(2>1\) ✓. Assume \(2^k>k\). Then \(2^{k+1}=2\cdot2^k>2k\ge k+1\) for \(k\ge1\) ✓ ∎

4. Proof by Contradiction

To prove \(P\) by contradiction: assume \(\neg P\), derive a logical contradiction, conclude \(P\) is true.
Prove \(\sqrt{2}\) is irrational.
Suppose \(\sqrt{2}=a/b\) in lowest terms. Then \(a^2=2b^2\), so \(a^2\) even \(\Rightarrow a\) even. Let \(a=2c\): \(4c^2=2b^2\), so \(b^2=2c^2\), hence \(b\) even. But then \(\gcd(a,b)\ge2\), contradicting lowest terms ∎

5. Common Pitfalls

  • Forgetting the base case — induction proves nothing without it.
  • Not explicitly stating the inductive hypothesis.
  • Omitting the concluding sentence — OCR typically awards a mark for it.
  • Using what you wish to prove (circular reasoning).
  • Believing that many examples constitute a proof — they don't (counter-example: \(n^2+n+41\) is prime for \(n=0,\dots,39\) but not for \(n=40\)).

6. Template for OCR Induction Questions

1. Base: for \(n=n_0\), LHS=\(\cdots\), RHS=\(\cdots\), so \(P(n_0)\) true.
2. Assume: \(P(k)\) true, i.e. \(\cdots\).
3. Step: consider \(P(k+1)\). Using the assumption, \(\cdots=\) RHS with \(n=k+1\), so \(P(k+1)\) true.
4. Conclusion: since \(P(n_0)\) is true and \(P(k)\Rightarrow P(k+1)\), by induction \(P(n)\) is true for all \(n\ge n_0\).
