1.9: Further Calculus

1. Improper Integrals

An integral is improper if either (a) a limit is \(\pm\infty\), or (b) the integrand is unbounded at a point in the interval.

Evaluate by replacing the problem point by a parameter, then taking a limit:

\(\displaystyle \int_{a}^{\infty} f(x)\,dx = \lim_{R\to\infty} \int_{a}^{R} f(x)\,dx\)
\(\displaystyle \int_{0}^{1} \dfrac{dx}{\sqrt{x}} = \lim_{\varepsilon\to 0^+}\int_{\varepsilon}^{1} x^{-1/2}\,dx = \lim_{\varepsilon\to 0^+} \left[2\sqrt{x}\right]_{\varepsilon}^{1} = 2.\)

If the limit is finite, the integral converges; otherwise it diverges.

OCR Exam Tip: Always set up with a limit and show the work — simply writing \(\int_0^{\infty}\) and substituting \(\infty\) directly loses marks.

2. The Mean Value of a Function

The mean (average) value of \(f\) on \([a, b]\) is \[\bar{f} = \dfrac{1}{b - a}\int_a^b f(x)\,dx.\]
Mean of \(f(x) = \sin x\) on \([0, \pi]\):
\(\bar{f} = \dfrac{1}{\pi}\int_0^{\pi}\sin x\,dx = \dfrac{1}{\pi}[-\cos x]_0^{\pi} = \dfrac{1}{\pi}(1 + 1) = \dfrac{2}{\pi}.\)

3. Integration by Partial Fractions

Once a rational function is split into partial fractions (see §1.3), each piece integrates as:

  • \(\int \dfrac{A}{ax + b}\,dx = \dfrac{A}{a}\ln|ax + b| + C\)
  • \(\int \dfrac{A}{(ax+b)^n}\,dx = \dfrac{A}{a(1-n)(ax+b)^{n-1}} + C\)  (for \(n \ne 1\))
  • For \(\dfrac{Ax + B}{x^2 + c^2}\), split into \(\tfrac{A}{2}\ln|x^2 + c^2|\) (from \(d/dx\) of denominator) + \(\tfrac{B}{c}\arctan(x/c)\).

5. Reduction Formulae

A reduction formula expresses an integral \(I_n\) (depending on integer \(n\)) in terms of \(I_{n-1}\) or earlier values, allowing successive calculation.

Usually derived by integration by parts.

Derive a reduction formula for \(I_n = \int_0^{\pi/2} \sin^n x\,dx\).
Let \(u = \sin^{n-1}x\), \(dv = \sin x\,dx\), so \(du = (n-1)\sin^{n-2}x \cos x\,dx\), \(v = -\cos x\).
\(I_n = [-\sin^{n-1}x \cos x]_0^{\pi/2} + (n-1)\int_0^{\pi/2}\sin^{n-2}x \cos^2 x\,dx\)
The boundary term vanishes (both endpoints give zero). Use \(\cos^2 x = 1 - \sin^2 x\):
\(I_n = (n-1)(I_{n-2} - I_n)\)  \(\Rightarrow\)  \(nI_n = (n-1)I_{n-2}\)  \(\Rightarrow\)  \(I_n = \dfrac{n-1}{n}I_{n-2}\).
Hence find \(\int_0^{\pi/2}\sin^6 x\,dx\).
\(I_0 = \pi/2\). \(I_2 = \tfrac{1}{2}I_0 = \pi/4\). \(I_4 = \tfrac{3}{4}I_2 = 3\pi/16\). \(I_6 = \tfrac{5}{6}I_4 = \dfrac{15\pi}{96} = \dfrac{5\pi}{32}\).
OCR Exam Tip: State the substitution or choice of \(u\) and \(dv\) clearly. When the boundary terms vanish, say so and give the values — don't just drop them silently.

6. Arc Length

The length of the curve \(y = f(x)\) from \(x = a\) to \(x = b\) is

\(\displaystyle L = \int_a^b \sqrt{1 + \left(\dfrac{dy}{dx}\right)^2}\,dx.\)

For parametric curves \(x = x(t)\), \(y = y(t)\):

\(\displaystyle L = \int_{t_1}^{t_2} \sqrt{\left(\dfrac{dx}{dt}\right)^2 + \left(\dfrac{dy}{dt}\right)^2}\,dt.\)
Arc length of \(y = \tfrac{2}{3}x^{3/2}\) from \(x=0\) to \(x=3\).
\(\dfrac{dy}{dx} = x^{1/2}\), so integrand \(= \sqrt{1 + x}\).
\(L = \int_0^3 \sqrt{1+x}\,dx = \left[\tfrac{2}{3}(1+x)^{3/2}\right]_0^3 = \tfrac{2}{3}(8 - 1) = \tfrac{14}{3}.\)

7. Surface Area of Revolution

Rotating \(y = f(x)\), \(a \le x \le b\), about the \(x\)-axis produces a surface of area

\(\displaystyle S = 2\pi\int_a^b y\,\sqrt{1 + \left(\dfrac{dy}{dx}\right)^2}\,dx.\)

About the \(y\)-axis (if \(x \ge 0\) throughout):

\(\displaystyle S = 2\pi\int_a^b x\,\sqrt{1 + \left(\dfrac{dy}{dx}\right)^2}\,dx.\)

Parametric form: replace \(\sqrt{1 + (dy/dx)^2}\,dx\) by \(\sqrt{(dx/dt)^2 + (dy/dt)^2}\,dt\) and \(y\) (or \(x\)) by its parametric expression.

8. Polar and Parametric Forms (Support)

Polar arc length and area are covered in §1.7; here we note that calculus on curves in any representation comes down to arc length element \(ds\):

\(ds^2 = dx^2 + dy^2\)

9. Exam Style Strategy

  • For improper integrals, show the limit explicitly.
  • For reduction formulae, establish the base case(s) \(I_0\) or \(I_1\) cleanly.
  • Arc length and surface integrals are sensitive to the axis of rotation — reread the question.
  • Trig / hyperbolic substitutions: write down both the substitution and \(dx\); update limits if definite.
