Shoe size is a common trick: it looks continuous but is actually discrete (sizes are 4, 4.5, 5, … — not every value possible). Number of siblings is always discrete (you cannot have 2.3 siblings). Height and weight are always continuous.

6.1.2 Averages and Range

The three averages and range are:

  • Mean: sum of all values ÷ number of values. Notation: $\bar{x} = \dfrac{\sum x}{n}$
  • Median: middle value when data is arranged in order. For $n$ values, position is $\dfrac{n+1}{2}$
  • Mode: most frequent value (there can be more than one, or none)
  • Range: largest value $-$ smallest value (measures spread, not average)

Comparing the Averages

6.1.3 Bar Charts, Pictograms and Pie Charts

Pie Charts

In a pie chart, each sector represents a category. The angle of each sector is:

$$\text{Angle} = \frac{\text{frequency}}{\text{total frequency}} \times 360°$$
30 students were asked their favourite sport. Results: Football 12, Tennis 8, Swimming 6, Other 4.

When drawing a pie chart, always check your angles add to 360°. When reading a pie chart to find a frequency, rearrange the formula: frequency = (angle / 360) × total.

6.1.4 Frequency Diagrams and Polygons

A frequency polygon is drawn for grouped data by:
  1. Finding the midpoint of each class interval.
  2. Plotting (midpoint, frequency) for each class.
  3. Joining the points with straight lines.
  4. Extending the line to the midpoints of the empty classes at each end (touching the x-axis).

Frequency polygons are particularly useful for comparing two distributions on the same diagram.

A frequency polygon is NOT a bar chart. Do not shade under the lines. Plot points at midpoints, not at the boundaries of the class intervals.

6.1.5 Scatter Diagrams and Correlation

A scatter diagram shows the relationship between two variables. Each point represents one data item (its two measurements).

  • Positive correlation: as one variable increases, so does the other.
  • Negative correlation: as one variable increases, the other decreases.
  • No correlation: no clear pattern between the variables.
  • Correlation can be strong (points close to a line) or weak (points more scattered).

Line of Best Fit

A line of best fit (or trend line) is a straight line that best represents the data on a scatter diagram. It passes through the mean point $(\bar{x}, \bar{y})$ and has roughly equal numbers of points above and below it.

Interpolation vs Extrapolation

InterpolationExtrapolation
Using the line of best fit within the data rangeUsing the line of best fit outside the data range
Generally reliableLess reliable — the pattern may not continue

Correlation vs Causation

Correlation does not imply causation. Just because two variables are correlated does not mean one causes the other — there may be a third (confounding) variable, or the correlation may be coincidental.
In exam questions about scatter diagrams: (1) describe the correlation (strong/weak, positive/negative/none), (2) use the line of best fit to make predictions only within the data range, (3) if asked whether correlation implies causation, the answer is always no — explain why.

6.1.6 Time Series and Line Graphs

A time series is a set of data recorded at regular time intervals. Points are joined with straight lines to show trends over time.

When reading a time series graph:

  • Identify the overall trend (increasing, decreasing, constant).
  • Identify any seasonal variation (regular up-down pattern).
Moving Averages smooth out seasonal variation to reveal the underlying trend.

For a 4-point moving average, average each group of 4 consecutive values:

$$\text{MA}_1 = \frac{Q1 + Q2 + Q3 + Q4}{4}, \quad \text{MA}_2 = \frac{Q2 + Q3 + Q4 + Q5}{4}, \quad \ldots$$
Plot each moving average at the midpoint of its group of 4 values.
Quarterly sales (£000s): Q1=12, Q2=18, Q3=22, Q4=14, Q1=15, Q2=20

4-point moving averages:
MA1 $= \dfrac{12+18+22+14}{4} = \dfrac{66}{4} = 16.5$

MA2 $= \dfrac{18+22+14+15}{4} = \dfrac{69}{4} = 17.25$

MA3 $= \dfrac{22+14+15+20}{4} = \dfrac{71}{4} = 17.75$

The moving averages are increasing from 16.5 to 17.75, suggesting an upward trend.
Moving averages are plotted at the mid-time point between the first and last data points in each group. For a 4-point MA, plot between the 2nd and 3rd point of each group of 4. The trend line is drawn through the moving average points.

6.1.7 Histograms

A histogram is used for continuous grouped data. Unlike a bar chart, the area of each bar represents the frequency — not the height. This is necessary because class widths may vary.

$$\text{Frequency density} = \frac{\text{Frequency}}{\text{Class width}}$$ $$\text{Frequency} = \text{Frequency density} \times \text{Class width}$$
The table shows the times (in minutes) students spent on homework. Draw a histogram.

Histogram of homework times. Bar area = frequency. Note the wide bars for 20–40 and 60–100 min.

The y-axis of a histogram must be labelled "Frequency density", not "Frequency". If a question gives a partly-drawn histogram and asks you to find a missing frequency, calculate: frequency = frequency density × class width using the scale on the y-axis.

Reading from a Cumulative Frequency Curve

From a cumulative frequency curve for $n$ data values, read off:

  • Median: the value at cumulative frequency $\frac{n}{2}$
  • Lower quartile (Q1): the value at cumulative frequency $\frac{n}{4}$
  • Upper quartile (Q3): the value at cumulative frequency $\frac{3n}{4}$
  • Interquartile range (IQR): $\text{IQR} = Q_3 - Q_1$
  • Percentiles: $p$th percentile is at cumulative frequency $\frac{pn}{100}$

Cumulative frequency curve. Red dashed: median (Q2). Green: lower quartile (Q1). Purple: upper quartile (Q3).

Box Plots (Box-and-Whisker Diagrams)

A box plot shows the distribution of data using five key values:
  • Minimum value (or lower fence if outliers are excluded)
  • Lower quartile Q1
  • Median Q2
  • Upper quartile Q3
  • Maximum value (or upper fence if outliers are excluded)

Box plot for heights data. The box spans Q1 to Q3 (the IQR). The red line is the median. Whiskers extend to minimum and maximum.

When comparing two box plots, comment on: (1) the median (typical value — which is higher?), (2) the IQR or range (spread — which is more consistent/variable?), (3) the skewness (if the median is closer to Q1, the data is positively skewed; closer to Q3, negatively skewed). Always use the context of the question in your comparison.

6.1.9 Outliers and Interquartile Range

An outlier is a data value that is unusually far from the rest of the data. The standard definition used in GCSE/A-Level: $$\text{Outlier if } x < Q_1 - 1.5 \times \text{IQR} \quad \text{or} \quad x > Q_3 + 1.5 \times \text{IQR}$$ These limits are called the lower fence and upper fence.
Data set: 12, 15, 18, 21, 23, 25, 27, 30, 55.

Step 1: Arrange in order (already done). $n = 9$.
Q1 = 18 (lower half: 12, 15, 18, 21; median = 16.5... actually Q1 = median of lower half = 15+18)/2=16.5)
Let's compute properly: Q1 = $\frac{15+18}{2} = 16.5$,   Q2 = 23,   Q3 = $\frac{27+30}{2} = 28.5$
IQR = 28.5 − 16.5 = 12
Lower fence = 16.5 − 1.5(12) = 16.5 − 18 = −1.5
Upper fence = 28.5 + 1.5(12) = 28.5 + 18 = 46.5
Outlier check: 55 > 46.5, so 55 is an outlier. All other values are between −1.5 and 46.5.
On a box plot, outliers are shown as individual crosses (×) or dots beyond the whiskers. The whiskers then extend only to the most extreme values that are not outliers (the fences). Always state the fence values when asked to identify outliers.

6.1.10 Standard Deviation

Standard deviation measures the average spread of data around the mean. A larger standard deviation means data is more spread out. $$\sigma = \sqrt{\frac{\sum(x - \bar{x})^2}{n}} = \sqrt{\frac{\sum x^2}{n} - \bar{x}^2}$$ The second formula (using $\sum x^2$) is usually easier to compute.
Find the mean and standard deviation of: 3, 5, 7, 8, 12.

$\bar{x} = \dfrac{3+5+7+8+12}{5} = \dfrac{35}{5} = 7$

$\sum x^2 = 9 + 25 + 49 + 64 + 144 = 291$

$\sigma = \sqrt{\dfrac{291}{5} - 7^2} = \sqrt{58.2 - 49} = \sqrt{9.2} \approx 3.03$

Standard Deviation from a Frequency Table

For data in a frequency table with values $x_i$ and frequencies $f_i$: $$\bar{x} = \frac{\sum f_i x_i}{\sum f_i} \qquad \sigma = \sqrt{\frac{\sum f_i x_i^2}{\sum f_i} - \bar{x}^2}$$ For grouped data, use the midpoint of each class in place of $x_i$.
Find the standard deviation for the grouped height data (using midpoints):

Standard deviation is always non-negative. A standard deviation of 0 means all values are identical. For grouped data, always use midpoints and call the answer an estimate. On a calculator, use the $\sigma_n$ function (not $\sigma_{n-1}$, which is the sample standard deviation) unless told otherwise.
